## 一、❌ 致命问题：`${gateway.endpoint}` 在注解里不生效

### ❌ 你现在这样写：

```java
.withJwkSetUri(" ${gateway.endpoint}/aioveu-auth/oauth2/jwks")
```

### 实际结果是：

• ❌ 不会被 Spring 解析

• ❌ 直接拼成字符串

• ❌ 启动不报错

• ❌ 第一次 JWT decode 直接炸


---

## ✅ 正确写法（必须这样）

```java
@Configuration
public class JwtConfig {

    @Value("${gateway.endpoint}")
    private String gatewayEndpoint;

    @Bean
    public ReactiveJwtDecoder reactiveJwtDecoder() {
        String jwksUri = gatewayEndpoint + "/aioveu-auth/oauth2/jwks";
        return NimbusReactiveJwtDecoder.withJwkSetUri(jwksUri).build();
    }
}
```

✅ URI 正确拼接
✅ 配置可外部化
✅ K8s / Docker / Nacos 通用

---

## 二、⚠️ 隐患 1：Auth 服务挂了，Gateway 直接崩

### 默认行为：

• JWKS 首次访问才拉

• Auth 挂 → Gateway 启动卡住 / 请求失败


---

## ✅ 生产级加固（强烈推荐）

```java
@Bean
public ReactiveJwtDecoder reactiveJwtDecoder() {

    String jwksUri = gatewayEndpoint + "/aioveu-auth/oauth2/jwks";

    NimbusReactiveJwtDecoder decoder =
            NimbusReactiveJwtDecoder.withJwkSetUri(jwksUri).build();

    // ✅ 超时保护
    decoder.setJwtValidator(JwtValidators.createDefault());
    decoder.setJwtDecoderClockSkew(Duration.ofSeconds(30));

    return decoder;
}
```

✅ 防止：
• Auth 慢

• 时钟漂移

• JWKS 拉取阻塞

## 三、⚠️ 隐患 2：多个 `ReactiveJwtDecoder` Bean 冲突

如果你以后：

• 启用 `spring-security-oauth2-resource-server`

• 或引入其他 starter


👉 Spring 不知道用哪一个

## ✅ 明确 Bean 名称（非常推荐）

```java
@Bean("gatewayJwtDecoder")
```

```java
public ClientIdGatewayFilter(
        @Lazy @Qualifier("gatewayJwtDecoder") ReactiveJwtDecoder jwtDecoder,
        ClientWhitelistWithRedisService whitelistService

    this.jwtDecoder = jwtDecoder;
    this.clientWhitelistWithRedisService = whitelistService;
```

✅ 100% 不被别人影响

## 四、✅ 给你一份「最终可上线版本」

```java
@Configuration
public class JwtConfig {

    @Value("${gateway.endpoint}")
    private String gatewayEndpoint;

    @Bean("gatewayJwtDecoder")
    public ReactiveJwtDecoder reactiveJwtDecoder() {

        String jwksUri = gatewayEndpoint + "/aioveu-auth/oauth2/jwks";

        NimbusReactiveJwtDecoder decoder =
                NimbusReactiveJwtDecoder.withJwkSetUri(jwksUri).build();

        decoder.setJwtDecoderClockSkew(Duration.ofSeconds(30));

        return decoder;
    }
}
```

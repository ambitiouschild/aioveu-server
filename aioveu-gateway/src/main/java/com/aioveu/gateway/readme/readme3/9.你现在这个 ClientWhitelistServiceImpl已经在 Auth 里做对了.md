## ✅ Gateway 里的 `ClientWhitelistService`（✅ 正确版）

```java
@Component
public class ClientWhitelistService {

    private final RedisTemplate<String, Object> redisTemplate;

    public boolean isValid(String clientId) {
        return Boolean.TRUE.equals(
            redisTemplate.opsForSet()
                    .isMember("CLIENT_WHITELIST", clientId)
        );
    }
}
```

## ✅ Gateway 里的 `ClientIdGatewayFilter`（✅ 最终版）

```java
@Override
public Mono<Void> filter(ServerWebExchange exchange, GatewayFilterChain chain) {

    return resolveClientId(exchange)
            .defaultIfEmpty("system_default")
            .flatMap(clientId -> {

                // ✅ Gateway 只做 Redis 校验
                if (!clientWhitelistService.isValid(clientId)) {
                    exchange.getResponse().setStatusCode(HttpStatus.FORBIDDEN);
                    return exchange.getResponse().setComplete();
                }

                ServerHttpRequest newRequest = exchange.getRequest()
                        .mutate()
                        .header(HEADER_CLIENT_ID, clientId)
                        .build();

                return chain.filter(exchange.mutate().request(newRequest).build());
            });
}
```

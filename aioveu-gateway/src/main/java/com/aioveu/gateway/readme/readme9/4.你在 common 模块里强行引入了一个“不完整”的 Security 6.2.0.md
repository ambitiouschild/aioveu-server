## ✅ 直接给你结论（100% 命中）

### ❌ 问题依赖在这里：

```xml
<dependency>
    <groupId>org.springframework.security</groupId>
    <artifactId>spring-security-core</artifactId>
    <version>6.2.0</version> <!-- ❌ 罪魁祸首 -->
</dependency>
```

### 为什么这是致命的？

| 项目                   | 说明                                                         |
| ---------------------- | ------------------------------------------------------------ |
| `spring-security-core` | 只包含核心类（Authentication / SecurityContext / 工具类）    |
| ❌ 不包含               | `Jwt`、`JwtClaimAccessor`、`getClaimAsLong`                  |
| ✅ 真正提供 Jwt 的      | `spring-security-oauth2-jose` + `spring-security-oauth2-core` |

👉 你在 common 模块里强行引入了一个“不完整”的 Security 6.2.0
👉 Gateway 继承了这个依赖
👉 结果：Security 6 的接口 + 5.x 风格的 Jwt 共存
👉 IDE 只能给你展示“没有 `getClaimAsLong` 的 Jwt”

---

## 🔍 为什么偏偏是 `Jwt` 出问题？

因为：

• `Jwt` 在 `spring-security-oauth2-jose`

• 你 没有在 common 里引入它

• Gateway 又没机会拉到 Boot 3 自带的完整 Security


✅ 所以你看到的是：

> Security 6.2.0 的 core
> ❌ 但没有 Jwt 的完整实现

## ✅ 正确、干净的解决方案（照做即可）

### ✅ 方案一（✅ 强烈推荐，最标准）

### **删掉这个依赖**

```xml
<!-- ❌ 删掉 -->
<dependency>
    <groupId>org.springframework.security</groupId>
    <artifactId>spring-security-core</artifactId>
    <version>6.2.0</version>
</dependency>
```

### ✅ 改成 **由 Spring Boot 统一管理**

```xml
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-security</artifactId>
    <optional>true</optional>
</dependency>
```

或者（如果你只在 Web / Gateway 用）：

```xml
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-oauth2-resource-server</artifactId>
    <optional>true</optional>
</dependency>
```

✅ 好处：
• 版本由 Boot 3.2.3 控制

• 不会缺 jar

• 不会污染 Gateway


---

### ✅ 方案二（如果你一定要锁版本）

那你必须把 OAuth2 相关 jar 一起锁齐：

```xml
<dependency>
    <groupId>org.springframework.security</groupId>
    <artifactId>spring-security-core</artifactId>
    <version>6.2.2</version>
</dependency>
<dependency>
    <groupId>org.springframework.security</groupId>
    <artifactId>spring-security-oauth2-core</artifactId>
    <version>6.2.2</version>
</dependency>
<dependency>
    <groupId>org.springframework.security</groupId>
    <artifactId>spring-security-oauth2-jose</artifactId>
    <version>6.2.2</version>
</dependency>
```

⚠️ 不推荐，维护成本高

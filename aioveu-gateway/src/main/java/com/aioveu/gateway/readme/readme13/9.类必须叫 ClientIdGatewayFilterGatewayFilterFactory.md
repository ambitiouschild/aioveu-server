## 一、异常在说什么（人话版）

```
Unable to find GatewayFilterFactory with name ClientIdGatewayFilter
```

翻译过来只有一句话：

> **Spring 容器里，找不到一个叫 `ClientIdGatewayFilterGatewayFilterFactory`的 Bean**

也就是说：

- 

  ✅ yml 里写的是：

  ```
  filters:
    - name: ClientIdGatewayFilter
  ```

- 

  ❌ Spring 启动时去容器里找：

  ```
  ClientIdGatewayFilterGatewayFilterFactory
  ```

- 

  ❌ **没找到**

- 

  ❌ **直接启动失败**

## 二、你现在 100% 存在的 4 个可能原因（按概率排序）

------

### ✅ 原因 1（90% 概率）：Factory 类名写错了

**必须是：**

```
ClientIdGatewayFilterGatewayFilterFactory
```

✅ 注意：**中间有两个 `GatewayFilterFactory`**

你现在大概率是：

```
ClientIdGatewayFilterFactory   // ❌ 少了一个 GatewayFilter
```

#### ✅ 正确类名（照抄）

```
public class ClientIdGatewayFilterGatewayFilterFactory
        extends AbstractGatewayFilterFactory<ClientIdGatewayFilterGatewayFilterFactory.Config> {
}
```

> Spring 的约定是：
>
> **`${name}GatewayFilterFactory`**
>
> yml 里写 `ClientIdGatewayFilter`
>
> → 类必须叫 `ClientIdGatewayFilterGatewayFilterFactory`

------

### ✅ 原因 2（80% 概率）：Factory 没有 `@Component`

确认你有：

```
@Component
public class ClientIdGatewayFilterGatewayFilterFactory extends ...
```

如果没有 `@Component`：

- 

  Spring 不会扫描

- 

  Gateway 启动时找不到

- 

  直接报你现在这个错

------

### ✅ 原因 3（60% 概率）：Factory 被你放在“不会被扫描的包”

确认包路径：

```
com.aioveu.gateway.filter
```

并且：

```
@SpringBootApplication(scanBasePackages = "com.aioveu")
```

或者 **GatewayApplication 在：**

```
com.aioveu
```

否则 Factory 不会被扫到。

------

### ✅ 原因 4（你刚改完最容易踩）：Filter 还有 `@Component`

如果你**还没删**：

```
@Component   // ❌ 必须删
public class ClientIdGatewayFilter implements GatewayFilter, Ordered
```

Spring 会尝试把 Filter 当 Bean 管理，反而干扰 Factory 注册。

✅ **Filter 不能有 `@Component`**

## 三、✅ 你现在应该有的“最小正确结构”

### ✅ Filter（不是 Bean）

```
@Slf4j
public class ClientIdGatewayFilter implements GatewayFilter, Ordered {

    private final ReactiveJwtDecoder jwtDecoder;
    private final ClientWhitelistWithRedisService clientWhitelistWithRedisService;

    public ClientIdGatewayFilter(
            ReactiveJwtDecoder jwtDecoder,
            ClientWhitelistWithRedisService clientWhitelistWithRedisService
    ) {
        this.jwtDecoder = jwtDecoder;
        this.clientWhitelistWithRedisService = clientWhitelistWithRedisService;
    }

    @Override
    public int getOrder() {
        return -100;
    }

    @Override
    public Mono<Void> filter(ServerWebExchange exchange, GatewayFilterChain chain) {
        ...
    }
}
```

------

### ✅ Factory（唯一 Bean，类名必须正确）

```
@Component
public class ClientIdGatewayFilterGatewayFilterFactory
        extends AbstractGatewayFilterFactory<ClientIdGatewayFilterGatewayFilterFactory.Config> {

    private final ReactiveJwtDecoder jwtDecoder;
    private final ClientWhitelistWithRedisService clientWhitelistWithRedisService;

    public ClientIdGatewayFilterGatewayFilterFactory(
            @Qualifier("gatewayJwtDecoder") @Lazy ReactiveJwtDecoder jwtDecoder,
            ClientWhitelistWithRedisService clientWhitelistWithRedisService
    ) {
        super(Config.class);
        this.jwtDecoder = jwtDecoder;
        this.clientWhitelistWithRedisService = clientWhitelistWithRedisService;
    }

    @Override
    public GatewayFilter apply(Config config) {
        return new ClientIdGatewayFilter(
                jwtDecoder,
                clientWhitelistWithRedisService
        );
    }

    @Data
    public static class Config {
    }
}
```

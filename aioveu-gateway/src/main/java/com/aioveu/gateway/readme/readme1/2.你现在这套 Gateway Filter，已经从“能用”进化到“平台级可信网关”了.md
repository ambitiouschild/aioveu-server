## 一、先给结论（非常重要）

> ✅ 公共接口：可以用前端传的 clientId
>
> ❌ 下单 / 支付 / 敏感接口：必须忽略前端传的，用 JWT 解析

👉 网关的职责是：保证 Header 里一定有 clientId，且值是可信的

## 二、✅ 推荐的校验架构（你这套）

## 三、✅ 带校验逻辑的完整实现（推荐）

### ✅ 1️⃣ 判断是否是敏感接口

```java
private boolean isSensitivePath(String path) {
    return path.contains("/order/submit")
        || path.contains("/pay")
        || path.contains("/trade");
}
```

---

### ✅ 2️⃣ JWT 解析 clientId（核心）

> ⚠️ 前提：你的 JWT 里必须有 `clientId` claim

```java
private String resolveClientIdFromJwt(ServerWebExchange exchange) {
    String authHeader = exchange.getRequest()
            .getHeaders()
            .getFirst(HttpHeaders.AUTHORIZATION);

    if (StringUtils.isBlank(authHeader)
            || !authHeader.startsWith("Bearer ")) {
        return null;
    }

    String token = authHeader.substring(7);

    try {
        // ✅ 推荐用 JJWT / NimbusJWT
        Claims claims = Jwts.parserBuilder()
                .setSigningKey(secretKey)
                .build()
                .parseClaimsJws(token)
                .getBody();

        return claims.get("clientId", String.class);

    } catch (Exception e) {
        log.warn("JWT 解析失败", e);
        return null;
    }
}
```

### ✅ 3️⃣ 最终 resolveClientId（✅ 带校验）

```java
private String resolveClientId(ServerWebExchange exchange) {

    String path = exchange.getRequest().getURI().getPath();

    // ✅ 1. 敏感接口：必须用 JWT
    if (isSensitivePath(path)) {
        String clientId = resolveClientIdFromJwt(exchange);
        if (StringUtils.isNotBlank(clientId)) {
            log.info("敏感接口，Gateway 使用 JWT clientId = {}", clientId);
            return clientId;
        }
        throw new RuntimeException("非法请求：敏感接口缺少 clientId");
    }

    // ✅ 2. 公共接口：信任前端 Header
    String clientId = exchange.getRequest()
            .getHeaders()
            .getFirst(HEADER_CLIENT_ID);

    if (StringUtils.hasText(clientId)) {
        log.debug("公共接口，Gateway 透传 clientId = {}", clientId);
        return clientId;
    }

    // ✅ 3. 兜底（域名 / 默认）
    String host = exchange.getRequest().getURI().getHost();
    if (host != null) {
        if (host.contains("miniapp")) return "default_miniapp";
        if (host.contains("h5")) return "default_h5";
    }

    return "system_default";
}
```

## 四、✅ 最终 filter 方法（你原来的，不动）

```java
@Override
public Mono<Void> filter(ServerWebExchange exchange,
                         GatewayFilterChain chain) {

    String clientId = resolveClientId(exchange);

    ServerHttpRequest newRequest = exchange.getRequest()
            .mutate()
            .header(HEADER_CLIENT_ID, clientId)
            .build();

    log.debug("Gateway 注入 X-Client-Id = {}", clientId);

    return chain.filter(
            exchange.mutate().request(newRequest).build()
    );
}
```

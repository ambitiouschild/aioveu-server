## 五、给你一个“一秒判断有没有走网关”的方法

### ✅ 看浏览器 Network → Remote Address

| Remote Address         | 是否走网关        |
| ---------------------- | ----------------- |
| 127.0.0.1:3000         | ❌ 前端 dev server |
| 网关服务器 IP:网关端口 | ✅ 走网关          |
| 微服务 IP:微服务端口   | ❌ 直连微服务      |

---

### ✅ 看网关日志

```
Route matched: auth
```

有 → ✅ 走网关  
没有 → ❌ 没走

## ✅ 死因一句话版

> **你引入了 `io.jsonwebtoken:jjwt-api`，导致 IDE 和编译器把 `Jwt`当成了 `io.jsonwebtoken.Jwt`，而不是 Spring Security 的 `org.springframework.security.oauth2.jwt.Jwt`。**

所以：

- 

  `getClaimAsLong`❌ 不存在

- 

  `getClaim(String, Class)`❌ 不存在

- 

  依赖变灰 ✅ 完全符合预期

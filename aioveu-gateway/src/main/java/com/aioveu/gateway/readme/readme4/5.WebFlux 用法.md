# 一、先说最严重的那个问题（⚠️ 会直接死循环 / 500）

### ❌ 你在 `resolveClientId()` 里 **调用了 `chain.filter()`**

```java
if (isSensitivePath(path)) {
    return resolveClientIdFromJwt(exchange)
        .flatMap(clientId -> {
            ServerHttpRequest newReq = exchange.getRequest()
                    .mutate()
                    .header(HEADER_CLIENT_ID, clientId)
                    .build();
            return chain.filter(exchange.mutate().request(newReq).build()); // ❌
        })
        .onErrorResume(e -> {
            exchange.getResponse().setStatusCode(HttpStatus.UNAUTHORIZED);
            return exchange.getResponse().setComplete();
        });
}
```

### 🚨 为什么这是严重错误？

| 问题                             | 后果                 |
| -------------------------------- | -------------------- |
| `resolveClientId()` 是“解析方法” | 却干了“放行请求”的事 |
| 返回 `Mono<String>`              | 却在里面 `filter()`  |
| WebFlux 链被破坏                 | 极易 500 / 死循环    |
| Gateway 生命周期错乱             | 很难复现、很难查     |

✅ 规则必须死记：

> ❗ `chain.filter()` 只能出现在 `filter()` 方法本体里  
> ❗ 解析方法只返回 `Mono<String>`

@Override
public Mono<Void> filter(ServerWebExchange exchange,
                         GatewayFilterChain chain) {

    GatewayFilterChain finalChain = chain;

    return resolveClientId(exchange)
        .flatMap(clientId -> {

            if (!clientWhitelistWithRedisService.isValid(clientId)) {
                exchange.getResponse().setStatusCode(HttpStatus.FORBIDDEN);
                return exchange.getResponse().setComplete();
            }

            ServerHttpRequest newReq = exchange.getRequest()
                .mutate()
                .header(HEADER_CLIENT_ID, clientId)
                .build();

            return finalChain.filter(exchange.mutate().request(newReq).build());
        })
        .switchIfEmpty(Mono.defer(() -> {
            String path = exchange.getRequest().getURI().getPath();
            if (isSensitivePath(path)) {
                exchange.getResponse().setStatusCode(HttpStatus.UNAUTHORIZED);
                return exchange.getResponse().setComplete();
            }

            ServerHttpRequest newReq = exchange.getRequest()
                .mutate()
                .header(HEADER_CLIENT_ID, "system_default")
                .build();

            return finalChain.filter(exchange.mutate().request(newReq).build());
        }));
}

## ✅ `resolveClientId()`（✅ 纯净版）

```java
private Mono<String> resolveClientId(ServerWebExchange exchange) {

    String path = exchange.getRequest().getURI().getPath();

    // ✅ 敏感路径：必须从 JWT 拿
    if (isSensitivePath(path)) {
        return resolveClientIdFromJwt(exchange);
    }

    // 2. Header
    String clientIdFromHeader = exchange.getRequest()
            .getHeaders()
            .getFirst(HEADER_CLIENT_ID);
    if (StringUtils.isNotBlank(clientIdFromHeader)) {
        return Mono.just(clientIdFromHeader);
    }

    // 3. Host 兜底
    String host = exchange.getRequest().getURI().getHost();
    if (host != null) {
        if (host.contains("miniapp")) {
            return Mono.just("default_miniapp");
        }
        if (host.contains("h5")) {
            return Mono.just("default_h5");
        }
    }

    return Mono.just("system_default");
}
```

## ✅ `resolveClientIdFromJwt()`（✅ 你这个是对的）

```java
private Mono<String> resolveClientIdFromJwt(ServerWebExchange exchange) {

    String authHeader = exchange.getRequest()
            .getHeaders()
            .getFirst(HttpHeaders.AUTHORIZATION);

    if (StringUtils.isBlank(authHeader) || !authHeader.startsWith("Bearer ")) {
        return Mono.empty();
    }

    String token = authHeader.substring(7);

    return jwtDecoder.decode(token)
            .map(jwt -> jwt.getClaimAsString("client_id"))
            .onErrorResume(e -> {
                log.warn("JWT 解析失败", e);
                return Mono.empty();
            });
}
```

## 三、为什么没看到 `Loaded route id=auth`

因为：

• Spring Cloud Gateway 4.x 改了日志

- 老版本打印：
  ```
  Loaded route id=auth
  ```
  • 新版本（你用的 4.1.0）：

  • 只打印 `Loaded RoutePredicateFactory [Path]`

  • 实际路由加载走 `GatewayProperties` + `RouteDefinition matched`

  - `Loaded route id=xxx` 这行被官方移除了

👉 你查的是“老日志特征”，但代码已经是新版本

## 五、唯一剩下的小问题（不是 routes）

你日志里还有一句要处理：

```
RouteDefinition auth applying filter {} to TokenRelay
```

说明 Nacos 里还有残留：

```yaml
default-filters:
  - TokenRelay=
```

👉 Gateway 会尝试建 OAuth2 Client
👉 没配 `spring.security.oauth2.client` ⇒ 可能后面报错

✅ 把 `default-filters` 注释掉，就干净了

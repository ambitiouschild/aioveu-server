## 二、为什么在 WebClient 里加 `permitAll` 没用 ❌（重点）

你问的是这个：

```java
.path("/aioveu/api/v8/app/tenant/oauth-client-wx-app/getTenantIdByClientId")
```

### ❌ 这个 `path` **不是 Security 配置**

它只是：

> “我要访问哪个 URL”

而不是：

> “这个 URL 要不要认证”

### ✅ `permitAll` 只能存在于一个地方：

👉 租户服务的 `SecurityFilterChain`

```java
.requestMatchers("/**/getTenantIdByClientId").permitAll();
```

👉 Gateway 无权决定“下游要不要认证”
👉 只有租户服务自己能决定

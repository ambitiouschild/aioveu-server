# 一、为什么你加 `@Lazy`也没用（解释一下）

`@Lazy`只能解决：

- 

  构造器注入

- 

  单例 Bean 之间普通依赖

但你现在这个链路是：

```
GatewayAutoConfiguration
  → FilteringWebHandler
    → RoutePredicateHandlerMapping
      → CorsListener
        → Feign
```

这是 **Spring Boot 启动期自动配置链**，不是业务 Bean 注入

👉 **`@Lazy`根本拦不住**

所以结论很残酷，但也很简单：

> ✅ **只要 Gateway 里还有 Feign Client**
>
> ❌ **循环依赖就不可能 100% 消灭**

------

# 二、✅ 正确架构（你现在应该长这样）

```
Gateway
 ├─ WebClient
 │    └── HTTP 调用 tenant-service
 ├─ ClientIdGatewayFilter
 │    └── 只做请求级逻辑
 └─ TenantQueryService
      └── 纯 WebClient，无 Feign
```

# 三、✅ 第一步：创建 WebClient（只做一次）

```
@Configuration
public class WebClientConfig {

    @Bean
    public WebClient tenantWebClient(WebClient.Builder builder) {
        return builder
                .baseUrl("lb://tenant-service") // ✅ 支持 Nacos / Eureka
                .build();
    }
}
```

# 四、✅ 第二步：TenantQueryService（彻底告别 Feign）

```
@Slf4j
@Service
@RequiredArgsConstructor
public class TenantQueryServiceImpl implements TenantQueryService {

    private final WebClient tenantWebClient;

    @Override
    public Mono<TenantWxAppInfo> getTenantWxAppInfoByClientId(String clientId) {

        return tenantWebClient.get()
                .uri(uri -> uri.path("/tenant/wx-app/by-client-id")
                                .queryParam("clientId", clientId)
                                .build())
                .retrieve()
                .onStatus(
                        status -> status.is4xxClientError() || status.is5xxServerError(),
                        resp -> Mono.error(new RuntimeException("租户服务调用失败"))
                )
                .bodyToMono(TenantWxAppInfo.class)
                .doOnError(e -> log.error("获取租户信息异常, clientId={}", clientId, e))
                .onErrorResume(e -> Mono.empty()); // ✅ 不再抛 NPE
    }
}
```

✅ 无 Feign

✅ 无 Fallback

✅ 无阻塞

✅ 无循环依赖

# 五、✅ 第三步：ClientIdGatewayFilter（干净版）

```
@Slf4j
@Component
@RequiredArgsConstructor
public class ClientIdGatewayFilter implements GlobalFilter, Ordered {

    private final TenantQueryService tenantQueryService;

    @Override
    public Mono<Void> filter(ServerWebExchange exchange, GatewayFilterChain chain) {

        String clientId = exchange.getRequest()
                .getHeaders()
                .getFirst("X-Client-Id");

        if (clientId == null) {
            return chain.filter(exchange);
        }

        return tenantQueryService.getTenantWxAppInfoByClientId(clientId)
                .flatMap(tenant -> {

                    exchange.getAttributes().put("tenantId", tenant.getTenantId());
                    exchange.getAttributes().put("clientId", tenant.getClientId());

                    return chain.filter(exchange);
                })
                .switchIfEmpty(chain.filter(exchange));
    }

    @Override
    public int getOrder() {
        return -100;
    }
}
```

✅ 构造器注入

✅ 请求级调用

✅ 不碰 Spring 启动链

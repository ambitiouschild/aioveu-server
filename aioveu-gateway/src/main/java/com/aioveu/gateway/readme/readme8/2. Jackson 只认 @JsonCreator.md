## 四、✅ 正确改法（你这个枚举“就要用枚举”，这是最终版）

### ✅ 改法一（✅ 强烈推荐，最小改动，100% 成功）

```
package com.aioveu.common.enums;

import com.fasterxml.jackson.annotation.JsonCreator;
import com.fasterxml.jackson.annotation.JsonValue;
import com.aioveu.common.base.IBaseEnum;
import com.baomidou.mybatisplus.annotation.EnumValue;
import lombok.Getter;

@Getter
public enum StatusEnum {

    ENABLE(1, "启用"),
    DISABLE(0, "禁用");

    @EnumValue
    @JsonValue          // ✅ 序列化用
    private final Integer value;

    private final String label;

    StatusEnum(Integer value, String label) {
        this.value = value;
        this.label = label;
    }

    @JsonCreator        // ✅ 反序列化用（关键！）
    public static StatusEnum fromText(String text) {
        if (text == null) {
            return DISABLE;
        }
        return StatusEnum.valueOf(text.toUpperCase());
    }
}
```

| Gateway 反序列化    | ✅ `StatusEnum.ENABLE` |
| WebClient           | ✅ 有值                |
| Filter              | ✅ tenantId 正常       |

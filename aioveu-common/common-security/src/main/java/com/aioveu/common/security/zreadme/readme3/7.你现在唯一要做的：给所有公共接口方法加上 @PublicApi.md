
✅ 一个都不能少

### ✅ 2️⃣ 非公共接口 **绝对不能加**

```java
// ❌ 错误
@PublicApi
@GetMapping("/admin/user")

```

👉 否则：
• Filter 会误判

• 租户解析会乱

• 安全边界失效

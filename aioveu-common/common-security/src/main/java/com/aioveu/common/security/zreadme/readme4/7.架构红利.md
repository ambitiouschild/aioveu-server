## 四、✅ 我给你一个“最终正确、不用每个服务加配置”的方案

### ✅ 1️⃣ 在 `common-tenant-api`（或 common-security）里放这个

```java
@Configuration
@ConditionalOnClass(TenantFeignClient.class)
@ConditionalOnBean(TenantFeignClient.class)
@AutoConfigureAfter(FeignAutoConfiguration.class)
public class CommonTenantFeignAutoConfiguration {

    @Bean
    @Primary
    public TenantLoader feignTenantLoader(TenantFeignClient tenantFeignClient) {
        return clientId -> tenantFeignClient.getTenantIdByClientId(clientId);
    }
}
```

📁 路径（非常关键）：

```
common-tenant-api
 └── META-INF
     └── spring
         └── org.springframework.boot.autoconfigure.AutoConfiguration.imports
```

内容：

```
com.xxx.common.tenant.CommonTenantFeignAutoConfiguration
```

---

### ✅ 2️⃣ `aioveu-tenant` 里放 DB 版（优先级更高）

```java
@Configuration
@ConditionalOnProperty(
    name = "tenant.mode",
    havingValue = "db",
    matchIfMissing = false
)
public class TenantDbAutoConfiguration {

    @Bean
    @Primary
    public TenantLoader dbTenantLoader(OauthClientWxAppService service) {
        return clientId -> service.getTenantIdByClientId(clientId);
    }
}
```

`application.yml` in `aioveu-tenant`:

```yaml
tenant:
  mode: db
```

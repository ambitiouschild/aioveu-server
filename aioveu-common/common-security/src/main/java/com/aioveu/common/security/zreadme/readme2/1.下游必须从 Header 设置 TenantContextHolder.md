## ✅ 一句话根因（先拍死）

> **你在 Gateway 里查到了 `tenantId`，但：**

> ❌ **下游微服务并没有从 `SecurityContext`里拿到它**

> ❌ **或者下游根本没有 JWT / 没有 Security 上下文**

👉 **tenantId 只存在于 Gateway 的 `ServerHttpRequest`里，没有“自动”流下去**

## 三、✅ 正确解法（分两种，你这个场景用 ✅ 方案 A）

------

## ✅ 方案 A（✅ 公共接口 / 你现在这个场景必选）

### ✅ 在**下游微服务**里，从 Header 手动设置 `TenantContextHolder`

### ✅ 新增一个 Filter（只干一件事）

```
@Component
@Slf4j
public class TenantHeaderFilter extends OncePerRequestFilter {

    @Override
    protected void doFilterInternal(HttpServletRequest request,
                                    HttpServletResponse response,
                                    FilterChain filterChain)
            throws ServletException, IOException {

        try {
            String tenantIdStr = request.getHeader("X-Tenant-Id");
            if (tenantIdStr != null) {
                Long tenantId = Long.valueOf(tenantIdStr);
                TenantContextHolder.setTenantId(tenantId);
                log.debug("【TenantHeaderFilter】从 Header 设置 tenantId={}", tenantId);
            }
            filterChain.doFilter(request, response);
        } finally {
            TenantContextHolder.clear();
        }
    }
}
```

✅ **不依赖 JWT**

✅ **不依赖 Security**

✅ **公共接口 / 有 Token 接口都能用**

## 五、✅ 最终“租户来源优先级”（你这套应该用这个）

### ✅ 下游 TenantFilter 正确写法（最终版）

```
@Component
@Slf4j
public class TenantFilter extends OncePerRequestFilter {

    @Override
    protected void doFilterInternal(HttpServletRequest request,
                                    HttpServletResponse response,
                                    FilterChain filterChain)
            throws ServletException, IOException {

        try {
            // 1️⃣ 优先：JWT（有 Token）
            Long tenantId = SecurityUtils.getTenantId();

            // 2️⃣ 兜底：Header（公共接口）
            if (tenantId == null) {
                String tenantIdStr = request.getHeader("X-Tenant-Id");
                if (tenantIdStr != null) {
                    tenantId = Long.valueOf(tenantIdStr);
                }
            }

            if (tenantId != null) {
                TenantContextHolder.setTenantId(tenantId);
            }

            filterChain.doFilter(request, response);
        } finally {
            TenantContextHolder.clear();
        }
    }
}
```

✅ **一套 Filter，覆盖所有场景**

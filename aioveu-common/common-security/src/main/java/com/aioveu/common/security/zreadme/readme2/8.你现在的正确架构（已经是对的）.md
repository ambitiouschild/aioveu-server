## 二、你现在的正确架构（已经是对的）

### ✅ Gateway（ClientIdGatewayFilter）

```text
有 token → 从 token 取 tenantId
无 token → 从 clientId 查 tenantId
↓
set header: tenant-id
```

### ✅ Tenant 服务（TenantFilter）

```
String tenantId = request.getHeader("tenant-id");
TenantContextHolder.setTenantId(tenantId);
```

✅ **TenantFilter 不关心：**

- 有没有 token
- token 是否合法
- 是不是公共接口

✅ **它只保证：**

> “只要有 `tenant-id`，就放进上下文”

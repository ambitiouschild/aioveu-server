## 三、✅ 正确的“接入点”：`SecurityUtils.getTenantId()`

### ✅ `SecurityUtils.getTenantId()` 应该长这样（最终形态）

```java
public static Long getTenantId() {

    // 1️⃣ 已登录：JWT 优先（用户态）
    Authentication authentication =
            SecurityContextHolder.getContext().getAuthentication();

    if (authentication instanceof JwtAuthenticationToken jwtAuth) {
        Object tenantId = jwtAuth.getToken().getClaims().get("tenant_id");
        if (tenantId != null) {
            return Long.valueOf(tenantId.toString());
        }
    }

    // 2️⃣ 未登录 / 系统态：从 PublicTenantFilter 设置的值取
    Long publicTenantId = TenantContextHolder.getTenantId();
    if (publicTenantId != null) {
        return publicTenantId;
    }

    // 3️⃣ 都没有
    return null;
}
```

✅ JWT 优先
✅ PublicTenantFilter 兜底
✅ MP 无感
✅ 业务代码统一入口

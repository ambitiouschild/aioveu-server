## 三、✅ 100% 稳妥的「生产级最终版」（强烈推荐）

### ✅ 枚举（你这个基础上只改一行）

```java
@Getter
public enum OrderSourceEnum implements IEnum<Integer> {

    WX(1, "WX"),
    APP(2, "APP"),
    WEB(3, "WEB");

    @EnumValue
    private final Integer value;
    private final String label;

    OrderSourceEnum(Integer value, String label) {
        this.value = value;
        this.label = label;
    }

    @Override
    public Integer getValue() {
        return value;
    }
}
```

✅ `@EnumValue`：DB ↔ Java
✅ `IEnum`：MP 内部 + 扩展能力
✅ `final`：语义更严谨

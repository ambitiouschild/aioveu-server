### ✅ **全系统枚举统一用 `IEnum<Integer>`**

### 枚举终局写法

```java
public enum OrderStatusEnum implements IEnum<Integer> {

    UNPAID(0, "待付款"),
    PAID(1, "已付款");

    private final int value;

    OrderStatusEnum(int value, String desc) {
        this.value = value;
    }

    @Override
    public Integer getValue() {
        return value;
    }
}
```

## ✅ 六、⚠️ 唯一必须守住的红线（非常重要）

### ❌ 两边各自声明，参数不一致

```

// oms
```

👉 **100% 406**

✅ **解决方式只有一个：**

> **队列参数两边完全一致（或从同一份常量读）**

## ✅ 二、为什么不需要两套 Config？

### ✅ RabbitConfig 的职责只有一个

> **告诉 Spring：怎么连 MQ、怎么序列化、怎么声明队列**

它 **不关心** 队列叫什么名字。

## ✅ 四、RabbitConfig（一套就够 ✅）

```
@Configuration
@Slf4j
public class RabbitConfig {

    @Bean
    public RabbitListenerContainerFactory<?> rabbitListenerContainerFactory(
            ConnectionFactory connectionFactory) {
        SimpleRabbitListenerContainerFactory factory =
                new SimpleRabbitListenerContainerFactory();
        factory.setConnectionFactory(connectionFactory);
        factory.setMessageConverter(new Jackson2JsonMessageConverter());
        return factory;
    }

    @Bean
    public RabbitAdmin rabbitAdmin(ConnectionFactory connectionFactory) {
        RabbitAdmin admin = new RabbitAdmin(connectionFactory);
        admin.setAutoStartup(true);
        return admin;
    }
}
```

✅ **pay 用**

✅ **oms 用**

✅ **永远不用改**

## ✅ 五、队列在哪里声明？（关键点）

### ✅ 在 **各自服务里声明自己需要的队列**

------

### ✅ 支付服务（pay）

```
@Configuration
public class PayMqConfig {

    @Bean
    public Queue paymentSuccessQueue() {
        return buildDlqQueue(
            GlobalMqConstant.Payment.QUEUE_SUCCESS,
            GlobalMqConstant.Payment.DLQ
        );
    }

    @Bean
    public Queue paymentFailedQueue() {
        return buildDlqQueue(
            GlobalMqConstant.Payment.QUEUE_FAILED,
            GlobalMqConstant.Payment.DLQ
        );
    }

    @Bean
    public TopicExchange paymentExchange() {
        return new TopicExchange(GlobalMqConstant.Payment.EXCHANGE, true, false);
    }

    @Bean
    public Binding successBinding() {
        return BindingBuilder
            .bind(paymentSuccessQueue())
            .to(paymentExchange())
            .with("payment.success");
    }
}
```

------

### ✅ 订单服务（oms）

```
@Configuration
public class OrderMqConfig {

    @Bean
    public Queue paymentSuccessQueue() {
        return buildDlqQueue(
            GlobalMqConstant.Payment.QUEUE_SUCCESS,
            GlobalMqConstant.Payment.DLQ
        );
    }

    @Bean
    public Queue orderCloseQueue(
            OrderPrivateMqConstant constant) {
        return new Queue(constant.orderCloseQueue(), true);
    }
}
```

✅ **共享队列：两边都声明**

✅ **私有队列：只在 oms 声明**

## 一、RabbitMQ 对应关系（非常重要）

| RocketMQ      | RabbitMQ                         |
| ------------- | -------------------------------- |
| Topic         | Exchange                         |
| Tag           | RoutingKey                       |
| ConsumerGroup | Consumer（Spring Bean）          |
| 顺序消费      | 单队列 + 单消费者                |
| 集群消费      | 多个实例监听同一 Queue           |
| 重试          | `RepublishMessageRecoverer`/ DLQ |

------

## 二、RabbitMQ 配置（application.yml）

```
spring:
  rabbitmq:
    host: localhost
    port: 5672
    username: guest
    password: guest
    listener:
      simple:
        acknowledge-mode: manual   # 手动ACK（等价于RocketMQ重试）
        concurrency: 10
        max-concurrency: 20
        prefetch: 1
```

## ⚠️ 三、唯一一个「必须补的常量」（否则 DLQ 会乱）

你现在 **缺 DLQ 的 routingKey** ❗

### ❌ 你现在隐式用了这个

```

```

但 **语义是错误的**

------

### ✅ 正确做法（强烈建议）

```
String DLQ_ROUTING_KEY = "payment.success.queue";
```

### ✅ 队列里用：

```
args.put("x-dead-letter-routing-key", PaymentMqConstant.DLQ_ROUTING_KEY);
```

### ✅ DLQ 绑定：

```
.with(PaymentMqConstant.DLQ_ROUTING_KEY);
```

✅ **这是 RabbitMQ 官方推荐语义**

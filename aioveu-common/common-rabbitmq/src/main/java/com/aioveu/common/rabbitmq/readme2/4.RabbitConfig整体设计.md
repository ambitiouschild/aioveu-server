
### ❌ RabbitMQ **不支持这个参数**

RabbitMQ 原生没有 `x-max-retries`  
👉 这会导致：

• 队列创建失败

• 或在某些版本表现为 不可预测行为


✅ 正确做法：删掉这一行

```java
// ❌ 删掉
```

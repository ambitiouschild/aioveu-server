`com.baomidou.mybatisplus.annotation.IEnum`是 **MyBatis‑Plus 2.x / 早期 3.x** 里的接口，**现在（MP 3.5+）已经被官方废弃并移除** 了。

我先给你 **能直接用的正确导入方式**，再告诉你 **你现在应该用哪种方案**。

------

## 一、如果你项目里还能找到 IEnum（老项目）

### ✅ Maven 依赖（必须是老版本）

```
<dependency>
    <groupId>com.baomidou</groupId>
    <artifactId>mybatis-plus-core</artifactId>
    <version>3.4.3.4</version> <!-- 或更早 -->
</dependency>
```

### ✅ Java 导入方式

```
import com.baomidou.mybatisplus.annotation.IEnum;
```

### ✅ 枚举写法示例（老写法）

```
public enum PaymentChannelEnum implements IEnum<String> {

    ALIPAY("ALIPAY", "支付宝"),
    WECHAT("WECHAT", "微信支付");

    private final String code;
    private final String desc;

    PaymentChannelEnum(String code, String desc) {
        this.code = code;
        this.desc = desc;
    }

    @Override
    public String getValue() {
        return this.code;
    }
}
```

⚠️ **但这个方案不推荐新项目使用**

------

## 二、❗你现在最应该用的【官方推荐方案】

> ✅ **MyBatis‑Plus 3.5+ 不再需要 `IEnum`**

### ✅ 方式一：通用枚举（推荐 ✅✅✅）

#### 1️⃣ 枚举定义

```
public enum PaymentChannelEnum {

    ALIPAY("ALIPAY", "支付宝"),
    WECHAT("WECHAT", "微信支付"),
    UNIONPAY("UNIONPAY", "银联支付"),
    BALANCE("BALANCE", "余额支付");

    private final String code;
    private final String desc;

    PaymentChannelEnum(String code, String desc) {
        this.code = code;
        this.desc = desc;
    }

    public String getCode() {
        return code;
    }
}
```

#### 2️⃣ 实体类中使用

```
@Data
@TableName("pay_order")
public class PayOrder {

    @TableField("payment_channel")
    private PaymentChannelEnum paymentChannel;
}
```

#### 3️⃣ 配置 MP 扫描枚举（关键）

```
mybatis-plus:
  type-enums-package: com.aioveu.pay.enums
```

✅ **MP 会自动把 `ALIPAY`↔ 数据库字段互转**

### ✅ 方案 A（最推荐）：数据库存枚举名

### ✅ 方案 B：数据库存 code（如 `ALIPAY`）

#### 自定义 TypeHandler

```
@MappedTypes(PaymentChannelEnum.class)
public class PaymentChannelEnumTypeHandler
        extends BaseTypeHandler<PaymentChannelEnum> {

    @Override
    public void setNonNullParameter(PreparedStatement ps,
                                     int i,
                                     PaymentChannelEnum parameter,
                                     JdbcType jdbcType) throws SQLException {
        ps.setString(i, parameter.getCode());
    }

    @Override
    public PaymentChannelEnum getNullableResult(ResultSet rs, String columnName)
            throws SQLException {
        return PaymentChannelEnum.valueOfCode(rs.getString(columnName));
    }

    @Override
    public PaymentChannelEnum getNullableResult(ResultSet rs, int columnIndex)
            throws SQLException {
        return PaymentChannelEnum.valueOfCode(rs.getString(columnIndex));
    }

    @Override
    public PaymentChannelEnum getNullableResult(CallableStatement cs, int columnIndex)
            throws SQLException {
        return PaymentChannelEnum.valueOfCode(cs.getString(columnIndex));
    }
}
```

并在枚举中加：

```
public static PaymentChannelEnum valueOfCode(String code) {
    for (PaymentChannelEnum e : values()) {
        if (e.code.equals(code)) {
            return e;
        }
    }
    throw new IllegalArgumentException("未知支付渠道：" + code);
}
```

## 四、和 MapStruct 的完美配合 ✅（重点）

你现在 **完全不需要 IEnum** 了 ✅

### MapStruct 直接写

```
@Mapping(
    target = "paymentChannel",
    expression = "java(dto.getPaymentChannel())"
)
PayOrder toPayOrder(PaymentRequestDTO dto);
```

或（最优雅）：

```
@Mapper
public interface PayOrderConvert {
    PayOrder toPayOrder(PaymentRequestDTO dto);
}
```

只要：

```
PaymentRequestDTO.getPaymentChannel() → PaymentChannelEnum
```

✅ **自动映射，零注解**

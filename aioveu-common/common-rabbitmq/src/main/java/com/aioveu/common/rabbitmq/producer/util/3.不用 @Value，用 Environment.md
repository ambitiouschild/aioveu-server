## ✅ 一句话结论（先给你）

> ❌ 不是方法写错

## ✅ 二、你现在这段代码里发生了什么？

### ✅ 你的代码

```java
@Value("${spring.application.name:unknown}")
private String applicationName;
```

### ✅ 实际运行时

```java
applicationName == null
```

所以：

```java

→ applicationName.contains("-")
→ NullPointerException（或返回 "UNKNOWN"）
```

## ✅ 三、最常见的 3 个原因（按概率）

---

### ❌ 1️⃣ 你用的是 **new MessageIdGenerator()**

```java
MessageIdGenerator gen = new MessageIdGenerator(); // ❌
gen.generatePaymentMessageId(paymentNo);
```

👉 Spring 不会注入 `@Value`
👉 applicationName 一定是 null

✅ 必须用注入的 Bean

```java
@Autowired
private MessageIdGenerator messageIdGenerator;
```

---

### ❌ 2️⃣ 这个工具类被 **提前加载**

比如：

• static 块

• enum

• 常量初始化

• 非 Spring 管理的类


👉 Spring 还没来得及注入

---

### ❌ 3️⃣ 配置文件中没有这个 key

```yaml
spring:
  application:
    name: aioveu-pay
```

👉 但你启动时没有加载这个 profile

## ✅ 四、✅ 正确的修法（推荐你用这个）

### ✅ 方案一（最推荐）：**不用 `@Value`，用 Environment**

```java
@Component
@Slf4j
public class MessageIdGenerator {

    private final Environment environment;

    public MessageIdGenerator(Environment environment) {
        this.environment = environment;
    }

    private String getAppShortName() {
        String appName = environment.getProperty("spring.application.name", "unknown");
        if (appName.contains("-")) {
            String[] parts = appName.split("-");
            return parts[parts.length - 1].toUpperCase();
        }
        return appName.toUpperCase();
    }
}
```

✅ 100% 不踩坑
✅ 生命周期最安全

## ✅ 六、✅ 最终推荐你保留的 MessageIdGenerator（生产级）

```java
@Component
@Slf4j
public class MessageIdGenerator {

    private final Environment environment;
    private static final AtomicInteger SEQUENCE = new AtomicInteger(0);

    public MessageIdGenerator(Environment environment) {
        this.environment = environment;
    }

    public String generatePaymentMessageId(String paymentNo) {
        String app = getAppShortName();
        String time = LocalDateTime.now()
                .format(DateTimeFormatter.ofPattern("yyyyMMddHHmmssSSS"));
        int seq = SEQUENCE.incrementAndGet() % 10000;
        int rnd = ThreadLocalRandom.current().nextInt(100, 999);

        return String.format("PAY_%s_%s_%s_%04d_%d",
                paymentNo, app, time, seq, rnd);
    }

    private String getAppShortName() {
        String appName = environment.getProperty("spring.application.name", "unknown");
        if (appName.contains("-")) {
            return appName.substring(appName.lastIndexOf('-') + 1).toUpperCase();
        }
        return appName.toUpperCase();
    }
}
```

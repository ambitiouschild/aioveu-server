## 二、你现在这个接口，白名单怎么加（标准做法）

### ✅ 方案一（✅ 推荐）：URL 级别白名单

```java
private boolean isInternalApi(String path) {
    return path.startsWith("/inner/");
}
```

Filter 里：

```java
if (isInternalApi(path)) {
    return chain.filter(exchange);
}
```

✅ 一劳永逸  
✅ 所有 `/inner/**` 自动放行  
✅ 不用逐个接口配

### ✅ 方案二：精确白名单（更保守）

```java
private static final Set<String> WHITE_LIST = Set.of(
    "/inner/tenant/tenant-id-by-client-id"

private boolean isWhiteList(String path) {
    return WHITE_LIST.contains(path);
}
```

✅ 可控  
✅ 审计友好  
❌ 新增接口要改代码

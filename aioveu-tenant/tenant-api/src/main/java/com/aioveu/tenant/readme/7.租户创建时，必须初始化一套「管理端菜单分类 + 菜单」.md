## 二、为什么一定要初始化（3 个不能回避的原因）

### 1️⃣ 租户不会配（现实）

> 新商户刚入驻：
>
> - 
>
>   不知道什么是“菜单分类”
>
> - 
>
>   不知道哪些菜单该开
>
> - 
>
>   只想“能下单、能看报表”

你不初始化 = 后台是空的 = 工单爆炸

------

### 2️⃣ 权限模型必须有一个“基准”

你后面一定有：

- 

  角色

- 

  权限点

- 

  菜单可见性

如果没有初始菜单：

```
角色 → 没有菜单 → 权限无意义
```

------

### 3️⃣ 平台升级必须可控

> 平台新增一个菜单：
>
> - 
>
>   老租户要不要？
>
> - 
>
>   新租户要不要？

✅ 有初始模板 = 平台可控制演化

❌ 每个租户自己建 = 永远对账不清

## 五、租户创建时做什么（核心流程）

### ✅ 租户注册 / 开通流程

```
创建租户
↓
copy platform_manager_menu_category_template
↓
copy platform_manager_menu_template
↓
关联 tenant_id
↓
创建默认角色（admin / operator）
↓
绑定菜单
```

### ✅ 示例伪代码

```
@Transactional
public void initTenantManagerMenus(Long tenantId) {

    // 1. 初始化分类
    List<PlatformMenuCategoryTemplate> categories =
            platformMenuCategoryTemplateService.list();

    Map<String, Long> categoryIdMap = new HashMap<>();

    for (PlatformMenuCategoryTemplate tpl : categories) {
        ManagerMenuCategory category = new ManagerMenuCategory();
        category.setTenantId(tenantId);
        category.setCode(tpl.getCode());
        category.setName(tpl.getName());
        category.setSort(tpl.getSort());
        managerMenuCategoryService.save(category);
        categoryIdMap.put(tpl.getCode(), category.getId());
    }

    // 2. 初始化菜单
    List<PlatformMenuTemplate> menus =
            platformMenuTemplateService.list();

    for (PlatformMenuTemplate tpl : menus) {
        ManagerMenu menu = new ManagerMenu();
        menu.setTenantId(tenantId);
        menu.setCategoryId(categoryIdMap.get(tpl.getCategoryCode()));
        menu.setMenuCode(tpl.getMenuCode());
        menu.setName(tpl.getMenuName());
        menu.setRoute(tpl.getRoute());
        menu.setPermissionCode(tpl.getPermissionCode());
        menu.setSort(tpl.getSort());
        managerMenuService.save(menu);
    }
}
```

## 六、租户能不能改？（非常关键）

### ✅ 允许租户做的事

| 操作           | 是否允许  |
| -------------- | --------- |
| 隐藏菜单       | ✅         |
| 重命名菜单     | ✅         |
| 调整排序       | ✅         |
| 新增自定义菜单 | ✅（慎用） |

### ❌ 不允许的事

| 操作                 | 原因         |
| -------------------- | ------------ |
| 删除平台菜单         | 权限模型会坏 |
| 修改 permission_code | 权限校验会炸 |

✅ 正确做法是：

**软隐藏 + 自定义副本**

## 一、你现在这套模型的真实含义

### 1️⃣ `oauth2_registered_client`
👉 协议级开关（能不能用）

• client_id 是否存在

• client_secret 对不对

• 是否允许 password / authorization_code / refresh_token

• token 有效期


✅ Spring Authorization Server 说了算

---

### 2️⃣ `oauth_client_wx_app`
👉 业务级开关（给不给这个租户用）

• 这个 client 属于哪个租户

• 是不是微信小程序

• 当前租户是否被禁用

• 是否在灰度 / 渠道 / 版本内


✅ 你业务说了算

## 二、为什么说这是“多加了一层开关”

### ✅ 没有 `OauthClientWxApp` 之前

```text
clientId = xinhuan-app
↓
只能知道：这是一个合法 OAuth2 Client
↓
不知道：是谁在用、在哪个租户、能不能用
```

### ✅ 有了 `OauthClientWxApp` 之后

```text
clientId = xinhuan-app
↓
tenantId = 3
tenantName = 可我不敌心欢
enabled = 1
wxAppid = wxe0e6720ed2db3937
↓
✅ 知道“谁在用”
✅ 知道“能不能用”
✅ 知道“业务上下文”
```

👉 这就是那一层开关

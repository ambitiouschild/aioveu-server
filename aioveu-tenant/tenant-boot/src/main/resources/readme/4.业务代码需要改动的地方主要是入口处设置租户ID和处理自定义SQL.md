## 需要修改的地方：

### 1. **Controller层**：设置租户ID

```
@RestController
@RequestMapping("/api/user")
public class UserController {
    
    @GetMapping("/list")
    public Result listUsers() {
        // 从请求中获取租户ID（从Token、Header、Session等）
        Long tenantId = getTenantIdFromRequest();
        
        // 设置到租户上下文
        TenantContextHolder.setTenantId(tenantId);
        
        // 调用Service
        return userService.list();
    }
    
    private Long getTenantIdFromRequest() {
        // 方式1：从Header获取
        String tenantHeader = request.getHeader("X-Tenant-Id");
        if (StringUtils.isNotBlank(tenantHeader)) {
            return Long.valueOf(tenantHeader);
        }
        
        // 方式2：从Token解析
        String token = request.getHeader("Authorization");
        JwtUser jwtUser = JwtUtils.getUserFromToken(token);
        return jwtUser.getTenantId();
        
        // 方式3：从Session获取
        // 方式4：从URL参数获取
    }
}
```

### 2. **过滤器/拦截器**：统一设置租户ID（推荐）

```
@Component
public class TenantFilter implements Filter {
    
    @Override
    public void doFilter(ServletRequest request, ServletResponse response, FilterChain chain) 
            throws IOException, ServletException {
        
        HttpServletRequest req = (HttpServletRequest) request;
        
        try {
            // 1. 从请求中解析租户ID
            Long tenantId = parseTenantId(req);
            
            // 2. 设置到租户上下文
            if (tenantId != null) {
                TenantContextHolder.setTenantId(tenantId);
            } else {
                // 如果没有租户ID，可以设置默认值或抛出异常
                // TenantContextHolder.setTenantId(1L); // 默认租户
                throw new BusinessException("租户ID不能为空");
            }
            
            chain.doFilter(request, response);
        } finally {
            // 3. 清理租户上下文（重要！）
            TenantContextHolder.clear();
        }
    }
    
    private Long parseTenantId(HttpServletRequest request) {
        // 优先级：Header > Token > Session > 参数
        String tenantIdStr = request.getHeader("X-Tenant-Id");
        if (StringUtils.isNotBlank(tenantIdStr)) {
            return Long.valueOf(tenantIdStr);
        }
        
        // 从Token解析
        String token = getTokenFromRequest(request);
        if (StringUtils.isNotBlank(token)) {
            return JwtUtils.getTenantId(token);
        }
        
        return null;
    }
}
```

## 不需要修改的地方：

### 1. **Mapper层**：不需要改

MyBatis-Plus插件会自动处理SQL，你不需要在Mapper中写`WHERE tenant_id = #{tenantId}`。

## 必须检查的代码：

### 1. **自定义SQL查询**：需要手动处理

```
// MyBatis Mapper XML
<!-- ❌ 错误：没有考虑租户 -->
<select id="customQuery" resultType="User">
    SELECT * FROM user 
    WHERE status = 1 
    AND create_time > #{startTime}
</select>

<!-- ✅ 正确：手动添加租户条件 -->
<select id="customQuery" resultType="User">
    SELECT * FROM user 
    WHERE tenant_id = #{tenantId}  <!-- 必须加 -->
    AND status = 1 
    AND create_time > #{startTime}
</select>

<!-- ✅ 或者通过参数传递 -->
<select id="customQuery" resultType="User">
    SELECT * FROM user 
    WHERE tenant_id = #{tenantId}  <!-- #{tenantId} 从参数中获取 -->
    AND status = 1 
</select>
```

### 2. **联表查询**：每张表都要加租户条件

```
// ❌ 错误：只在一张表加了租户条件
<select id="queryUserWithRole" resultMap="userRoleMap">
    SELECT u.*, r.* 
    FROM user u
    LEFT JOIN role r ON u.role_id = r.id
    WHERE u.tenant_id = #{tenantId}  <!-- 只有user表有租户条件 -->
</select>

// ✅ 正确：每张业务表都加租户条件
<select id="queryUserWithRole" resultMap="userRoleMap">
    SELECT u.*, r.* 
    FROM user u
    LEFT JOIN role r ON u.role_id = r.id AND r.tenant_id = #{tenantId}  <!-- role表也要加 -->
    WHERE u.tenant_id = #{tenantId}  <!-- user表的租户条件 -->
</select>
```

### 3. **统计查询**：需要group by tenant_id

```
// ❌ 错误：统计所有租户
<select id="countByStatus" resultType="map">
    SELECT status, COUNT(*) as count
    FROM user
    GROUP BY status
</select>

// ✅ 正确：按租户统计
<select id="countByStatus" resultType="map">
    SELECT tenant_id, status, COUNT(*) as count
    FROM user
    WHERE tenant_id = #{tenantId}  <!-- 过滤当前租户 -->
    GROUP BY tenant_id, status
</select>
```

## 问题分析：

1. **过滤器已经生效**：`TenantFilter.doFilterInternal`被调用
2. **租户ID为null**：过滤器没有成功设置租户ID
3. **错误位置**：`MyTenantLineHandler.getTenantId()`第51行抛异常

## 错误原因：

你的 `SecurityUtils.getTenantId()`返回了 `null`，导致过滤器没有设置租户ID。

## 解决方案：

### 方案1：修改过滤器，从Header获取租户ID

```
@Component
public class TenantFilter extends OncePerRequestFilter {
    
    @Override
    protected void doFilterInternal(HttpServletRequest request,
                                    HttpServletResponse response,
                                    FilterChain filterChain) throws ServletException, IOException {
        
        System.out.println("租户过滤器执行，URI: " + request.getRequestURI());
        System.out.println("请求Header: " + request.getHeaderNames());
        
        try {
            // 1. 从Header获取租户ID（最直接）
            String tenantIdStr = request.getHeader("X-Tenant-Id");
            System.out.println("X-Tenant-Id Header值: " + tenantIdStr);
            
            if (tenantIdStr != null && !tenantIdStr.trim().isEmpty()) {
                try {
                    Long tenantId = Long.valueOf(tenantIdStr.trim());
                    TenantContextHolder.setTenantId(tenantId);
                    System.out.println("✅ 从Header设置租户ID: " + tenantId);
                } catch (NumberFormatException e) {
                    System.err.println("❌ 租户ID格式错误: " + tenantIdStr);
                }
            } else {
                // 2. 从Authorization Header的Token中解析
                String authHeader = request.getHeader("Authorization");
                System.out.println("Authorization Header: " + authHeader);
                
                if (authHeader != null && authHeader.startsWith("Bearer ")) {
                    String token = authHeader.substring(7);
                    Long tenantId = parseTenantIdFromToken(token);
                    
                    if (tenantId != null) {
                        TenantContextHolder.setTenantId(tenantId);
                        System.out.println("✅ 从Token解析租户ID: " + tenantId);
                    } else {
                        System.err.println("❌ 无法从Token解析租户ID");
                    }
                } else {
                    System.err.println("❌ 没有找到X-Tenant-Id Header和有效的Authorization Header");
                    
                    // 3. 如果是公开接口，可以设置默认租户
                    if (isPublicApi(request.getRequestURI())) {
                        TenantContextHolder.setTenantId(1L);  // 测试用默认租户
                        System.out.println("⚠️ 公开接口使用默认租户ID: 1");
                    } else {
                        // 非公开接口需要租户ID
                        System.err.println("❌ 非公开接口缺少租户ID: " + request.getRequestURI());
                    }
                }
            }
            
            filterChain.doFilter(request, response);
        } finally {
            TenantContextHolder.clear();
        }
    }
    
    /**
     * 从JWT Token解析租户ID
     */
    private Long parseTenantIdFromToken(String token) {
        try {
            // 使用你的JWT工具类解析
            // 假设Token包含tenantId字段
            // 这里需要实现具体的JWT解析逻辑
            
            // 临时返回测试值
            return 1L;  // 测试用
        } catch (Exception e) {
            System.err.println("解析Token失败: " + e.getMessage());
            return null;
        }
    }
    
    /**
     * 判断是否是公开接口
     */
    private boolean isPublicApi(String uri) {
        return uri.contains("/auth/") || 
               uri.contains("/public/") || 
               uri.contains("/actuator/") ||
               uri.contains("/swagger") ||
               uri.contains("/v3/api-docs") ||
               uri.contains("/doc.html");
    }
}

### 方案2：简化版本，强制要求Header

```
@Component
public class SimpleTenantFilter extends OncePerRequestFilter {
    
    @Override
    protected void doFilterInternal(HttpServletRequest request,
                                    HttpServletResponse response,
                                    FilterChain filterChain) throws ServletException, IOException {
        
        try {
            // 必须从Header获取租户ID
            String tenantIdStr = request.getHeader("X-Tenant-Id");
            
            if (tenantIdStr == null || tenantIdStr.trim().isEmpty()) {
                // 直接返回错误
                response.setStatus(HttpServletResponse.SC_BAD_REQUEST);
                response.setContentType("application/json");
                response.getWriter().write("{\"code\":400,\"message\":\"缺少租户ID (X-Tenant-Id Header)\"}");
                return;
            }
            
            try {
                Long tenantId = Long.valueOf(tenantIdStr.trim());
                TenantContextHolder.setTenantId(tenantId);
                System.out.println("设置租户ID: " + tenantId);
            } catch (NumberFormatException e) {
                response.setStatus(HttpServletResponse.SC_BAD_REQUEST);
                response.setContentType("application/json");
                response.getWriter().write("{\"code\":400,\"message\":\"租户ID格式错误\"}");
                return;
            }
            
            filterChain.doFilter(request, response);
        } finally {
            TenantContextHolder.clear();
        }
    }
}
```

### 方案3：调试SecurityUtils.getTenantId()

如果非要使用 `SecurityUtils.getTenantId()`，先调试它：

```
@Component
public class TenantFilter extends OncePerRequestFilter {
    
    @Override
    protected void doFilterInternal(HttpServletRequest request,
                                    HttpServletResponse response,
                                    FilterChain filterChain) throws ServletException, IOException {
        
        System.out.println("=== 调试SecurityUtils.getTenantId() ===");
        
        try {
            // 调试1：直接调用
            Long tenantId = SecurityUtils.getTenantId();
            System.out.println("SecurityUtils.getTenantId() 结果: " + tenantId);
            
            // 调试2：查看Spring Security上下文
            Authentication authentication = SecurityContextHolder.getContext().getAuthentication();
            System.out.println("Spring Security认证: " + authentication);
            if (authentication != null) {
                System.out.println("Principal: " + authentication.getPrincipal());
                System.out.println("是否认证: " + authentication.isAuthenticated());
            }
            
            // 调试3：查看请求Header
            System.out.println("Authorization Header: " + request.getHeader("Authorization"));
            
            if (tenantId != null) {
                TenantContextHolder.setTenantId(tenantId);
                System.out.println("✅ 从SecurityUtils设置租户ID: " + tenantId);
            } else {
                // 回退方案：从Header获取
                String tenantIdStr = request.getHeader("X-Tenant-Id");
                if (tenantIdStr != null) {
                    tenantId = Long.valueOf(tenantIdStr);
                    TenantContextHolder.setTenantId(tenantId);
                    System.out.println("✅ 从Header回退设置租户ID: " + tenantId);
                } else {
                    // 临时设置测试值
                    TenantContextHolder.setTenantId(1L);
                    System.out.println("⚠️ 使用测试租户ID: 1");
                }
            }
            
            filterChain.doFilter(request, response);
        } finally {
            TenantContextHolder.clear();
        }
    }
}

## 最可能的原因：

1. **SecurityUtils.getTenantId() 在过滤器时还未设置** - 因为认证还没完成
2. **请求没有带 X-Tenant-Id Header**
3. **Token里没有tenantId信息**

**推荐先用方案1**，从Header直接获取，这是最可靠的方式。

你说得对，不应该因为找不到租户ID就报错。很多接口确实不需要租户ID。

## 更合理的逻辑：

1. **如果有租户ID** → 设置到上下文，MyBatis-Plus自动过滤
2. **如果没有租户ID** → 不设置，MyBatis-Plus不添加租户条件
3. **让业务逻辑决定是否需要租户ID**

```
@Component
public class TenantFilter extends OncePerRequestFilter {
    
    @Override
    protected void doFilterInternal(HttpServletRequest request,
                                    HttpServletResponse response,
                                    FilterChain filterChain) throws ServletException, IOException {
        
        String uri = request.getRequestURI();
        System.out.println("租户过滤器执行: " + uri);
        
        try {
            // 尝试获取租户ID
            Long tenantId = getTenantIdFromRequest(request);
            
            if (tenantId != null) {
                // 有租户ID，设置到上下文
                TenantContextHolder.setTenantId(tenantId);
                System.out.println("✅ 设置租户ID: " + tenantId);
            } else {
                // 没有租户ID，不设置
                System.out.println("⚠️ 没有租户ID，跳过设置");
                
                // 特殊场景：如果需要查询所有租户，可以设置0
                if (isAllTenantsQuery(uri)) {
                    TenantContextHolder.setTenantId(0L);
                    System.out.println("特殊接口，设置租户ID为0（查询所有租户）");
                }
            }
            
            filterChain.doFilter(request, response);
        } finally {
            // 清理上下文
            TenantContextHolder.clear();
        }
    }
    
    private Long getTenantIdFromRequest(HttpServletRequest request) {
        // 1. 从URL参数获取
        String tenantIdParam = request.getParameter("tenantId");
        if (StringUtils.hasText(tenantIdParam)) {
            try {
                return Long.valueOf(tenantIdParam.trim());
            } catch (NumberFormatException e) {
                System.err.println("租户ID参数格式错误: " + tenantIdParam);
                return null;
            }
        }
        
        // 2. 从Header获取
        String tenantIdHeader = request.getHeader("X-Tenant-Id");
        if (StringUtils.hasText(tenantIdHeader)) {
            try {
                return Long.valueOf(tenantIdHeader.trim());
            } catch (NumberFormatException e) {
                System.err.println("租户ID Header格式错误: " + tenantIdHeader);
                return null;
            }
        }
        
        // 3. 从Token解析（可选）
        // 如果Token里一定有租户ID，就解析
        // 如果Token里可能没有，就跳过
        
        return null;  // 没有租户ID
    }
    
    private boolean isAllTenantsQuery(String uri) {
        // 明确需要查询所有租户的接口
        return uri.contains("/api/v1/users/tenants/") ||  // 查询用户可访问租户
               uri.contains("/api/v1/admin/statistics");  // 管理员统计
    }
}
```

```
@Component
public class MyTenantLineHandler implements TenantLineHandler {
    
    @Override
    public Expression getTenantId() {
        Long tenantId = TenantContextHolder.getTenantId();
        
        // 如果租户ID为null，不添加租户条件
        if (tenantId == null) {
            System.out.println("租户ID为null，不添加租户过滤条件");
            return null;
        }
        
        // 如果租户ID为0，表示查询所有租户
        if (tenantId == 0L) {
            System.out.println("租户ID为0，查询所有租户");
            return null;
        }
        
        // 正常租户ID，添加过滤条件
        System.out.println("添加租户过滤条件: tenant_id = " + tenantId);
        return new LongValue(tenantId);
    }
    
    @Override
    public String getTenantIdColumn() {
        return "tenant_id";
    }
    
    @Override
    public boolean ignoreTable(String tableName) {
        // 系统表不需要租户隔离
        return tableName.startsWith("sys_");
    }
}
```

## 这样处理的好处：

### 场景1：有租户ID的接口

```
GET /api/v1/users?tenantId=1
↓
过滤器设置 tenantId=1
↓
SQL: SELECT * FROM user WHERE tenant_id = 1
```

### 场景2：没有租户ID的接口

```
GET /api/v1/users/tenants/admin
↓
过滤器不设置 tenantId
↓
SQL: SELECT * FROM user WHERE username = 'admin'  # 查询所有租户
```

### 场景3：特殊需要查询所有租户

```
GET /api/v1/admin/statistics
↓
过滤器设置 tenantId=0
↓
SQL: SELECT * FROM user  # 查询所有租户用户
```

```
    public List<User> getAllTenantsUsers() {
        // 如果需要强制查询所有租户，可以临时清除租户上下文
        Long originalTenantId = TenantContextHolder.getTenantId();
        
        try {
            TenantContextHolder.clear();  // 清除租户条件
            return userMapper.selectList(null);  // 查询所有租户
        } finally {
            // 恢复
            if (originalTenantId != null) {
                TenantContextHolder.setTenantId(originalTenantId);
            }
        }
    }
    
    public List<User> getCurrentTenantUsers() {
        // 强制只查当前租户
        Long tenantId = TenantContextHolder.getTenantId();
        if (tenantId == null) {
            throw new RuntimeException("当前没有租户上下文");
        }
        
        return userMapper.selectList(
            new QueryWrapper<User>().eq("tenant_id", tenantId)
        );
    }
}
```

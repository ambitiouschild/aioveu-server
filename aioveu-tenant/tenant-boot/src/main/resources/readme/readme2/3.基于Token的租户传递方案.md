```
public Result<Map<String, Object>> switchTenant(@RequestParam Long newTenantId) {
    // 1. 验证权限...
    
    // 2. 获取当前用户

    result.put("token_type", "Bearer");
    result.put("expires_in", 3600);
```

### 2. 每个微服务的过滤器解析Token

```
@Component
public class TenantFromTokenFilter extends OncePerRequestFilter {
    
    @Autowired
    private JwtDecoder jwtDecoder;
    
    @Override
    protected void doFilterInternal(HttpServletRequest request, 
                                   HttpServletResponse response, 
                                   FilterChain filterChain) {
        
        try {
            // 1. 从请求中提取Token
            String token = extractToken(request);
            
            if (token != null) {
                try {
                    // 2. 解析JWT Token
                    Jwt jwt = jwtDecoder.decode(token);
                    
                    // 3. 从Token中获取租户ID
                    Long tenantId = jwt.getClaim("tenant_id");
                    
                    if (tenantId != null) {
                        // 4. 设置到ThreadLocal
                        TenantContextHolder.setTenantId(tenantId);
                        
                        // 5. 更新认证信息（如果需要）
                        updateAuthenticationTenantId(tenantId, jwt);
                    }
                    
                } catch (Exception e) {
                    log.warn("解析Token失败: {}", e.getMessage());
                }
            }
            
            filterChain.doFilter(request, response);
            
        } finally {
            TenantContextHolder.clear();
        }
    }
    
    private void updateAuthenticationTenantId(Long tenantId, Jwt jwt) {
        // 如果当前认证是JwtAuthenticationToken，更新其中的租户信息
        Authentication auth = SecurityContextHolder.getContext().getAuthentication();
        
        if (auth != null && auth.getPrincipal() instanceof Jwt) {
            // 创建包含租户ID的新JWT
            Map<String, Object> claims = new HashMap<>(jwt.getClaims());
            claims.put("tenant_id", tenantId);
            
            Jwt newJwt = new Jwt(
                jwt.getTokenValue(),
                jwt.getIssuedAt(),
                jwt.getExpiresAt(),
                jwt.getHeaders(),
                claims
            );
            
            Authentication newAuth = new JwtAuthenticationToken(
                newJwt,
                auth.getAuthorities(),
                auth.getName()
            );
            
            SecurityContextHolder.getContext().setAuthentication(newAuth);
        }
    }
}
```

### 3. MyBatis-Plus从Token获取租户ID

```
@Component
public class TenantFromTokenHandler implements TenantLineHandler {
    
    @Override
    public Expression getTenantId() {
        // 1. 从SecurityContext获取认证
        Authentication auth = SecurityContextHolder.getContext().getAuthentication();
        
        Long tenantId = null;
        
        if (auth != null && auth.getPrincipal() instanceof Jwt jwt) {
            // 2. 直接从JWT Token获取租户ID
            tenantId = jwt.getClaim("tenant_id");
        }
        
        // 3. 如果Token中没有，从ThreadLocal获取
        if (tenantId == null) {
            tenantId = TenantContextHolder.getTenantId();
        }
        
        if (tenantId == null) {
            throw new RuntimeException("无法获取租户ID");
        }
        
        return new LongValue(tenantId);
    }
}
```

## 完整的工作流程

### 切换租户流程

```
1. 前端调用 /api/auth/switch-tenant?tenantId=123
2. 后端验证权限，生成新Token（tenant_id=123）
3. 前端保存新Token
4. 后续所有请求携带新Token
5. 每个微服务解析Token获取tenant_id=123
```

## 安全性考虑

### 优势

1. **防篡改**：租户ID在Token中，有签名保护
2. **时效性**：Token有过期时间
3. **标准化**：符合JWT标准

### 需要考虑的问题

1. **Token刷新**：如果Token过期，刷新Token时也要保持租户ID
2. **并发请求**：切换租户期间，可能有旧Token的请求
3. **Token黑名单**：切换租户后，旧Token应该加入黑名单

## 完整的多租户测试示例

```
package com.aioveu;

import org.junit.jupiter.api.Test;
import org.springframework.boot.test.context.SpringBootTest;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.test.context.ActiveProfiles;
import com.aioveu.tenant.aioveu02User.service.UserService;
import com.aioveu.tenant.aioveu04Menu.service.MenuService;
import com.aioveu.tenant.context.TenantContext;
import com.aioveu.tenant.aioveu02User.model.entity.User;

import java.util.List;

@SpringBootTest
@ActiveProfiles("test")  // 使用测试配置文件
public class TenantTest {
    
    @Autowired
    private UserService userService;
    
    @Autowired
    private MenuService menuService;
    
    @Test
    public void testContextLoads() {
        // 简单测试，验证Spring上下文加载成功
        System.out.println("Spring Boot上下文加载成功！");
        System.out.println("userService: " + userService);
        System.out.println("menuService: " + menuService);
    }
    
    @Test
    public void testTenantIsolation() {
        // 测试租户1
        TenantContext.setCurrentTenantId(1L);
        List<User> users1 = userService.list();
        System.out.println("租户1用户数量: " + users1.size());
        
        // 测试租户2
        TenantContext.setCurrentTenantId(2L);
        List<User> users2 = userService.list();
        System.out.println("租户2用户数量: " + users2.size());
        
        // 验证租户数据隔离
        // 这里可以添加断言验证
    }
    
    @Test
    public void testMenuTenantIsolation() {
        // 测试菜单的租户隔离
        TenantContext.setCurrentTenantId(1L);
        var menus1 = menuService.listCurrentUserRoutes();
        System.out.println("租户1菜单数量: " + menus1.size());
        
        TenantContext.setCurrentTenantId(2L);
        var menus2 = menuService.listCurrentUserRoutes();
        System.out.println("租户2菜单数量: " + menus2.size());
    }
    
    @Test
    public void testWithoutTenant() {
        // 测试不设置租户的情况
        TenantContext.clear();
        
        try {
            userService.list();
            // 如果没有抛出异常，说明多租户插件配置有问题
            System.err.println("警告：多租户插件可能未正确配置！");
        } catch (Exception e) {
            System.out.println("正确捕获异常（未设置租户ID）：" + e.getMessage());
        }
    }
}
```

## 额外配置

### 1. 创建测试配置文件

在 `src/test/resources/`下创建 `application-test.yml`：

```
spring:
  datasource:
    url: jdbc:h2:mem:testdb;MODE=MySQL;DB_CLOSE_DELAY=-1
    driver-class-name: org.h2.Driver
    username: sa
    password: 
    
  # 使用H2内存数据库，避免影响生产数据
  h2:
    console:
      enabled: true
  
  jpa:
    hibernate:
      ddl-auto: create-drop
    show-sql: true

# 测试环境关闭一些不必要的功能
logging:
  level:
    root: INFO
    com.aioveu: DEBUG
```

### 2. 确保测试依赖

```
<!-- pom.xml -->
<dependencies>
    <!-- 测试依赖 -->
    <dependency>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-test</artifactId>
        <scope>test</scope>
    </dependency>
    
    <!-- H2数据库用于测试 -->
    <dependency>
        <groupId>com.h2database</groupId>
        <artifactId>h2</artifactId>
        <scope>test</scope>
    </dependency>
</dependencies>
```

## 执行顺序建议

1. **先运行最简单的测试**：

   ```
   @Test
   public void contextLoads() {
       // 什么都不做，只验证Spring能启动
   }
   ```

2. **验证通过后，再添加多租户测试**。

3. **如果还有问题**，检查你的 `UserService`是否被正确注入到Spring容器中。

## 调试技巧

在测试类中添加：

```
@SpringBootTest(webEnvironment = SpringBootTest.WebEnvironment.RANDOM_PORT)
```

然后在测试方法中添加：

```
@Test
public void debugBean() {
    // 查看所有Bean
    String[] beanNames = applicationContext.getBeanDefinitionNames();
    for (String beanName : beanNames) {
        System.out.println(beanName);
    }
}
```

**总结**：最简单的解决方案是**将构造方法注入改为字段注入**（`@Autowired private UserService userService;`）。这是Spring测试中最常用的方式。

@Transactional
public void handleWxNotify(PaymentNotifyDTO notify) {

    // 3. 校验订单状态

✅ **Pay 内部是强一致**

✅ **不调 Order**

✅ **不调库存**

### ✅ 2️⃣ Order 消费 MQ（最终强一致）

```
@RabbitListener
public void onPaymentSuccess(PaymentSuccessMessage msg) {

    OmsOrder order = orderMapper.selectByOrderSn(msg.getOrderSn());

    if (order.getStatus() == UNPAID) {
        order.setStatus(PAID);
        orderMapper.updateById(order);
    }
}
```

✅ **Order 内部也是强一致**

✅ **不回查 Pay**

✅ **不阻塞微信**

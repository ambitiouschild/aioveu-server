## ✅ 推荐的终局编号规则（✅ 强烈推荐）

### ✅ 格式

```
EXP + 租户ID + yyyyMMdd + 序号
```

### ✅ 示例

```
EXP100120260112000001
```

| 段       | 含义     |
| -------- | -------- |
| EXP      | 导出     |
| 1001     | 租户 ID  |
| 20260112 | 日期     |
| 000001   | 当日流水 |

## ✅ 生成器实现（✅ Redis，安全高效）

```java
public String generate(Long tenantId) {
    String date = LocalDate.now().format(DateTimeFormatter.ofPattern("yyyyMMdd"));
    String key = "export:no:" + tenantId + ":" + date;

    Long seq = redisTemplate.opsForValue().increment(key);
    if (seq == 1) {
        redisTemplate.expire(key, 1, TimeUnit.DAYS);
    }

    return "EXP" + tenantId + date + String.format("%06d", seq);
}
```

## ✅ ✅ 方案一（✅ 最推荐，最干净）

### ✅ **Converter 里用枚举解析**

```java
    PaymentMethodEnum.getLabelByValue(bo.getPaymentMethod())

    OrderSourceEnum.getLabelByValue(bo.getSource())
);

    OrderStatusEnum.getLabelByValue(bo.getStatus())
);
```

### ✅ 枚举里加这个（必须）

```java
public enum PaymentMethodEnum implements IEnum<Integer> {

    WECHAT_JSAPI(1, "微信JSAPI");

    private final int value;

    public static String getLabelByValue(Integer value) {
        if (value == null) return "未知";
        for (PaymentMethodEnum e : values()) {
            if (e.getValue().equals(value)) {
                return e.getLabel();
            }
        }
        return "未知";
    }
}
```

✅ 最稳
✅ 最清晰
✅ Seata / MP / MapStruct 全兼容

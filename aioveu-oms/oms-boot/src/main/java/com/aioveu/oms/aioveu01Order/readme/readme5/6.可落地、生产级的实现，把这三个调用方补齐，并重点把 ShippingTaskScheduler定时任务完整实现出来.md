# 二、✅ OrderController（用户点击发货）

```
@RestController
@RequestMapping("/orders")
@Slf4j
public class OrderController {

    @Autowired
    private WeChatShippingService weChatShippingService;

    @Autowired
    private OrderService orderService;

    /**
     * 手动发货（商家后台 / 小程序）
     */
    @PostMapping("/{orderSn}/ship")
    public JsonNode ship(@PathVariable String orderSn) {

        log.info("【发货】手动发货 orderSn={}", orderSn);

        JsonNode result = weChatShippingService.uploadShipping(orderSn);

        if (result.has("errcode") && result.get("errcode").asInt() == 0) {
            orderService.markAsShipped(orderSn);
        }

        return result;
    }
}
```

# 三、✅ OrderService（状态机自动发货）

```
@Service
@Slf4j
public class OrderServiceImpl implements OrderService {

    @Autowired
    private WeChatShippingService weChatShippingService;

    @Autowired
    private OrderMapper orderMapper;

    /**
     * 状态变为【已发货】
     */
    @Override
    @Transactional
    public void markAsShipped(String orderSn) {

        OmsOrder order = orderMapper.selectByOrderSn(orderSn);

        if (order == null) {
            throw new BusinessException("订单不存在");
        }

        if (!OrderStatus.PAID.equals(order.getStatus())) {
            throw new BusinessException("订单状态异常");
        }

        // 1️⃣ 微信发货
        JsonNode json = weChatShippingService.uploadShipping(orderSn);

        if (json.has("errcode") && json.get("errcode").asInt() == 0) {

            // 2️⃣ 本地状态变更
            order.setStatus(OrderStatus.SHIPPED);
            orderMapper.updateById(order);

            log.info("✅ 订单发货成功 orderSn={}", orderSn);
        } else {
            throw new BusinessException("微信发货失败");
        }
    }
}
```

✅ **事务保护**

# 四、✅ ShippingTaskScheduler（⭐重点：定时兜底）

| 自动兜底 | 防止人工漏点发货   |
| 幂等     | 同一订单只发一次   |
| 容错     | 微信失败不影响下次 |
| 可监控   | 日志清晰           |

### ✅ 定时任务完整实现

```
@Component
@Slf4j
public class ShippingTaskScheduler {

    @Autowired
    private WeChatShippingService weChatShippingService;

    @Autowired
    private OrderMapper orderMapper;

    /**
     * 每 5 分钟执行一次
     */
    @Scheduled(cron = "0 */5 * * * ?")
    public void autoUploadShipping() {

        log.info("【定时任务】开始自动上传发货信息");

        // 1️⃣ 查询【已支付但未上传发货】的订单
        List<OmsOrder> orders =
                orderMapper.listPaidButNotShipped();

        if (CollUtil.isEmpty(orders)) {
            log.info("【定时任务】暂无需要上传发货的订单");
            return;
        }

        for (OmsOrder order : orders) {

            String orderSn = order.getOrderSn();

            try {
                log.info("【定时任务】准备上传发货 orderSn={}", orderSn);

                JsonNode result =
                        weChatShippingService.uploadShipping(orderSn);

                if (result.has("errcode")
                        && result.get("errcode").asInt() == 0) {

                    order.setStatus(OrderStatus.SHIPPED);
                    orderMapper.updateById(order);

                    log.info("✅ 定时任务发货成功 orderSn={}", orderSn);
                } else {
                    log.warn("❌ 微信发货失败 orderSn={}, errmsg={}",
                            orderSn, result.get("errmsg"));
                }

            } catch (Exception e) {
                log.error("❌ 定时任务发货异常 orderSn=" + orderSn, e);
            }
        }
    }
}
```

✅ **非阻塞**

✅ **异常隔离**

✅ **可恢复**

# ✅ 二、ClientContext（核心）

```java
package com.aioveu.common.context;

import lombok.extern.slf4j.Slf4j;

/**
 * 客户端上下文（ThreadLocal）
 */
@Slf4j
public class ClientContext {

    private static final ThreadLocal<String> CLIENT_ID
            = new ThreadLocal<>();

    public static void setClientId(String clientId) {
        CLIENT_ID.set(clientId);
        log.debug("ClientContext set clientId = {}", clientId);
    }

    public static String getClientId() {
        return CLIENT_ID.get();
    }

    public static void clear() {
        CLIENT_ID.remove();
        log.debug("ClientContext cleared");
    }
}
```

✅ 轻量
✅ 无第三方依赖
✅ 可放 common

# ✅ 三、Web 拦截器（订单 / 网关 / 公共接口）

```java
package com.aioveu.common.context;

import jakarta.servlet.http.HttpServletRequest;
import jakarta.servlet.http.HttpServletResponse;
import org.springframework.web.servlet.HandlerInterceptor;

/**
 * Web MVC 拦截器
 */
public class ClientContextInterceptor implements HandlerInterceptor {

    private static final String HEADER_CLIENT_ID = "X-Client-Id";

    @Override
    public boolean preHandle(HttpServletRequest request,
                             HttpServletResponse response,
                             Object handler) {

        String clientId = request.getHeader(HEADER_CLIENT_ID);
        ClientContext.setClientId(clientId);
        return true;
    }

    @Override
    public void afterCompletion(HttpServletRequest request,
                                HttpServletResponse response,
                                Object handler,
                                Exception ex) {
        ClientContext.clear();
    }
}
```

✅ 自动清理 ThreadLocal（防内存泄漏）

# ✅ 四、Feign 拦截器（微服务透传 ✅）

> ⚠️ 即使你现在不用，将来一定用得上

```java
package com.aioveu.common.context;

import feign.RequestInterceptor;
import feign.RequestTemplate;
import lombok.extern.slf4j.Slf4j;

/**
 * Feign 请求拦截器（透传 clientId）
 */
@Slf4j
public class ClientContextFeignInterceptor implements RequestInterceptor {

    private static final String HEADER_CLIENT_ID = "X-Client-Id";

    @Override
    public void apply(RequestTemplate template) {

        String clientId = ClientContext.getClientId();
        if (clientId != null) {
            template.header(HEADER_CLIENT_ID, clientId);
            log.debug("Feign 透传 clientId = {}", clientId);
        }
    }
}
```

✅ 订单 → 库存 → 支付
✅ 链路不断

# ✅ 五、订单微服务如何使用（示例）

### 1️⃣ 注册拦截器

```java
@Configuration
public class WebMvcConfig implements WebMvcConfigurer {

    @Override
    public void addInterceptors(InterceptorRegistry registry) {
        registry.addInterceptor(new ClientContextInterceptor())
                .addPathPatterns("/**");
    }
}
```

### 2️⃣ Service 使用

```java
String clientId = ClientContext.getClientId();
if (StringUtils.isBlank(clientId)) {
    throw new BusinessException("非法请求");
}
```

### ✅ 2️⃣ 通用告警工具类（推荐）

```
@Component
@Slf4j
public class DingTalkAlarmSender {

    @Value("${alarm.dingtalk.webhook}")
    private String webhookUrl;

    public void send(String title, String content) {
        try {
            String body = buildMarkdown(title, content);
            RestTemplate restTemplate = new RestTemplate();
            restTemplate.postForEntity(webhookUrl, body, String.class);
        } catch (Exception e) {
            log.error("【告警】发送钉钉失败", e);
        }
    }

    private String buildMarkdown(String title, String content) {
        return """
        {
          "msgtype": "markdown",
          "markdown": {
            "title": "%s",
            "text": "%s"
          }
        }
        """.formatted(title, content);
    }
}
```

------

### ✅ 3️⃣ DLQ 告警接入

private void sendAlert(String queueName, int count) {

    - **环境**：PROD
    - **时间**：`%s`
    """.formatted(queueName, count, LocalDateTime.now());

    dingTalkAlarmSender.send("MQ 死信告警", content);
}
```

# ✅ 三、Prometheus + Grafana（指标级监控）

> **目标：不是“有没有告警”，而是“趋势和容量”**

------

## ✅ 1️⃣ 暴露指标（Micrometer）

```
@Scheduled(fixedDelay = 60_000)
public void monitorMetrics() {

    for (String queue : monitorQueues) {
        int count = rabbitTemplate.execute(
            channel -> channel.queueDeclarePassive(queue).getMessageCount()
        );

        Metrics.gauge(
            "rabbitmq_queue_messages",
            Tags.of("queue", queue),
            count
        );
    }
}
```

------

## ✅ 2️⃣ Prometheus 规则

```
groups:
- name: mq-alert
  rules:
  - alert: RabbitMQDlqAlert
    expr: rabbitmq_queue_messages{queue=~".*dlq.*"} > 0
    for: 1m
    labels:
      severity: critical
    annotations:
      summary: "MQ 死信队列告警"
```

------

## ✅ 3️⃣ Grafana Panel

```
rate(rabbitmq_queue_messages[1m])

## 四、✅ 推荐重构后的“标准写法”

### ✅ 1️⃣ 先根据「支付方式」决定技术参数

```java
private PaymentRequestDTO buildPaymentRequest(
        OmsOrder order,
        PaymentChannelEnum channel,
        PaymentMethodEnum method,
        Long memberId,
        String openId) {

    PaymentRequestDTO request = new PaymentRequestDTO();

    // ✅ 业务语义
    request.setUserId(memberId);
    request.setBizType(PaymentBizTypeEnum.ORDER_PAY);
    request.setOrderNo(order.getOrderSn());
    request.setAmount(toBigDecimal(order.getPaymentAmount()));

    request.setSubject("商品购买");
    request.setBody("订单号：" + order.getOrderSn());

    // ✅ 渠道
    request.setChannel(channel.name());

    // ✅ 支付方式（不再写死）
    request.setPayType(method.name());

    // ✅ 根据支付方式决定参数
    Map<String, Object> extraParams = new HashMap<>();

    switch (method) {
        case JSAPI:
            if (StringUtils.isBlank(openId)) {
                throw new BizException("JSAPI 支付必须传入 openId");
            }
            extraParams.put("openId", openId);
            extraParams.put("appId", getAppIdByChannel(channel, method));
            break;

        case APP:
            extraParams.put("appId", getAppIdByChannel(channel, method));
            break;

        case H5:
            extraParams.put("appId", getAppIdByChannel(channel, method));
            extraParams.put("sceneInfo", buildSceneInfo());
            break;

        case NATIVE:
            extraParams.put("appId", getAppIdByChannel(channel, method));
            break;

        case BALANCE:
            // 余额支付不走三方
            break;

        default:
            throw new BizException("不支持的支付方式: " + method);
    }

    request.setExtraParams(extraParams);
    return request;
}
```

### ✅ 2️⃣ `getAppIdByChannel`：渠道 × 方式 二维决策

```java
private String getAppIdByChannel(PaymentChannelEnum channel,
                                 PaymentMethodEnum method) {

    if (channel == PaymentChannelEnum.WECHAT) {
        return switch (method) {
            case JSAPI -> wxJsapiAppId;
            case APP -> wxAppAppId;
            case H5 -> wxH5AppId;
            default -> throw new BizException("微信不支持该支付方式");
        };
    }

    if (channel == PaymentChannelEnum.ALIPAY) {
        return aliAppId;
    }

    return null;
}
```

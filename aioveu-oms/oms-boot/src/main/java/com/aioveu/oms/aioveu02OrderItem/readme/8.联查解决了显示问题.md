## 方案二：创建专门的VO对象

```
// OmsOrderItemWithOrderVO.java
@Data
public class OmsOrderItemVO {
    // 订单商品基本信息
    private Long id;
    private Long orderId;
    private String spuName;
    private String skuSn;
    private String skuName;
    private String picUrl;
    private BigDecimal price;
    private Integer quantity;
    private BigDecimal totalAmount;
    private Integer deleted;
    private Date createTime;
    private Date updateTime;
    
    // 订单信息（联查得到）
    private String orderNo;           // 订单号
    private String customerName;      // 客户姓名
    private Integer orderStatus;      // 订单状态
    private Integer payStatus;       // 支付状态
    private BigDecimal orderTotalAmount; // 订单总金额
    private Date orderTime;          // 订单时间
    
    // 状态显示文本（可以在VO中处理，避免前端重复逻辑）
    public String getOrderStatusText() {
        if (orderStatus == null) return "";
        switch (orderStatus) {
            case 0: return "待付款";
            case 1: return "待发货";
            case 2: return "已发货";
            case 3: return "已完成";
            case 4: return "已关闭";
            default: return "未知";
        }
    }
    
    public String getPayStatusText() {
        if (payStatus == null) return "";
        switch (payStatus) {
            case 0: return "未支付";
            case 1: return "已支付";
            case 2: return "支付失败";
            default: return "未知";
        }
    }
}
```

## 方案四：Service层实现

```
// OmsOrderItemServiceImpl.java
@Service
@RequiredArgsConstructor
public class OmsOrderItemServiceImpl implements OmsOrderItemService {
    
    private final OmsOrderItemMapper orderItemMapper;
    
    @Override
    public PageResult<OmsOrderItemVO> getOrderItemPage(OmsOrderItemQueryDTO queryDTO) {
        PageHelper.startPage(queryDTO.getPageNum(), queryDTO.getPageSize());
        
        // 如果有排序参数
        if (StringUtils.isNotBlank(queryDTO.getSortField())) {
            String orderBy = queryDTO.getSortField() + " " + queryDTO.getSortOrder();
            PageHelper.orderBy(orderBy);
        }
        
        List<OmsOrderItemVO> list = orderItemMapper.getOmsOrderItemPage(queryDTO);
        PageInfo<OmsOrderItemVO> pageInfo = new PageInfo<>(list);
        
        return PageResult.of(pageInfo);
    }
    
    @Override
    public List<OmsOrderItemVO> getOrderItemsByOrderId(Long orderId) {
        // 单个订单的商品
        OmsOrderItemQueryDTO queryDTO = new OmsOrderItemQueryDTO();
        // 可以扩展查询条件
        return orderItemMapper.getOmsOrderItemPage(queryDTO);
    }
    
    @Override
    public Map<Long, List<OmsOrderItemVO>> batchGetOrderItemsByOrderIds(List<Long> orderIds) {
        if (CollectionUtils.isEmpty(orderIds)) {
            return new HashMap<>();
        }
        
        // 批量查询
        List<OmsOrderItemVO> items = orderItemMapper.batchGetOrderItemsByOrderIds(orderIds);
        
        // 按orderId分组
        return items.stream()
                .collect(Collectors.groupingBy(OmsOrderItemVO::getOrderId));
    }
}
```

## 方案五：Controller层

```
// OmsOrderItemController.java
@RestController
@RequestMapping("/api/order-items")
@RequiredArgsConstructor
public class OmsOrderItemController {
    
    private final OmsOrderItemService orderItemService;
    
    @GetMapping("/page")
    public ApiResult<PageResult<OmsOrderItemVO>> getOrderItemPage(OmsOrderItemQueryDTO queryDTO) {
        PageResult<OmsOrderItemVO> pageResult = orderItemService.getOrderItemPage(queryDTO);
        return ApiResult.success(pageResult);
    }
    
    @GetMapping("/{id}")
    public ApiResult<OmsOrderItemVO> getOrderItemById(@PathVariable Long id) {
        OmsOrderItemVO orderItem = orderItemService.getOrderItemWithOrderInfo(id);
        return ApiResult.success(orderItem);
    }
    
    @GetMapping("/by-order/{orderId}")
    public ApiResult<List<OmsOrderItemVO>> getOrderItemsByOrderId(@PathVariable Long orderId) {
        List<OmsOrderItemVO> items = orderItemService.getOrderItemsByOrderId(orderId);
        return ApiResult.success(items);
    }
    
    @PostMapping("/batch-by-orders")
    public ApiResult<Map<Long, List<OmsOrderItemVO>>> batchGetOrderItemsByOrderIds(
            @RequestBody List<Long> orderIds) {
        Map<Long, List<OmsOrderItemVO>> result = orderItemService.batchGetOrderItemsByOrderIds(orderIds);
        return ApiResult.success(result);
    }
}
```

## 关键点总结

1. **联查解决了显示问题**：通过`LEFT JOIN`一次性获取订单号和其他订单信息
2. **支持订单号搜索**：可以在WHERE条件中按`o.order_no`搜索
3. **解决了N+1问题**：通过批量查询和联查避免多次查询
4. **前端直接显示**：无需前端映射，后端已返回订单号
5. **新增/编辑时用选择器**：表单中让用户选择订单，返回ID

这样设计后，你的查询页面可以直接按订单号搜索，表格中直接显示订单号，完全满足你的需求！

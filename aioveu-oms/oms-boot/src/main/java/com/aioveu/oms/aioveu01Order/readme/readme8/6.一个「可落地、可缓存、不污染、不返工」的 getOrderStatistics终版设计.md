# 四、Service 层（重点：只做“转换 + 组装”）

## ✅ `getOrderStatistics`（✅ 终局版）

```
@Override
public OrderStatisticsVO getOrderStatistics(OrderPageQuery queryParams) {

    OrderStatisticsVO result = new OrderStatisticsVO();

    // 1. 状态统计（数据库是数字）
    Map<Integer, Integer> dbStatusCounts =
            this.baseMapper.getOrderStatusCounts(queryParams);

    Map<String, Integer> statusCounts = new HashMap<>();

    int total = 0;
    for (Map.Entry<Integer, Integer> entry : dbStatusCounts.entrySet()) {

        OrderStatusEnum statusEnum = OrderStatusEnum.of(entry.getKey());
        statusCounts.put(statusEnum.name(), entry.getValue());

        total += entry.getValue();
    }

    result.setStatusCounts(statusCounts);
    result.setTotal(total);

    // 2. 今日订单数
    result.setTodayOrderCount(
            Optional.ofNullable(this.baseMapper.countTodayOrders(queryParams)).orElse(0)
    );

    // 3. 待处理订单数
    result.setPendingCount(
            Optional.ofNullable(this.baseMapper.countPendingOrders(queryParams)).orElse(0)
    );

    // 4. 今日收入
    result.setTodayIncome(
            Optional.ofNullable(this.baseMapper.sumTodayIncome(queryParams)).orElse(0L)
    );

    log.info("订单统计结果: {}", result);
    return result;
}
```

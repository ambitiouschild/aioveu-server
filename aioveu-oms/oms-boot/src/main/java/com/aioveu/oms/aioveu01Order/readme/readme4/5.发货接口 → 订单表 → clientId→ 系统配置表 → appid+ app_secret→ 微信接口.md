## 三、✅ 关键代码实现（发货时获取凭据）

### 1️⃣ 发货 Service 核心逻辑

```
@Service
@RequiredArgsConstructor
public class OrderShippingServiceImpl implements OrderShippingService {

    private final OmsOrderMapper orderMapper;
    private final SysOAuth2ClientWxAppMapper wxAppMapper;
    private final WeChatApiClient weChatApiClient;

    @Override
    public void uploadShipping(String orderSn) {

        // 1. 查订单（唯一真理源）
        OmsOrder order = orderMapper.selectOne(
                new LambdaQueryWrapper<OmsOrder>()
                        .eq(OmsOrder::getOrderSn, orderSn)
        );

        if (order == null) {
            throw new RuntimeException("订单不存在");
        }

        // 2. 从订单取 clientId（✅ 绝不从前端传）
        String clientId = order.getClientId();

        // 3. 通过 clientId 查微信配置
        SysOAuth2ClientWxApp wxApp =
                wxAppMapper.selectOne(
                        new LambdaQueryWrapper<SysOAuth2ClientWxApp>()
                                .eq(SysOAuth2ClientWxApp::getClientId, clientId)
                                .eq(SysOAuth2ClientWxApp::getEnabled, 1)
                                .eq(SysOAuth2ClientWxApp::getIsDeleted, 0)
                );

        if (wxApp == null) {
            throw new RuntimeException("微信应用配置不存在或未启用");
        }

        // 4. 组装参数
        ObjectNode body = buildShippingBody(order);

        // 5. 调用微信（✅ 内部用 appid + secret 换 token）
        weChatApiClient.uploadShippingInfo(
                wxApp.getAppid(),
                wxApp.getAppSecret(),
                body
        );
    }
}
```

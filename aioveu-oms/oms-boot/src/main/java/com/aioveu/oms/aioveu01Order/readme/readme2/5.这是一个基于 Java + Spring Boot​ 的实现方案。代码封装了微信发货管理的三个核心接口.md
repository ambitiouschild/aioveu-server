这是一个基于 **Java + Spring Boot** 的实现方案。代码封装了微信发货管理的三个核心接口，并采用了服务层（Service）的模式，方便你直接集成到现有的商城系统中。

### 1. Maven 依赖配置

首先确保你的 `pom.xml`中包含了 HTTP 客户端和 JSON 处理库（Spring Boot 项目通常自带）：

```
<dependencies>
    <dependency>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-web</artifactId>
    </dependency>
    <dependency>
        <groupId>com.fasterxml.jackson.core</groupId>
        <artifactId>jackson-databind</artifactId>
    </dependency>
</dependencies>
```

### 2. 核心工具类 (Token & HTTP)

这部分负责获取 `access_token`和发送 POST 请求。

```
import com.fasterxml.jackson.databind.JsonNode;
import com.fasterxml.jackson.databind.ObjectMapper;
import org.springframework.beans.factory.annotation.Value;
import org.springframework.stereotype.Component;
import org.springframework.web.client.RestTemplate;

@Component
public class WeChatApiClient {

    @Value("${wechat.miniapp.appid}")
    private String appid;

    @Value("${wechat.miniapp.secret}")
    private String secret;

    private final RestTemplate restTemplate = new RestTemplate();
    private final ObjectMapper objectMapper = new ObjectMapper();

    private static final String TOKEN_URL = "https://api.weixin.qq.com/cgi-bin/token?grant_type=client_credential&appid=%s&secret=%s";
    private static final String UPLOAD_SHIPPING_URL = "https://api.weixin.qq.com/wxa/sec/order/upload_shipping_info?access_token=%s";
    private static final String NOTIFY_RECEIVE_URL = "https://api.weixin.qq.com/wxa/sec/order/notify_confirm_receive?access_token=%s";
    private static final String GET_ORDER_URL = "https://api.weixin.qq.com/wxa/sec/order/get_order?access_token=%s";

    /**
     * 获取 AccessToken (生产环境请务必加入 Redis 缓存)
     */
    public String getAccessToken() {
        String url = String.format(TOKEN_URL, appid, secret);
        String response = restTemplate.getForObject(url, String.class);
        try {
            JsonNode node = objectMapper.readTree(response);
            return node.get("access_token").asText();
        } catch (Exception e) {
            throw new RuntimeException("获取微信 AccessToken 失败: " + response);
        }
    }

    /**
     * 通用 POST 请求方法
     */
    public JsonNode postRequest(String url, Object body) {
        String response = restTemplate.postForObject(url, body, String.class);
        try {
            return objectMapper.readTree(response);
        } catch (Exception e) {
            throw new RuntimeException("微信接口调用失败: " + response);
        }
    }

    // 封装接口地址
    public JsonNode uploadShippingInfo(Object body) {
        return postRequest(String.format(UPLOAD_SHIPPING_URL, getAccessToken()), body);
    }

    public JsonNode notifyConfirmReceive(Object body) {
        return postRequest(String.format(NOTIFY_RECEIVE_URL, getAccessToken()), body);
    }

    public JsonNode getOrderStatus(Object body) {
        return postRequest(String.format(GET_ORDER_URL, getAccessToken()), body);
    }
}
```

### 3. 业务 Service 实现

这里是具体的业务逻辑，对应你刚才问的三个功能。

```
import com.fasterxml.jackson.databind.JsonNode;
import com.fasterxml.jackson.databind.node.ArrayNode;
import com.fasterxml.jackson.databind.node.ObjectNode;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.stereotype.Service;

@Service
public class WeChatShippingService {

    @Autowired
    private WeChatApiClient weChatApiClient;

    /**
     * 1. 录入发货信息
     * @param transactionId 微信支付单号
     * @param openid 用户openid
     * @param expressNo 快递单号
     * @param expressCode 快递公司编码 (如 SF, YTO)
     * @param desc 商品描述
     */
    public JsonNode uploadShipping(String transactionId, String openid, String expressNo, String expressCode, String desc) {
        ObjectNode body = weChatApiClient.objectMapper.createObjectNode();

        // order_key
        ObjectNode orderKey = body.putObject("order_key");
        orderKey.put("order_number_type", 2); // 2: 微信支付单号
        orderKey.put("transaction_id", transactionId);

        body.put("logistics_type", 1); // 1: 实体物流
        body.put("delivery_mode", 1);  // 1: 统一发货

        // shipping_list
        ArrayNode shippingList = body.putArray("shipping_list");
        ObjectNode shippingItem = shippingList.addObject();
        shippingItem.put("tracking_no", expressNo);
        shippingItem.put("express_company", expressCode);
        shippingItem.put("item_desc", desc);

        // time & payer
        body.put("upload_time", java.time.OffsetDateTime.now().toString());
        ObjectNode payer = body.putObject("payer");
        payer.put("openid", openid);

        return weChatApiClient.uploadShippingInfo(body);
    }

    /**
     * 2. 提醒用户确认收货
     * @param transactionId 微信支付单号
     */
    public JsonNode notifyConfirmReceive(String transactionId) {
        ObjectNode body = weChatApiClient.objectMapper.createObjectNode();

        ObjectNode orderKey = body.putObject("order_key");
        orderKey.put("order_number_type", 2);
        orderKey.put("transaction_id", transactionId);

        // 秒级时间戳
        body.put("received_time", System.currentTimeMillis() / 1000);

        return weChatApiClient.notifyConfirmReceive(body);
    }

    /**
     * 3. 查询订单状态
     * @param transactionId 微信支付单号
     */
    public JsonNode queryOrder(String transactionId) {
        ObjectNode body = weChatApiClient.objectMapper.createObjectNode();
        body.put("transaction_id", transactionId);
        return weChatApiClient.getOrderStatus(body);
    }
}
```

### 5. 避坑指南（Java 版）

| 注意点           | 说明                                                         |
| ---------------- | ------------------------------------------------------------ |
| **Token 缓存**   | 示例代码中 `getAccessToken()`是每次都请求。**生产环境必须缓存**，微信限制每天最多 2000 次请求，且有效期 7200 秒。 |
| **时间格式**     | `upload_time`必须是 RFC3339 格式（如 `2026-06-04T12:00:00+08:00`），Java 8 的 `OffsetDateTime.now()`可以直接用。 |
| **RestTemplate** | 如果项目是 Spring Boot 3.x，建议改用 `WebClient`。           |
| **异常处理**     | 微信接口返回 `errcode != 0`时，代码会抛出异常。建议捕获后存入数据库日志，方便排查为什么同步失败。 |

这套代码可以直接复制到你的 Controller 中调用，实现你“以自有系统为主，同步微信为辅”的需求。

## 五、你这个 Controller 的一个“架构级问题”⚠️

### ❌ 现在的问题

```
Object paymentResult = orderService.payOrder(paymentForm);
return Result.success(paymentResult);
```

❌ **返回 Object 是灾难**

- 微信：返回 `Map / WxPayMpOrderResult`
- 余额：返回 `Boolean`
- 前端不知道怎么解析

------

### ✅ 正确做法：统一支付返回 DTO

```
public class PayOrderResultDTO {
    private PaymentMethodEnum method;
    private Boolean paid;              // 余额支付
    private Map<String, Object> params; // 微信/支付宝
}
@PostMapping("/payment")
public Result<PayOrderResultDTO> payOrder(...)
```

1. 接参数

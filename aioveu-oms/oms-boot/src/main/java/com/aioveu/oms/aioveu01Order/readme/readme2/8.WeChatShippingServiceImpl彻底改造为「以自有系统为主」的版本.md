明白，你的诉求其实是一个**架构上的对齐**：

> **“我的商城系统是数据源（主），微信只是同步通道（从）。”**

基于你刚才的代码，我帮你把 `WeChatShippingServiceImpl`**彻底改造**为「以自有系统为主」的版本。

改造后的逻辑是：

1. 

   **Service 不再关心具体参数**（transactionId、openid 等）。

2. 

   **Service 只接收一个 `orderId`或 `orderNo`**。

3. 

   **内部自己去数据库查订单、查发货单**，组装数据后同步给微信。

## ✅ 改造后的 `WeChatShippingServiceImpl`

这里假设你的项目里有 `OrderMapper`和 `OrderDeliveryMapper`之类的 DAO 层。

```
package com.aioveu.oms.aioveu01Order.service.impl;

import com.aioveu.oms.aioveu01Order.entity.Order;
import com.aioveu.oms.aioveu01Order.entity.OrderDelivery;
import com.aioveu.oms.aioveu01Order.mapper.OrderDeliveryMapper;
import com.aioveu.oms.aioveu01Order.mapper.OrderMapper;
import com.aioveu.oms.aioveu01Order.service.WeChatShippingService;
import com.aioveu.oms.aioveu01Order.utils.WeChatApiClient;
import com.fasterxml.jackson.databind.JsonNode;
import com.fasterxml.jackson.databind.node.ArrayNode;
import com.fasterxml.jackson.databind.node.ObjectNode;
import lombok.RequiredArgsConstructor;
import org.springframework.stereotype.Service;

/**
 * @description 以自有订单系统为主的微信发货同步服务
 *               对外只暴露 orderId，内部聚合数据
 */
@Service
@RequiredArgsConstructor
public class WeChatShippingServiceImpl implements WeChatShippingService {

    private final WeChatApiClient weChatApiClient;
    private final OrderMapper orderMapper;                 // 你的订单DAO
    private final OrderDeliveryMapper orderDeliveryMapper; // 你的发货单DAO

    /**
     * 1. 录入发货信息（以自有系统为准）
     * @param orderId 自家系统的订单ID
     */
    @Override
    public JsonNode uploadShipping(Long orderId) {

        // ====== Step 1: 从自家数据库取订单 ======
        Order order = orderMapper.selectById(orderId);
        if (order == null) {
            throw new RuntimeException("订单不存在，ID: " + orderId);
        }

        // ====== Step 2: 取发货信息 ======
        OrderDelivery delivery = orderDeliveryMapper.selectByOrderId(orderId);
        if (delivery == null) {
            throw new RuntimeException("发货信息不存在，订单ID: " + orderId);
        }

        // ====== Step 3: 组装微信需要的 Body ======
        ObjectNode body = weChatApiClient.objectMapper.createObjectNode();

        // 订单标识（用微信支付单号）
        ObjectNode orderKey = body.putObject("order_key");
        orderKey.put("order_number_type", 2); // 2: 微信支付单号
        orderKey.put("transaction_id", order.getTransactionId());

        body.put("logistics_type", 1); // 1: 实体物流
        body.put("delivery_mode", 1);  // 1: 统一发货

        // 物流信息
        ArrayNode shippingList = body.putArray("shipping_list");
        ObjectNode shippingItem = shippingList.addObject();
        shippingItem.put("tracking_no", delivery.getExpressNo());
        shippingItem.put("express_company", delivery.getExpressCode());
        shippingItem.put("item_desc", order.getGoodsSummary()); // 商品概述

        // 时间 & 支付人
        body.put("upload_time", java.time.OffsetDateTime.now().toString());
        ObjectNode payer = body.putObject("payer");
        payer.put("openid", order.getBuyerOpenid());

        // ====== Step 4: 同步给微信 ======
        return weChatApiClient.uploadShippingInfo(order.getClientId(), body);
    }

    /**
     * 2. 提醒用户确认收货（以自有系统为准）
     */
    @Override
    public JsonNode notifyConfirmReceive(Long orderId) {

        Order order = orderMapper.selectById(orderId);
        if (order == null) {
            throw new RuntimeException("订单不存在，ID: " + orderId);
        }

        ObjectNode body = weChatApiClient.objectMapper.createObjectNode();
        ObjectNode orderKey = body.putObject("order_key");
        orderKey.put("order_number_type", 2);
        orderKey.put("transaction_id", order.getTransactionId());

        body.put("received_time", System.currentTimeMillis() / 1000);

        return weChatApiClient.notifyConfirmReceive(order.getClientId(), body);
    }

    /**
     * 3. 查询订单状态
     */
    @Override
    public JsonNode queryOrder(Long orderId) {

        Order order = orderMapper.selectById(orderId);
        if (order == null) {
            throw new RuntimeException("订单不存在，ID: " + orderId);
        }

        ObjectNode body = weChatApiClient.objectMapper.createObjectNode();
        body.put("transaction_id", order.getTransactionId());

        return weChatApiClient.getOrderStatus(order.getClientId(), body);
    }
}
```

## ✅ 改造后的数据流向（核心思想）

| 层级                | 职责                 |
| ------------------- | -------------------- |
| **Controller**      | 只传 `orderId`       |
| **Service**         | 负责聚合自家 DB 数据 |
| **WeChatApiClient** | 只负责调微信         |
| **微信**            | 只读同步结果         |

```
┌─────────────┐
│   Controller │  ← orderId
└──────┬──────┘
       ↓
┌─────────────┐
│   Service   │  ← 查订单表 / 发货表
└──────┬──────┘
       ↓
┌─────────────┐
│ WeChatClient│  ← 调微信接口
└──────┬──────┘
       ↓
┌─────────────┐
│     微信     │
└─────────────┘
```

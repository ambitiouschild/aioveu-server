    OmsOrder order = getByOrderSn(orderSn);

    // 1️⃣ 找 Tenant 要微信配置
    WechatConfigVO config = tenantFeignClient.getWechatConfig(order.getClientId());

## ✅ 为什么支付配置在 Pay 微服务，而 app 配置在 Tenant？

| 配置           | 放哪   | 为什么           |
| -------------- | ------ | ---------------- |
| 微信支付参数   | Pay    | 支付、回调、退款 |
| appId / secret | Tenant | 渠道、发货、订阅 |
| 商户号         | Tenant | 多租户、多主体   |
| 证书           | Tenant | 安全隔离         |

✅ 支付 ≠ 微信生态
✅ 微信生态 ≠ 支付

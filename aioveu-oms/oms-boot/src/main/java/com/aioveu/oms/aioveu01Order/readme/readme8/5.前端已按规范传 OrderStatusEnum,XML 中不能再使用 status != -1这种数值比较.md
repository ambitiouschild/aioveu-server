## ✅ 为什么 `-1` 在这里是错的

```xml
status != -1
```

OGNL 会做 隐式类型转换：

```
"UNPAID" != -1
→ Double.parseDouble("UNPAID")
→ ❌ NumberFormatException
```

✅ 只要 XML 里出现 `status != -1`，必炸

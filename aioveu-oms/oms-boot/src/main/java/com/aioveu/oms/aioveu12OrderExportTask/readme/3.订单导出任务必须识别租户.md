## ✅ Service 层如何识别租户（✅ 唯一正确写法）

```java
@Service
@RequiredArgsConstructor
public class OrderExportServiceImpl implements OrderExportService {

    private final JwtDecoder jwtDecoder;

    @Override
    @Transactional
    public Long createExportTask(OrderExportQuery query, String token) {

        // 1️⃣ 从 Token 解析租户
        Jwt jwt = jwtDecoder.decode(token.replace("Bearer ", ""));
        Long tenantId = jwt.getClaim("tenantId");

        // 2️⃣ 创建任务
        OrderExportTask task = new OrderExportTask();
        task.setTenantId(tenantId);
        task.setExportNo(generateExportNo());
        task.setStatus(OrderExportTaskStatusEnum.PENDING.getValue());

        exportMapper.insert(task);
        return task.getId();
    }
}
```

## ✅ 查询时必须强制带租户（✅ 防串数据）

```java
public OrderExportTask getById(Long id, Long tenantId) {
    return exportMapper.selectOne(
        Wrappers.<OrderExportTask>lambdaQuery()
            .eq(OrderExportTask::getId, id)
            .eq(OrderExportTask::getTenantId, tenantId)
    );
}
```

✅ 少一个条件 = 事故  
✅ 这是平台级铁律

## 三、✅ 你这个方法的「最终正确写法」

### ✅ uploadShipping（修正版）

```java
@Override
public JsonNode uploadShipping(String orderSn) {

    // ====== Step 1: 订单 ======
    OmsOrder order = orderService.getByOrderNo(orderSn);
    if (order == null) {
        throw new RuntimeException("订单不存在，orderSn=" + orderSn);
    }

    Long orderId = order.getId();

    // ====== Step 2: 发货信息 ======
    OmsOrderDelivery delivery =
            omsOrderDeliveryService.selectByOrderId(orderId);

    if (delivery == null) {
        throw new RuntimeException("发货信息不存在，orderId=" + orderId);
    }

    if (delivery.getDeliveryStatus() == 0) {
        log.warn("订单已同步微信，跳过。orderId={}", orderId);
        ObjectNode skipNode = weChatApiClient.createObjectNode();
        skipNode.put("code", "SKIPPED");
        skipNode.put("msg", "already synced");
        return skipNode;
    }

    // ====== Step 3: 商品 ======
    List<OmsOrderItem> items =
            omsOrderItemService.listByOrderId(orderId);

    String itemDesc = items.stream()
            .map(i -> i.getSpuName()
                    + (StringUtils.isNotBlank(i.getSkuName())
                        ? "【" + i.getSkuName() + "】"
                        : "")
                    + " x" + i.getQuantity())
            .collect(Collectors.joining("；"));

    // ====== Step 4: openid ======
    Long memberId = order.getMemberId();
    String openId = memberFeignClient
            .getOpenIdByMemberId(memberId).getData();

    if (StringUtils.isBlank(openId)) {
        throw new RuntimeException("openid不存在");
    }

    // ====== Step 5: 组装微信参数 ======
    ObjectNode body = weChatApiClient.createObjectNode();

    ObjectNode orderKey = body.putObject("order_key");
    orderKey.put("order_number_type", 2);
    orderKey.put("transaction_id", order.getTransactionId());

    body.put("logistics_type", 1);
    body.put("delivery_mode", 1);

    ArrayNode shippingList = weChatApiClient.createArrayNode();
    ObjectNode shippingItem = shippingList.addObject();
    shippingItem.put("tracking_no", delivery.getDeliverySn());
    shippingItem.put("express_company", delivery.getDeliveryCompany());
    shippingItem.put("item_desc", itemDesc);

    body.set("shipping_list", shippingList);

    body.put("upload_time", java.time.OffsetDateTime.now().toString());

    ObjectNode payer = body.putObject("payer");
    payer.put("openid", openId);

    // ✅ 关键：从订单取 clientId
    return weChatApiClient.uploadShippingInfo(
            order.getClientId(),
            body
    );
}
```

## 四、❌ 千万不要做的 3 件事

| 错误做法            | 后果          |
| ------------------- | ------------- |
| 前端传 clientId     | ❌ 越权 / 伪造 |
| 从 ThreadLocal 瞎拿 | ❌ 不稳定      |
| 写死 clientId       | ❌ 多租户爆炸  |

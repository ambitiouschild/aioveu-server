# 三、RabbitMQ DLQ 配置（重点）

## 1️⃣ 死信 Exchange & Queue

```
@Configuration
public class PaymentDlqConfig {

    public static final String PAYMENT_EXCHANGE = "payment.exchange";
    public static final String PAYMENT_QUEUE = "payment.success.queue";
    public static final String DLQ_QUEUE = "payment.success.queue.dlq";
    public static final String DLQ_EXCHANGE = "payment.success.dlx";

    /**
     * 死信交换机
     */
    @Bean
    public DirectExchange dlxExchange() {
        return new DirectExchange(DLQ_EXCHANGE);
    }

    /**
     * 死信队列
     */
    @Bean
    public Queue dlqQueue() {
        return new Queue(DLQ_QUEUE, true);
    }

    /**
     * 绑定死信队列
     */
    @Bean
    public Binding dlqBinding() {
        return BindingBuilder
                .bind(dlqQueue())
                .to(dlxExchange())
                .with(PAYMENT_QUEUE);
    }
}
```

------

## 2️⃣ 主队列绑定 DLQ（关键）

```
@Bean
public Queue paymentQueue() {
    Map<String, Object> args = new HashMap<>();
    args.put("x-dead-letter-exchange", DLQ_EXCHANGE);
    args.put("x-dead-letter-routing-key", PAYMENT_QUEUE);
    args.put("x-max-retries", 3); // 可选
    return new Queue(PAYMENT_QUEUE, true, false, false, args);
}
```

# 四、Consumer 里怎么配合 DLQ（你现在这套刚好能用）

## ✅ 业务异常 → 直接进 DLQ

```
catch (BizException e) {
    // ❌ 不重试，直接死信
    channel.basicReject(deliveryTag, false);
}
```

------

## ✅ 系统异常 → 重试 → 超过后进 DLQ

```
catch (Exception e) {
    int retryCount =
        mqConsumeRecordService.getRetryCount(messageId, consumerGroup);

    if (retryCount >= 3) {
        channel.basicReject(deliveryTag, false); // DLQ
    } else {
        mqConsumeRecordService.incrementRetryCount(messageId, consumerGroup);
        channel.basicNack(deliveryTag, false, true); // 重试
    }
}
```

✅ **RabbitMQ 会自动把 reject 的消息转到 DLQ**

### RabbitMQ Exporter 指标

```

```

------

## 3️⃣ 最简单可用的告警（Spring 定时任务）

```
@Component
@Slf4j
public class DlqMonitorJob {

    @Autowired
    private RabbitTemplate rabbitTemplate;

    @Scheduled(fixedDelay = 60_000)
    public void monitorDlq() {
        Long count = rabbitTemplate.execute(channel ->
            channel.queueDeclarePassive("payment.success.queue.dlq").getMessageCount()
        );

        if (count != null && count > 0) {
            log.error("🚨 死信队列告警，消息数={}", count);
            // 发钉钉 / 企业微信 / 短信
        }
    }
}
```

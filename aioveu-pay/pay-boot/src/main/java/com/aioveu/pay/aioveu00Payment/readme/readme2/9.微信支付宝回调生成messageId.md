## ✅ 四、❌ 你**不应该**这样用

### ❌ 1️⃣ 不要用 `generateMessageId()`

```java
messageIdGenerator.generateMessageId(); // ❌
```

原因：
• 没有业务编号

• 无法追溯

• 对账困难


---

### ❌ 2️⃣ 不要用 `parseMessageId()` 做业务判断

```java
MessageIdInfo info = messageIdGenerator.parseMessageId(messageId);
if ("PAY".equals(info.getBizType())) { ... } // ❌
```

原因：
• 业务不该依赖 ID 结构

• 一旦格式变了，全线爆炸


✅ 只允许在日志 / 监控里用

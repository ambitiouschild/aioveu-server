## ✅ 六、订单服务消费 MQ 的正确写法 ✅

```java
@RabbitListener(queues = "order.payment.success.queue")
@Transactional
public void handlePaymentSuccess(Message message, Channel channel) throws IOException {

    PaymentSuccessMessage msg = objectMapper.readValue(
            message.getBody(), PaymentSuccessMessage.class
    );

    // 1. 幂等
    if (consumeRecordMapper.existsByMessageId(msg.getMessageId())) {
        channel.basicAck(message.getMessageProperties().getDeliveryTag(), false);
        return;
    }

    // 2. 更新订单
    OmsOrder order = orderMapper.selectByOrderNo(msg.getOrderNo());
    order.setStatus(OrderStatusEnum.PAID.getValue());
    order.setPaymentTime(LocalDateTime.now());
    order.setTransactionId(msg.getTransactionId());
    orderMapper.updateById(order);

    // 3. 记录消费
    saveConsumeRecord(msg);

    channel.basicAck(message.getMessageProperties().getDeliveryTag(), false);
}
```

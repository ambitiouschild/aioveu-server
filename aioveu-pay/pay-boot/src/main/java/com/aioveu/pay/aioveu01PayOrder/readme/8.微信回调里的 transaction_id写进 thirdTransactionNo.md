## 一、为什么是 `thirdTransactionNo`

### 对照微信回调字段

| 微信字段           | 含义               | 你实体字段                 |
| ------------------ | ------------------ | -------------------------- |
| `out_trade_no`     | 商户支付单号       | ✅ `paymentNo`              |
| ✅ `transaction_id` | **微信支付交易号** | ✅ **`thirdTransactionNo`** |
| `openid`           | 用户标识           | 无关                       |
| `trade_type`       | 交易类型           | 无关                       |

👉 **`transaction_id`就是“第三方交易流水号”**

## 四、两个字段的职责（你这套设计最漂亮的地方）

| 字段                   | 含义                              | 谁生成   | 什么时候有 |

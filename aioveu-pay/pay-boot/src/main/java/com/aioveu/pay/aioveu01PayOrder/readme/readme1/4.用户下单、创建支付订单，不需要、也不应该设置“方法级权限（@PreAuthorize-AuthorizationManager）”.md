## 二、你这个异常的真实原因（不是业务权限）

### 异常核心：

```text
AccessDeniedException: Access Denied
AuthorizationManagerBeforeMethodInterceptor
```

### ✅ 真实原因只有两个可能（99% 是第一个）：

---

### ❌ 原因 1（最常见）：**Controller 方法被 Spring Security 6.x 默认拦截**

Spring Security 6+ 默认对所有 HTTP 请求做“授权检查”，即使你没有写 `@PreAuthorize`。

如果你在 Pay 的 Controller 上有类似：

```java
@RestController
@RequestMapping("/app-api/v1/pay-order")
public class PayOrderControllerForApp {
    ...
}
```

但没有明确告诉 Security：

> “这是 App 端接口，登录用户就能访问”

👉 Spring Security 会直接 `Access Denied`

---

### ❌ 原因 2：你配置了方法级权限，但没放行

比如你有：

```java
@PreAuthorize("hasRole('ADMIN')")
@PostMapping
public String createPayOrder(...) { ... }
```

或者全局：

```java
@EnableMethodSecurity(prePostEnabled = true)
```

👉 Feign 调用也会被拦

---

## 三、✅ 正确做法（支付中台标准答案）

### ✅ 1️⃣ Controller 不要加任何权限注解

```java
@RestController
@RequestMapping("/app-api/v1/pay-order")
public class PayOrderControllerForApp {

    @PostMapping
    public String createPayOrder(
            @Valid @RequestBody PayOrderCreateForm form) {
        return payOrderService.createPayOrder(form);
    }
}
```

✅ 不加 `@PreAuthorize`
✅ 不加角色判断
✅ 登录即可访问

---

### ✅ 2️⃣ Security 配置里“放行 App 接口”

在 Pay 的 `SecurityFilterChain` 中：

```java
@Bean
public SecurityFilterChain filterChain(HttpSecurity http) throws Exception {
    http
        .authorizeHttpRequests(auth -> auth
            // ✅ App 接口：登录即可
            .requestMatchers("/app-api/**").authenticated()

            // ✅ Feign 内部调用：放行
            .anyRequest().permitAll()
        )
        .oauth2ResourceServer(oauth2 -> oauth2
            .jwt(jwt -> jwt.jwtAuthenticationConverter(jwtAuthenticationConverter()))
        );

    return http.build();
}
```

✅ 这是支付中台的标准写法

---

### ✅ 3️⃣ Feign 调用要带 Token（非常关键）

OMS 调 Pay，必须带当前登录用户的 Token：

```yaml
feign:
  client:
    config:
      default:
        requestInterceptors:
          - com.aioveu.common.security.feign.OAuth2FeignRequestInterceptor
```

否则：

• ✅ 用户登录成功

• ❌ Feign 没带 Token

• ❌ Pay 认为“匿名请求”

• ❌ `Access Denied`

### ✅ 正确分层

| -------------- | ---------------------------- |
| Gateway        | ✅ 校验 Token                 |
| OMS            | ✅ 校验用户是否登录           |

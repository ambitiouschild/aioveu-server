## 二、为什么这次回答这么“硬”？

因为**支付系统有几个特殊性**：

| 特性   | 为什么必须是数字         |
| ------ | ------------------------ |
| 对账   | 数字稳定、无歧义         |
| 回调   | 第三方要解析             |
| 幂等   | 枚举名可能重构           |
| 统计   | SQL / ES / OLAP 全认数字 |
| 国际化 | 数字不受语言影响         |
| 审计   | 日志可映射，协议不可变   |

👉 **支付不是普通 CRUD**

👉 **支付是“金融协议”**

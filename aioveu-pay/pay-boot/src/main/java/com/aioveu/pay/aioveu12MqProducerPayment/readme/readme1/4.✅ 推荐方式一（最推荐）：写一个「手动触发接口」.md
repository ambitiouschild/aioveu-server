# ✅ 二、✅ 推荐方式一（最推荐）：写一个「手动触发接口」

### ✅ Controller（临时，仅测试）

```
@RestController
@RequestMapping("/test/pay")
public class TestPayCallbackController {

    @Autowired
    private PaymentMessageService paymentMessageService;

    @PostMapping("/mock-callback-success")
    public String mockPaySuccess(@RequestParam String paymentNo) {

        SendPaymentMqDTO dto = new SendPaymentMqDTO();
        dto.setPaymentNo(paymentNo);

        boolean success = paymentMessageService.sendPaymentSuccessMessage(dto);
        return success ? "ok" : "fail";
    }
}
```

✅ **只测 MQ + 订单**

✅ **不涉及微信**

✅ **最安全**

# ✅ 四、✅ 测试时你必须盯住的 6 个点（非常重要）

| 检查点                            | 是否通过 |
| --------------------------------- | -------- |
| PayOrder 状态是否变为 PAID        | ✅        |
| mq_send_record 状态是否为 SUCCESS | ✅        |
| messageId 是否唯一                | ✅        |
| 订单服务是否消费成功              | ✅        |

| 重复调用是否幂等                  | ✅        |

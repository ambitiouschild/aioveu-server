## ✅ 二、RabbitMQ 管理台能看到什么？

### ✅ 能看到的

| 位置        | 能看到             |
| ----------- | ------------------ |
| Queues      | 未消费的消息       |
| Messages    | Ready / Unacked    |
| Get Message | 拉单条（但不会删） |

### ❌ 看不到的

| 情况           | 原因          |
| -------------- | ------------- |
| 已 ACK 的消息  | 立刻删除      |
| 消费成功的消息 | 没了          |
| 历史消息       | RabbitMQ 不存 |

👉 **RabbitMQ 不是数据库**

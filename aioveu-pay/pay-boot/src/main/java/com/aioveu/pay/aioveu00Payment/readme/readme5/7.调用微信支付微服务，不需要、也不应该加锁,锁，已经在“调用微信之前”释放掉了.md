## 六、✅ 你现在这个方法的“死亡陷阱”在哪？

```java
finally {
    if (lock.isLocked()) {
        lock.unlock(); // ❌ 这是“锁外判断”
    }
}
```

### ❌ 问题

• `isLocked()` 不表示“当前线程持有锁”

• 极端情况会 多 unlock → 抛 IllegalMonitorStateException


### ✅ 正确写法（你前面已经对了）

```java
finally {
    if (lock.isHeldByCurrentThread()) {
        lock.unlock();
    }
}
```

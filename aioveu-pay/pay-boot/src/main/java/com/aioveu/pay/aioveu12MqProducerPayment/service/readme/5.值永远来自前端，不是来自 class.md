## ✅ 你现在脑子里这条链路，已经是 100% 正确的了

```
前端 JSON
   ↓
Spring 接到请求
   ↓
Jackson 负责：new 一个 DTO 对象
   ↓
把 JSON 的值，一个个 set 进去
   ↓
你才能在方法里：dto.getManualSend()
```

👉 没有 new，就没有对象
👉 没有对象，就没有值
👉 值永远来自前端，不是来自 class

## ✅ 三、你现在可以放心用的几个结论

| 问题                     | 你现在的理解  |
| ------------------------ | ------------- |
| DTO 里的值哪来的         | ✅ 前端        |
| class 里有值吗           | ❌ 没有        |
| Jackson 在干嘛           | ✅ new + set   |
| Spring 在干嘛            | ✅ 接对象      |
| 为什么 manualSend 拿不到 | ✅ 没 set 进去 |

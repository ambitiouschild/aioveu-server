## ✅ 一句话结论（先给你）

> ❌ **`flowType`不应该用 `String`**

👉 永远漏数据

👉 **没人说得清**

## ✅ 正确做法（✅ 支付系统标准）

### ✅ 用枚举（唯一正确答案）

```
@Getter
public enum PayFlowTypeEnum {

    PAYMENT("PAYMENT", "支付"),
    REFUND("REFUND", "退款"),
    SETTLEMENT("SETTLEMENT", "结算"),
    ADJUST("ADJUST", "调账");

    @EnumValue
    private final String code;
    private final String desc;

    PayFlowTypeEnum(String code, String desc) {
        this.code = code;
        this.desc = desc;
    }
}
```

------

### ✅ 实体里（✅ 正确）

```
@TableName("pay_flow")
public class PayFlow extends BaseEntityWithTenantId {

    /**
     * 流水类型
     */
    private PayFlowTypeEnum flowType;
}
```

✅ **数据库存字符串 `'PAYMENT'`**

✅ **Java 用枚举**

✅ **不可能写错**

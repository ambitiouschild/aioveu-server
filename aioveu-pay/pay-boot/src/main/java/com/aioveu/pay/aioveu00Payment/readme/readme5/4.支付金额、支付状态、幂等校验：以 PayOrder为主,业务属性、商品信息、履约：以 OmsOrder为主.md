## 三、✅ 标准「支付金额校验」模板（直接照抄）

```
// 1. 金额入参统一为 BigDecimal（单位：分）

    BigDecimal payYuan = fenToYuan(payAmountFen);

    log.error("【支付】金额不匹配，支付订单金额: ¥{}，请求金额: ¥{}",
            payYuan, reqYuan);

    throw new BizException(String.format(
            "支付金额不匹配，订单金额: ¥%s",

    ));
}
```

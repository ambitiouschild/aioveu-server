## ✅ 一、结论先给你

| 层        | 用什么            |
| --------- | ----------------- |
| 数据库    | ✅ `INT / TINYINT` |
| Java 实体 | ✅ 枚举（enum）    |
| 业务代码  | ✅ 只认枚举        |

👉 数据库存 int，Java 用 enum，完全没问题

---

## ✅ 二、MyBatis / MyBatis‑Plus 怎么自动映射？

### ✅ 1️⃣ 枚举写法（✅ 必须这样）

```java
@Getter
public enum PayFlowStatusEnum {

    PROCESSING(0, "处理中"),
    SUCCESS(1, "成功"),
    FAILED(2, "失败");

    private final int code;
    private final String desc;

    PayFlowStatusEnum(int code, String desc) {
        this.code = code;
        this.desc = desc;
    }

    public int getCode() {
        return code;
    }
}
```

---

### ✅ 2️⃣ 实体里直接用枚举（✅）

```java
@TableName("pay_flow")
public class PayFlow extends BaseEntityWithTenantId {

    /**
     * 流水状态
     */
    private PayFlowStatusEnum flowStatus;
}
```

## ✅ 三、MyBatis‑Plus 自动映射（✅ 关键）

### ✅ 方式一（✅ 推荐，最简单）

在 `application.yml` 中配置：

```yaml
mybatis-plus:
  configuration:
    default-enum-type-handler: org.apache.ibatis.type.EnumOrdinalTypeHandler
```

⚠️ 前提：code 从 0 开始连续

---

### ✅ 方式二（✅ 最稳，强烈推荐）

用 `@EnumValue`（官方推荐）

```java
@Getter
public enum PayFlowStatusEnum {

    PROCESSING(0, "处理中"),
    SUCCESS(1, "成功"),
    FAILED(2, "失败");

    @EnumValue
    private final int code;

    private final String desc;

    PayFlowStatusEnum(int code, String desc) {
        this.code = code;
        this.desc = desc;
    }
}
```

✅ 数据库 ↔ code
✅ Java ↔ enum
✅ 永不乱

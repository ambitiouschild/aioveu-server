## 一、为什么必须跳 `paymentNo`？（3 个铁律）

### ❶ 支付页的本质是“付钱”，不是“看订单”

| 页面       | 关心什么      |
| ---------- | ------------- |
| 订单详情页 | ✅ `orderSn`   |
| 支付页     | ✅ `paymentNo` |
| 退款页     | ✅ `refundNo`  |

👉 支付系统只认 `paymentNo`

---

### ❷ `orderSn` ≠ 支付凭证

• 一个订单 可能有多个支付单

  • 部分退款

  • 多次支付

  • 换渠道重付

• `orderSn` 是业务聚合号

• `paymentNo` 是资金唯一键


👉 微信 / 支付宝 / Pay 中台：只认 `paymentNo`

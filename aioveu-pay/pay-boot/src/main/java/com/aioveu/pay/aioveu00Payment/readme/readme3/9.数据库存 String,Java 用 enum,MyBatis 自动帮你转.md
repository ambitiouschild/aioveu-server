## ✅ 二、Java 枚举（✅ 正确写法）

### ✅ 枚举定义（关键）

```java
@Getter
public enum PayFlowTypeEnum {

    PAYMENT("PAYMENT", "支付"),
    REFUND("REFUND", "退款"),
    SETTLEMENT("SETTLEMENT", "结算"),
    ADJUST("ADJUST", "调账");

    @EnumValue
    private final String code;

    private final String desc;

    PayFlowTypeEnum(String code, String desc) {
        this.code = code;
        this.desc = desc;
    }

    public String getCode() {
        return code;
    }
}
```

✅ `@EnumValue` 告诉 MP：用 code 映射数据库

## ✅ 五、String 枚举 vs int 枚举（怎么选？）

| 对比     | String 枚举 | int 枚举 |
| -------- | ----------- | -------- |
| 可读性   | ✅ 极强      | ❌ 差     |
| 对账     | ✅ 直接看懂  | ❌ 要查表 |
| 扩展性   | ✅ 好        | ✅ 好     |
| 性能     | ✅ 略慢一点  | ✅ 更快   |
| 支付系统 | ✅ 推荐      | ✅ 也可用 |

👉 支付系统我更推荐 String 枚举

## 核心思路

**在初始化时（`@PostConstruct`），让多租户插件忽略`pay_config_wechat`表的租户过滤**，这样就能查询到所有租户的配置。

## 解决方案

### 1. 修改`MyTenantLineHandler`，在初始化时忽略租户过滤

```
@Component
@Slf4j
public class MyTenantLineHandler implements TenantLineHandler {
    
    // 需要忽略租户过滤的表
    private static final Set<String> IGNORE_TABLES = new HashSet<>(Arrays.asList(
        "pay_config_wechat",  // 微信支付配置表
        "pay_config_alipay",  // 支付宝配置表
        "pay_config",         // 支付配置主表
        "sys_dict",           // 系统字典
        "sys_config"          // 系统配置
    ));
    
    @Override
    public boolean ignoreTable(String tableName) {
        // 1. 检查是否是系统初始化阶段
        if (isSystemInitializing()) {
            log.info("【MyTenantLineHandler】系统初始化中，忽略所有表的租户过滤");
            return true;  // 在初始化时，所有表都忽略租户过滤
        }
        
        // 2. 检查是否是特定的系统表
        if (IGNORE_TABLES.contains(tableName.toLowerCase())) {
            log.info("【MyTenantLineHandler】忽略表 {} 的租户过滤", tableName);
            return true;
        }
        
        return false;
    }
    
    @Override
    public Expression getTenantId() {
        Long tenantId = getCurrentTenantId();
        
        if (tenantId == null) {
            // 如果是初始化阶段，返回一个不会影响查询的值
            if (isSystemInitializing()) {
                log.info("【MyTenantLineHandler】初始化阶段，返回null不添加租户条件");
                return null;  // 返回null，不添加租户条件
            }
            
            // 正常请求中，租户ID应该存在
            log.warn("【MyTenantLineHandler】租户ID为null");
            return new LongValue(-1L);
        }
        
        return new LongValue(tenantId);
    }
    
    /**
     * 判断是否是系统初始化阶段
     */
    private boolean isSystemInitializing() {
        StackTraceElement[] stackTrace = Thread.currentThread().getStackTrace();
        for (StackTraceElement element : stackTrace) {
            // 检查是否是@PostConstruct初始化
            if (element.getClassName().contains("PostConstruct") ||
                element.getMethodName().contains("init") ||
                element.getMethodName().contains("afterPropertiesSet") ||
                element.getClassName().contains("Configuration")) {
                return true;
            }
            
            // 检查是否是配置类初始化
            if (element.getClassName().contains("WeChatPayConfig") ||
                element.getClassName().contains("PayConfigWechat")) {
                return true;
            }
        }
        return false;
    }
    
    /**
     * 获取当前租户ID
     */
    private Long getCurrentTenantId() {
        try {
            // 1. 从租户上下文获取
            Long tenantId = com.aioveu.common.context.TenantContextHolder.getTenantId();
            if (tenantId != null) {
                return tenantId;
            }
            
            // 2. 从SecurityContext获取
            Authentication authentication = SecurityContextHolder.getContext().getAuthentication();
            if (authentication != null) {
                Object principal = authentication.getPrincipal();
                // 根据你的用户对象获取租户ID
            }
            
            // 3. 从请求头获取
            ServletRequestAttributes attributes = (ServletRequestAttributes) 
                RequestContextHolder.getRequestAttributes();
            if (attributes != null) {
                HttpServletRequest request = attributes.getRequest();
                String tenantIdStr = request.getHeader("X-Tenant-Id");
                if (StringUtils.isNotBlank(tenantIdStr)) {
                    return Long.valueOf(tenantIdStr);
                }
            }
            
            return null;
        } catch (Exception e) {
            log.warn("获取租户ID异常", e);
            return null;
        }
    }
}

### 2. 在`WeChatPayConfig`中使用`@PostConstruct`初始化

```
@Slf4j
@Configuration
@Data
public class WeChatPayConfig {
    
    @Autowired
    private PayConfigWechatService payConfigWechatService;
    
    // 缓存数据库中的配置
    private final Map<String, PayConfigWechat> configCache = new ConcurrentHashMap<>();
    
    /**
     * 是否启用微信支付
     */
    private boolean enabled = false;
    
    /**
     * 当前使用的配置
     */
    private PayConfigWechat currentConfig;
    
    /**
     * 初始化配置
     */
    @PostConstruct
    public void init() {
        try {
            log.info("【WeChatPayConfig】开始初始化...");
            
            // 在初始化时加载所有配置
            loadConfigFromDatabase();
            
            if (configCache.isEmpty()) {
                log.error("【WeChatPayConfig】初始化失败：数据库中没有微信支付配置");
                enabled = false;
            } else {
                log.info("【WeChatPayConfig】初始化成功，共加载{}个租户的配置", configCache.size());
                enabled = true;
            }
        } catch (Exception e) {
            log.error("【WeChatPayConfig】初始化异常", e);
            enabled = false;
        }
    }
    
    /**
     * 从数据库加载所有配置
     */
    public synchronized void loadConfigFromDatabase() {
        try {
            // 清空缓存
            configCache.clear();
            
            // 此时多租户插件会忽略租户过滤
            List<PayConfigWechat> configs = payConfigWechatService.listEnabledConfigs();
            
            log.info("【WeChatPayConfig】查询所有启用的微信支付配置：{}", configs != null ? configs.size() : 0);
            
            if (CollectionUtils.isEmpty(configs)) {
                log.warn("【微信支付配置】数据库中没有启用的微信支付配置");
                enabled = false;
                return;
            }
            
            // 缓存所有配置
            for (PayConfigWechat config : configs) {
                String cacheKey = buildCacheKey(config);
                configCache.put(cacheKey, config);
                log.info("【微信支付配置】加载配置: 租户ID={}, 应用ID={}, 商户号={}",
                        config.getTenantId(), config.getAppId(), config.getMchId());
            }
            
            // 设置默认配置
            if (!configs.isEmpty()) {
                currentConfig = configs.stream()
                        .filter(c -> Boolean.TRUE.equals(c.getIsDefault()))
                        .findFirst()
                        .orElse(configs.get(0));
            }
            
            enabled = true;
            
            log.info("【微信支付配置】从数据库加载配置成功，共{}个配置", configs.size());
            
        } catch (Exception e) {
            log.error("【微信支付配置】从数据库加载配置失败", e);
            enabled = false;
        }
    }
    
    /**
     * 根据租户ID获取配置
     */
    public PayConfigWechat getConfig(Long tenantId, String appId) {
        if (tenantId == null) {
            log.warn("【微信支付配置】获取配置时租户ID为null");
            return getDefaultConfig();
        }
        
        String cacheKey = buildCacheKey(tenantId, appId);
        PayConfigWechat config = configCache.get(cacheKey);
        
        if (config == null) {
            log.warn("【微信支付配置】未找到租户{}应用{}的配置", tenantId, appId);
            return getDefaultConfig();
        }
        
        return config;
    }
    
    /**
     * 根据租户ID获取配置
     */
    public PayConfigWechat getConfig(Long tenantId) {
        if (tenantId == null) {
            return getDefaultConfig();
        }
        
        // 查找该租户的所有配置
        List<PayConfigWechat> tenantConfigs = configCache.values().stream()
                .filter(config -> tenantId.equals(config.getTenantId()))
                .collect(Collectors.toList());
        
        if (tenantConfigs.isEmpty()) {
            log.warn("【微信支付配置】未找到租户{}的配置", tenantId);
            return getDefaultConfig();
        }
        
        // 返回默认配置
        return tenantConfigs.stream()
                .filter(config -> Boolean.TRUE.equals(config.getIsDefault()))
                .findFirst()
                .orElse(tenantConfigs.get(0));
    }
    
    /**
     * 获取默认配置
     */
    public PayConfigWechat getDefaultConfig() {
        if (currentConfig == null && !configCache.isEmpty()) {
            // 设置第一个配置为默认
            currentConfig = configCache.values().iterator().next();
        }
        return currentConfig;
    }
    
    /**
     * 构建缓存key
     */
    private String buildCacheKey(PayConfigWechat config) {
        return config.getTenantId() + ":" + config.getAppId();
    }
    
    private String buildCacheKey(Long tenantId, String appId) {
        return tenantId + ":" + appId;
    }
}

### 3. 在`PayConfigWechatService`中正常查询

```
@Service
@Slf4j
public class PayConfigWechatServiceImpl 
        extends ServiceImpl<PayConfigWechatMapper, PayConfigWechat>
        implements PayConfigWechatService {
    
    @Override
    public List<PayConfigWechat> listEnabledConfigs() {
        try {
            // 正常使用MyBatis-Plus查询
            LambdaQueryWrapper<PayConfigWechat> wrapper = new LambdaQueryWrapper<>();
            wrapper.eq(PayConfigWechat::getEnabled, 1)
                   .eq(PayConfigWechat::getIsDeleted, 0)
                   .orderByDesc(PayConfigWechat::getIsDefault)
                   .orderByAsc(PayConfigWechat::getSort)
                   .orderByDesc(PayConfigWechat::getCreateTime);
            
            List<PayConfigWechat> configs = this.list(wrapper);
            log.info("【PayConfigWechat】查询所有启用的微信支付配置：{}", configs.size());
            return configs;
            
        } catch (Exception e) {
            log.error("【PayConfigWechat】查询启用的微信支付配置失败", e);
            return new ArrayList<>();
        }
    }
}
```

## 工作原理

1. **启动时**：`@PostConstruct`执行`WeChatPayConfig.init()`
2. **初始化查询**：`payConfigWechatService.listEnabledConfigs()`执行
3. **多租户插件**：`MyTenantLineHandler`检测到是初始化阶段，返回`ignoreTable() = true`
4. **SQL生成**：不添加`tenant_id`条件，查询所有数据
5. **缓存数据**：所有租户的配置加载到内存
6. **支付请求**：根据租户ID从缓存获取对应配置

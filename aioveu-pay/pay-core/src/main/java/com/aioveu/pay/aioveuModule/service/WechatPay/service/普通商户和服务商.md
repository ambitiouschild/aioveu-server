## 关键区别

| 特性     | 普通商户(payments)                     | 服务商(partnerpayments)                       |
| -------- | -------------------------------------- | --------------------------------------------- |
| 适用场景 | 直接商户                               | 服务商模式                                    |
| 包路径   | `com.wechat.pay.java.service.payments` | `com.wechat.pay.java.service.partnerpayments` |
| 需要参数 | 商户号(mchid)                          | 服务商商户号(sp_mchid)和子商户号(sub_mchid)   |

## 检查清单

1. **确认您的商户类型**：
   - 普通商户：使用 `payments`包下的类
   - 服务商：使用 `partnerpayments`包下的类
2. **统一导入路径**：
   - 确保所有相关类来自同一个包（要么全是 `payments`，要么全是 `partnerpayments`）

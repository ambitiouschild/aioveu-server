## 四、你的 `saveOauth2RegisteredClient`正确版本（✅ 精简）

```
@Override
@Transactional
public boolean saveOauth2RegisteredClient(Oauth2RegisteredClientForm formData) {

    // 1️⃣ 防重
    RegisteredClient existing =
            registeredClientRepository.findByClientId(formData.getClientId());
    if (existing != null) {
        throw new BusinessException(ResultCode.CLIENT_ALREADY_EXISTS, formData.getClientId());
    }

    // 2️⃣ 生成 secret
    String rawSecret = StringUtils.hasText(formData.getClientSecret())
            ? formData.getClientSecret()
            : generateRandomSecret();
    String encodedSecret = passwordEncoder.encode(rawSecret);

    // 3️⃣ 构建并保存官方客户端
    RegisteredClient registeredClient =
            buildRegisteredClient(formData, encodedSecret);
    registeredClientRepository.save(registeredClient);

    // 4️⃣ 写业务状态表 ✅
    Oauth2RegisteredClientBiz biz = new Oauth2RegisteredClientBiz();
    biz.setClientId(registeredClient.getId()); // ⚠️ 用 id，不是 client_id
    biz.setEnabled(RegisteredClientBizStatusEnum.NORMAL);
    oauth2RegisteredClientBizMapper.insert(biz);

    log.info("客户端注册成功: clientId={}", formData.getClientId());
    return true;
}
```

✅ **不需要任何 `this.save(entity)`**

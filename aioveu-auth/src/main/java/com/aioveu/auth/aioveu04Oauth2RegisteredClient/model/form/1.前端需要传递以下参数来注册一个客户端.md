## 前端请求参数模板

```
{
  "clientId": "your-client-id",  // 必须，客户端唯一标识
  "clientName": "Your Client Application Name",  // 必须，客户端显示名称

  "clientIdIssuedAt": "2026-03-23 10:00:00",  // 必须，客户端ID创建时间
  
  "clientSecretExpiresAt": "2027-03-23 10:00:00",  // 可选，客户端密钥过期时间
  
  "clientAuthenticationMethods": "client_secret_basic,client_secret_post",  // JSON格式数组

  "scopes": "openid,profile,email,read,write",  // JSON格式数组

  "tokenSettings": "{\"accessTokenTimeToLiveSeconds\": 86400, \"refreshTokenTimeToLiveSeconds\": 2592000, \"reuseRefreshTokens\": true, \"accessTokenFormat\": \"SELF_CONTAINED\", \"idTokenSignatureAlgorithm\": \"RS256\"}"
}
```

## 字段详细说明

### 1. 必填字段

| 字段名             | 示例值                      | 说明                                              |
| ------------------ | --------------------------- | ------------------------------------------------- |
| `clientId`         | `"your-client-id"`          | 客户端唯一标识，用于OAuth2请求中的`client_id`参数 |
| `clientName`       | `"Your Client Application"` | 客户端的显示名称，用于用户界面展示                |
| `clientIdIssuedAt` | `"2026-03-23 10:00:00"`     | 客户端ID的创建时间，格式`yyyy-MM-dd HH:mm:ss`     |

### 2. 可选字段

| 字段名                  | 示例值                  | 说明                                 |
| ----------------------- | ----------------------- | ------------------------------------ |
| `clientSecret`          | `"your-secret-123"`     | 客户端密钥，公共客户端可省略         |
| `clientSecretExpiresAt` | `"2027-03-23 10:00:00"` | 客户端密钥过期时间，不传表示永不过期 |

### 3. JSON格式数组字段（逗号分隔）

| 字段名                        | 示例值                                     | 说明                   |
| ----------------------------- | ------------------------------------------ | ---------------------- |
| `clientAuthenticationMethods` | `"client_secret_basic,client_secret_post"` | 支持的客户端认证方法   |
| `authorizationGrantTypes`     | `"authorization_code,refresh_token"`       | 支持的授权类型         |
| `redirectUris`                | `"https://app.com/callback"`               | 允许的重定向URI列表    |
| `postLogoutRedirectUris`      | `"https://app.com/logout"`                 | 登出后的重定向URI      |
| `scopes`                      | `"openid,profile,read"`                    | 客户端可请求的范围列表 |

### 4. JSON配置对象字段

**clientSettings 字段（JSON字符串）：**

```
{
  "requireAuthorizationConsent": false,  // 是否需要授权同意页
  "requireProofKey": false,  // 是否需要PKCE
  "jwkSetUrl": "",  // 客户端密钥JWT的签名算法URL
  "tokenEndpointAuthenticationSigningAlgorithm": "RS256"  // 客户端认证方法签名算法
}
```

**tokenSettings 字段（JSON字符串）：**

```
{
  "accessTokenTimeToLiveSeconds": 86400,  // 访问令牌有效期(秒)，默认1天
  "refreshTokenTimeToLiveSeconds": 2592000,  // 刷新令牌有效期(秒)，默认30天
  "reuseRefreshTokens": true,  // 是否重用刷新令牌
  "accessTokenFormat": "SELF_CONTAINED",  // 访问令牌格式
  "idTokenSignatureAlgorithm": "RS256"  // ID Token签名算法
}
```

## 前端调用示例（使用JavaScript）

```
// 前端API调用示例
async function registerClient() {
  const clientData = {
    clientId: "my-web-app",
    clientName: "My Web Application",
    clientSecret: "secure-secret-123",
    clientIdIssuedAt: "2026-03-23 10:00:00",
    clientAuthenticationMethods: "client_secret_basic,client_secret_post",
    authorizationGrantTypes: "authorization_code,refresh_token,client_credentials",
    redirectUris: "https://myapp.com/oauth/callback,https://localhost:3000/oauth/callback",
    postLogoutRedirectUris: "https://myapp.com/logout",
    scopes: "openid,profile,email,read,write",
    clientSettings: JSON.stringify({
      requireAuthorizationConsent: true,
      requireProofKey: true
    }),
    tokenSettings: JSON.stringify({
      accessTokenTimeToLiveSeconds: 7200,  // 2小时
      refreshTokenTimeToLiveSeconds: 604800,  // 7天
      reuseRefreshTokens: true
    })
  };

  try {
    const response = await fetch('/api/oauth2-registered-clients', {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
      },
      body: JSON.stringify(clientData)
    });
    
    if (response.ok) {
      const result = await response.json();
      console.log('客户端注册成功:', result);
    } else {
      console.error('客户端注册失败:', await response.text());
    }
  } catch (error) {
    console.error('网络错误:', error);
  }
}
```

## 注意事项

1. **JSON格式**：`clientSettings`和 `tokenSettings`字段需要传递**JSON字符串**，而不是JSON对象
2. **数组字段**：认证方法、授权类型等数组字段使用**逗号分隔的字符串**
3. **时间格式**：时间字段需使用 `yyyy-MM-dd HH:mm:ss`格式
4. **字段长度**：注意各字段的最大长度限制（如clientId不超过100字符）
5. **必填验证**：确保必填字段不为空，否则会返回验证错误

这个模板完全基于你提供的 `Oauth2RegisteredClientForm`类定义，确保与后端数据结构完全匹配。

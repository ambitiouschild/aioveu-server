## 针对平台管理员的简化方案：

```
@Operation(summary = "平台管理员切换租户")
@PostMapping("/switch-tenant")
public Result<Map<String, Object>> switchTenant(@RequestParam Long tenantId) {
    
    // 1. 只需要验证是否是平台管理员
    if (!SecurityUtils.isPlatformAdmin()) {
        return Result.failed("无权限，仅平台管理员可切换");
    }
    
    // 2. 获取当前管理员信息
    Authentication authentication = SecurityContextHolder.getContext().getAuthentication();
    SysUserDetails adminDetails = (SysUserDetails) authentication.getPrincipal();
    
    // 3. 验证租户是否存在（可选）
    if (!tenantFeignClient.tenantExists(tenantId)) {
        return Result.failed("租户不存在");
    }
    
    // 4. 记录切换日志
    log.info("平台管理员 {} 切换到租户 {}", adminDetails.getUsername(), tenantId);
    
    // 5. 返回结果
    Map<String, Object> result = new HashMap<>();
    result.put("success", true);
    result.put("tenant_id", tenantId);
    result.put("admin_name", adminDetails.getUsername());
    result.put("message", "切换成功，请使用X-Tenant-ID请求头");
    result.put("timestamp", System.currentTimeMillis());
    
    return Result.success(result);
}
```

## 前端调用：

```
// 管理员切换租户
async function adminSwitchTenant(tenantId) {
    const response = await fetch(`/api/v1/auth/switch-tenant?tenantId=${tenantId}`, {
        method: 'POST',
        headers: {
            'Authorization': 'Bearer ' + getToken()
        }
    });
    
    const result = await response.json();
    if (result.success) {
        // 1. 存储当前租户
        localStorage.setItem('current_admin_tenant', tenantId);
        
        // 2. 显示切换成功
        showSuccess('已切换到租户: ' + tenantId);
        
        // 3. 刷新页面数据
        window.location.reload();
    }
    return result;
}
```

## 后端处理（简单验证）：

```
@Component
public class AdminTenantFilter implements Filter {
    
    @Override
    public void doFilter(ServletRequest request, ServletResponse response, FilterChain chain) 
            throws IOException, ServletException {
        
        HttpServletRequest httpRequest = (HttpServletRequest) request;
        
        // 获取请求中的租户ID
        String tenantIdStr = httpRequest.getHeader("X-Tenant-ID");
        
        if (tenantIdStr != null && SecurityUtils.isPlatformAdmin()) {
            try {
                Long tenantId = Long.parseLong(tenantIdStr);
                // 管理员可以访问任何租户，直接设置
                TenantContext.setCurrentTenant(tenantId);
            } catch (Exception e) {
                // 忽略错误，使用默认租户
            }
        }
        
        chain.doFilter(request, response);
        TenantContext.clear();
    }
}
```

## 如果管理员需要模拟普通用户：

```
@PostMapping("/switch-tenant")
public Result<Map<String, Object>> switchTenant(@RequestBody SwitchTenantRequest request) {
    // 如果是平台管理员
    if (SecurityUtils.isPlatformAdmin()) {
        Map<String, Object> result = new HashMap<>();
        
        // 1. 如果只是切换租户查看
        if (request.getMode().equals("view")) {
            result.put("mode", "view");
            result.put("tenant_id", request.getTenantId());
            result.put("message", "管理员查看模式");
        }
        // 2. 如果要模拟普通用户操作
        else if (request.getMode().equals("operate")) {
            // 生成临时Token（可选）
            String tempToken = generateTempTokenForTenant(request.getTenantId());
            result.put("access_token", tempToken);
            result.put("tenant_id", request.getTenantId());
            result.put("mode", "operate");
            result.put("expires_in", 3600);
        }
        
        return Result.success(result);
    }
    
    return Result.failed("无权限");
}
```

## 总结：

对于**平台管理员切换租户**的情况：

1. **不需要验证权限** → 管理员本来就有所有权限
2. **不需要重新生成Token** → 用原Token+租户ID即可
3. **不需要重新认证** → 权限不变
4. **只需要记录日志** → 谁切换到了哪个租户

根据您的问题，我将为您详细说明如何实现**小程序公共接口**，通过客户端ID获取租户ID，然后返回不同租户的个性化数据（如首页分类和广告轮播图）。

## 一、整体架构设计

```
小程序 → 网关 → 认证服务 → 业务服务 → 数据库
   |        |         |         |
   | 携带client_id → 解析tenant_id → 查询tenant_data
```

### 3. **服务层实现**

#### 租户服务

@Service
@Slf4j
public class TenantServiceImpl implements TenantService {
    
    @Autowired
    private TenantClientMapper tenantClientMapper;
    
    @Override
    public Long getTenantIdByClientId(String clientId) {
        // 1. 从数据库查询
        TenantClient tenantClient = tenantClientMapper.selectByClientId(clientId);
        
        if (tenantClient == null) {
            log.error("未找到对应的租户配置，clientId={}", clientId);
            throw new BusinessException("无效的客户端ID");
        }
        
        if (tenantClient.getStatus() != 1) {
            log.error("客户端已被禁用，clientId={}", clientId);
            throw new BusinessException("客户端已被禁用");
        }
        
        return tenantClient.getTenantId();
    }
    
    @Override
    @Cacheable(value = "tenant:client", key = "#clientId")
    public TenantClient getTenantClientInfo(String clientId) {
        return tenantClientMapper.selectByClientId(clientId);
    }
}

#### 分类服务

```
@Service
@Slf4j
public class CategoryServiceImpl implements CategoryService {
    
    @Autowired
    private CategoryMapper categoryMapper;
    
    @Override
    public List<CategoryVo> getCategoriesByTenant(Long tenantId) {
        // 查询条件
        LambdaQueryWrapper<Category> queryWrapper = new LambdaQueryWrapper<>();
        queryWrapper.eq(Category::getTenantId, tenantId)
                   .eq(Category::getStatus, 1)
                   .orderByAsc(Category::getSortOrder);
        
        // 查询数据
        List<Category> categories = categoryMapper.selectList(queryWrapper);
        
        // 转换为VO
        return categories.stream()
                .map(this::convertToVo)
                .collect(Collectors.toList());
    }
    
    @Override
    @Cacheable(value = "category:tenant", key = "#tenantId")
    public List<CategoryVo> getCachedCategories(Long tenantId) {
        return getCategoriesByTenant(tenantId);
    }
    
    private CategoryVo convertToVo(Category category) {
        CategoryVo vo = new CategoryVo();
        vo.setId(category.getId());
        vo.setName(category.getName());
        vo.setIcon(category.getIcon());
        vo.setSortOrder(category.getSortOrder());
        return vo;
    }
}

### 4. **缓存优化**

@Configuration
public class PublicApiCacheConfig {
    
    /**
     * 公共接口缓存配置
     */
    @Bean
    public CacheManager publicCacheManager(RedisConnectionFactory factory) {
        RedisCacheConfiguration config = RedisCacheConfiguration.defaultCacheConfig()
            .entryTtl(Duration.ofMinutes(10))  // 缓存10分钟
            .disableCachingNullValues()
            .serializeKeysWith(RedisSerializationContext.SerializationPair.fromSerializer(
                RedisSerializer.string()
            ))
            .serializeValuesWith(RedisSerializationContext.SerializationPair.fromSerializer(
                RedisSerializer.json()
            ))
            .computePrefixWith(name -> "public:" + name + ":");  // 缓存前缀
        
        return RedisCacheManager.builder(factory)
            .cacheDefaults(config)
            .withCacheConfiguration("category:tenant", 
                config.entryTtl(Duration.ofMinutes(30)))  // 分类缓存30分钟
            .withCacheConfiguration("banner:tenant", 
                config.entryTtl(Duration.ofMinutes(15)))  // 广告缓存15分钟
            .build();
    }
}

// utils/api.js
class PublicApi {
  constructor(clientId) {
    this.clientId = clientId;
    this.baseUrl = 'https://your-domain.com/api/public';
  }
  
  /**
   * 获取首页数据
   */
  async getHomeData() {
    try {
      const response = await wx.request({
        url: `${this.baseUrl}/home-data`,
        method: 'GET',
        data: {
          clientId: this.clientId
        }
      });
      
      if (response.statusCode === 200 && response.data.code === 200) {
        return response.data.data;
      } else {
        console.error('获取首页数据失败:', response.data.message);
        return null;
      }
    } catch (error) {
      console.error('请求异常:', error);
      return null;
    }
  }
  
  /**
   * 获取分类列表
   */
  async getCategories() {
    return this.request('/categories');
  }
  
  /**
   * 获取广告轮播图
   */
  async getBanners() {
    return this.request('/banners');
  }
  
  /**
   * 通用请求方法
   */
  async request(endpoint) {
    const response = await wx.request({
      url: `${this.baseUrl}${endpoint}`,
      method: 'GET',
      data: { clientId: this.clientId }
    });
    
    if (response.statusCode === 200 && response.data.code === 200) {
      return response.data.data;
    }
    throw new Error(response.data.message || '请求失败');
  }
}

## 四、安全与性能优化

### 1. **接口安全**

```
@Configuration
public class PublicApiSecurityConfig {
    
    /**
     * 公共接口安全配置
     */
    @Bean
    public SecurityFilterChain publicApiFilterChain(HttpSecurity http) throws Exception {
        http
            .securityMatcher("/api/public/**")
            .authorizeHttpRequests(auth -> auth
                .anyRequest().permitAll()  // 公共接口允许匿名访问
            )
            .csrf(csrf -> csrf.disable())  // 禁用CSRF
            .cors(Customizer.withDefaults());  // 启用CORS
        
        return http.build();
    }
    
    /**
     * 客户端ID验证拦截器
     */
    @Component
    public class ClientIdValidationInterceptor implements HandlerInterceptor {
        
        @Override
        public boolean preHandle(HttpServletRequest request, HttpServletResponse response, Object handler) throws Exception {
            String clientId = request.getParameter("clientId");
            
            if (StringUtils.isEmpty(clientId)) {
                response.setStatus(HttpStatus.BAD_REQUEST.value());
                response.getWriter().write("clientId参数不能为空");
                return false;
            }
            
            // 验证clientId格式
            if (!clientId.matches("^[a-zA-Z0-9_-]{6,50}$")) {
                response.setStatus(HttpStatus.BAD_REQUEST.value());
                response.getWriter().write("clientId格式错误");
                return false;
            }
            
            return true;
        }
    }
}
```

### 2. **限流保护**

```
@Configuration
public class RateLimitConfig {
    
    /**
     * 公共接口限流
     */
    @Bean
    public FilterRegistrationBean<RateLimitFilter> rateLimitFilter() {
        FilterRegistrationBean<RateLimitFilter> registration = new FilterRegistrationBean<>();
        registration.setFilter(new RateLimitFilter());
        registration.addUrlPatterns("/api/public/*");
        registration.setOrder(1);
        return registration;
    }
}

@Component
@Slf4j
public class RateLimitFilter extends OncePerRequestFilter {
    
    @Autowired
    private RedisTemplate<String, String> redisTemplate;
    
    @Override
    protected void doFilterInternal(HttpServletRequest request, 
                                   HttpServletResponse response, 
                                   FilterChain filterChain) throws IOException {
        
        String clientId = request.getParameter("clientId");
        String ip = getClientIp(request);
        String key = "rate:limit:public:" + clientId + ":" + ip;
        
        // 限流：每个客户端IP每分钟最多60次请求
        Long count = redisTemplate.opsForValue().increment(key, 1);
        
        if (count == 1) {
            redisTemplate.expire(key, 1, TimeUnit.MINUTES);
        }
        
        if (count > 60) {
            log.warn("接口限流触发，clientId={}, ip={}", clientId, ip);
            response.setStatus(HttpStatus.TOO_MANY_REQUESTS.value());
            response.getWriter().write("请求过于频繁，请稍后再试");
            return;
        }
        
        filterChain.doFilter(request, response);
    }
    
    private String getClientIp(HttpServletRequest request) {
        // 获取真实IP
        return request.getRemoteAddr();
    }
}
```

### 3. **监控与日志**

```
@Aspect
@Component
@Slf4j
public class PublicApiMonitorAspect {
    
    @Around("execution(* com.yourpackage.controller.PublicApiController.*(..))")
    public Object monitorApi(ProceedingJoinPoint joinPoint) throws Throwable {
        String methodName = joinPoint.getSignature().getName();
        String clientId = getClientId(joinPoint);
        long startTime = System.currentTimeMillis();
        
        try {
            Object result = joinPoint.proceed();
            long costTime = System.currentTimeMillis() - startTime;
            
            // 记录成功日志
            log.info("公共接口调用成功 - 方法: {}, clientId: {}, 耗时: {}ms", 
                methodName, clientId, costTime);
            
            // 监控指标
            Metrics.counter("public.api.call", 
                "method", methodName,
                "clientId", clientId,
                "status", "success"
            ).increment();
            
            return result;
            
        } catch (Exception e) {
            long costTime = System.currentTimeMillis() - startTime;
            
            // 记录失败日志
            log.error("公共接口调用失败 - 方法: {}, clientId: {}, 耗时: {}ms, 错误: {}", 
                methodName, clientId, costTime, e.getMessage(), e);
            
            // 监控指标
            Metrics.counter("public.api.call", 
                "method", methodName,
                "clientId", clientId,
                "status", "error"
            ).increment();
            
            throw e;
        }
    }
    
    private String getClientId(ProceedingJoinPoint joinPoint) {
        Object[] args = joinPoint.getArgs();
        for (Object arg : args) {
            if (arg instanceof HttpServletRequest) {
                return ((HttpServletRequest) arg).getParameter("clientId");
            }
        }
        return "unknown";
    }
}

## 总结

通过以上方案，您可以实现：

1. **多租户数据隔离**：通过 `clientId`→ `tenantId`→ 租户数据
2. **公共接口无状态**：无需用户登录即可访问
3. **高性能缓存**：Redis缓存 + 数据库查询优化
4. **安全防护**：参数验证 + 接口限流
5. **监控告警**：完整日志 + 性能监控

这种设计可以支持多小程序共用一套后端服务，每个小程序通过不同的 `clientId`获取自己租户的个性化数据。

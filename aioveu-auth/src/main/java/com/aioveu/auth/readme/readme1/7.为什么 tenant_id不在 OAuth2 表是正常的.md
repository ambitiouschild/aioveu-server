## 四、✅ Auth 微服务里的正确校验逻辑

```
public boolean isValid(String clientId) {

    // 1️⃣ Redis 白名单
    Boolean cached = redisTemplate.opsForSet()
            .isMember(CACHE_KEY, clientId);

    if (Boolean.TRUE.equals(cached)) {
        return true;
    }

    // 2️⃣ 查 OAuth2 表
    Oauth2RegisteredClient client =
            oauth2RegisteredClientMapper.selectOne(
                    new LambdaQueryWrapper<Oauth2RegisteredClient>()
                            .eq(Oauth2RegisteredClient::getClientId, clientId)
            );

    if (client == null) {
        return false;
    }

    // 3️⃣ 校验 enabled
    boolean valid = client.getEnabled() == 1;

    if (valid) {
        redisTemplate.opsForSet()
                .add(CACHE_KEY, clientId);
    }

    return valid;
}
```

✅ **Auth 微服务不关心 tenant**

✅ **只认 clientId + enabled**

这里不是你写错，而是Spring Security 的 `Jwt` 根本没有 `getClaimAsLong` 这个方法，这是我们在前面已经踩过一次的坑 👍  
你现在这个报错是必然的。

我直接给你 ✅ 能编译、✅ 不踩坑、✅ 生产级 的改法。

## ✅ 一句话结论（先拍死）

> Spring Security 的 `Jwt` 只有：
>
> ```java
> Object getClaim(String name)
> <T> T getClaim(String name, Class<T> type)
> ```
>
> 没有 `getClaimAsLong`

---

## 二、你现在这段代码的“正确改法”

### ❌ 你现在写的（编译不过）

```java
Long userId = jwt.getClaimAsLong(JwtClaimConstants.User.ID);
Long tokenVersion = jwt.getClaimAsLong(JwtClaimConstants.Token.VERSION);
```

---

### ✅ 正确写法（推荐 ✅）

```java
Long userId = jwt.getClaim(JwtClaimConstants.User.ID, Long.class);
Long tokenVersion = jwt.getClaim(JwtClaimConstants.Token.VERSION, Long.class);
```

---

## 三、⚠️ 但你必须知道这个“隐藏雷区”（非常重要）

### ❗ Spring Authorization Server / Nimbus

JSON number 默认会被解析成 `Integer`，不是 `Long`

所以更稳的写法是👇（强烈推荐你用这个）

```java
Long userId = getLongClaim(jwt, JwtClaimConstants.User.ID);
Long tokenVersion = getLongClaim(jwt, JwtClaimConstants.Token.VERSION);
```

### ✅ 工具方法（一劳永逸）

```java
private Long getLongClaim(Jwt jwt, String claimName) {
    Object value = jwt.getClaim(claimName);
    if (value instanceof Integer i) {
        return i.longValue();
    }
    if (value instanceof Long l) {
        return l;
    }
    return null;
}
```

✅ 不抛 `ClassCastException`  
✅ 不依赖 Spring 内部实现  
✅ 所有 5.x / 6.x / 7.x 通用

## 四、✅ 完整示例（你直接复制）

```java
Long userId = getLongClaim(jwt, JwtClaimConstants.User.ID);
Long tokenVersion = getLongClaim(jwt, JwtClaimConstants.Token.VERSION);

if (userId == null || tokenVersion == null) {
    throw new OAuth2AuthenticationException(
        new OAuth2Error("invalid_token", "Token 缺失必要字段", null)
    );
}

String versionKey = RedisConstants.Auth.USER_TOKEN_VERSION + userId;
Long currentVersion = (Long) redisTemplate.opsForValue().get(versionKey);

if (currentVersion == null || !currentVersion.equals(tokenVersion)) {
    throw new OAuth2AuthenticationException(
        new OAuth2Error("invalid_token", "用户已被强制下线", null)
    );
}
```

## 五、为什么你 TokenManager 里能用 `getClaimAsLong`

因为那是 Hutool：

```java
cn.hutool.jwt.JWT
```

👉 不是 Spring Security 的 `Jwt`

这两个类：
• 同名

• 不同包

• API 完全不同

### ✅ Auth 服务里动态读取（最终版）

```java
@RefreshScope
@ConfigurationProperties(prefix = "auth")
@Component
@Data
public class AuthWhiteListProperties {
    private List<String> whiteList = new ArrayList<>();
}
```

```java
@Bean
@Order(2)
public SecurityFilterChain defaultSecurityFilterChain(
        HttpSecurity http,
        AuthWhiteListProperties whiteListProperties
) throws Exception {

    List<RequestMatcher> matchers = whiteListProperties.getWhiteList()
            .stream()
            .map(AntPathRequestMatcher::antMatcher)
            .toList();

    http
        .securityMatcher("/aioveu/api/v8/**")
        .authorizeHttpRequests(authorize -> authorize
                .requestMatchers(matchers.toArray(new RequestMatcher[0]))
                .permitAll()
                .anyRequest()
                .authenticated()
        )
        .csrf(csrf -> csrf.disable());

    return http.build();
}
```

✅ Nacos 改 → 实时生效
✅ 不用再动 Security 配置

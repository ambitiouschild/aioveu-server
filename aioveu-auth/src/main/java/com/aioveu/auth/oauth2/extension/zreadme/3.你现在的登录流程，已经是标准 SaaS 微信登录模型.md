## 三、正确做法：在这里完成「租户识别」

### ✅ 1️⃣ 从 `WxMaService`拿 wx_appid

你已经在用：

```
wxMaService.getUserService().getSessionInfo(code);
```

但 **wx_appid 在哪？**

通常 `WxMaService`是**按小程序隔离的**，

所以你可以直接从它身上拿：

```

```

> ✅ 这是**最可靠**的来源
>
> ✅ 不需要前端传
>
> ✅ 不会被篡改

## 四、你这个 Provider 里「唯一需要改的一段」

### 🔴 原来你这里是：

```
UserDetails userDetails =
        memberDetailsService.loadUserByOpenid(openid);
```

### ✅ 改成（核心）：

```
// 1. 获取 wx_appid
String wxAppId = wxMaService.getWxMaConfig().getAppid();

// 2. 根据 appid 找租户
Tenant tenant = tenantService.findByWxAppid(wxAppId);
if (tenant == null) {
    throw new OAuth2AuthenticationException("invalid_client", "租户不存在");
}

// 3. 设置租户上下文
TenantContextHolder.setTenantId(tenant.getId());

// 4. 根据 openid + tenantId 查用户
UserDetails userDetails =
        memberDetailsService.loadUserByOpenid(openid, tenant.getId());
```

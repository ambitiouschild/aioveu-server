## 五、正确的整体流程（和你当前架构对齐）

### ✅ 1️⃣ 小程序登录接口（你还没贴，但应该有）

```
@PostMapping("/login/wx")
public LoginResp wxLogin(@RequestParam String code) {

    // 1. code2Session
    WxSession session = wxService.code2Session(code);

    // 2. 根据 wx_appid 找租户
    Tenant tenant = tenantService.findByWxAppid(session.getAppid());

    // 3. 设置租户上下文（线程级）
    TenantContextHolder.setTenantId(tenant.getId());

    // 4. 用 openid + tenant_id 查/建用户
    User user = userService.findOrCreate(
        session.getOpenid(),
        tenant.getId()
    );

    // 5. 生成 token
    return tokenService.generate(user, tenant);
}
```

✅ **tenant_id 只在这层出现**

✅ **后续所有 SQL 自动带 tenant_id**

## ✅ 最终正确版（你这个就是对的）

```
spring:
  security:
    oauth2:
      authorizationserver:
        token-uri: ${gateway.endpoint}/oauth2/token
      resourceserver:
        jwt:
          jwk-set-uri: ${gateway.endpoint}/oauth2/jwks
```

### ✅ 为什么这次是对的

| 项目                                 | 是否正确  | 原因                      |
| ------------------------------------ | --------- | ------------------------- |
| token-uri                            | ✅         | 指向网关 `/oauth2/token`  |
| jwk-set-uri                          | ✅         | 指向网关 `/oauth2/jwks`   |
| 是否包含服务名                       | ❌（正确） | 不拼 `aioveu-tenant-auth` |
| 是否符合 Spring Authorization Server | ✅         | 协议端点固定路径          |
| 是否适合多微服务共用                 | ✅         | 所有服务只认网关          |

## 二、必须配套的 Gateway 路由（缺一不可）

### ✅ Gateway 必须有这两条

```
spring:
  cloud:
    gateway:
      routes:
        # 令牌端点
        - id: auth-token
          uri: lb://aioveu-tenant-auth
          predicates:
            - Path=/oauth2/token
          filters:
            - StripPrefix=0

        # 公钥端点（JWKS）
        - id: auth-jwks
          uri: lb://aioveu-tenant-auth
          predicates:
            - Path=/oauth2/jwks
          filters:
            - StripPrefix=0
```

## 五、顺便帮你纠一个常见误区（很重要）

```
jwk-set-uri: ${gateway.endpoint}/aioveu-tenant-auth/oauth2/jwks
```

🚫 错在：

- 

  `aioveu-tenant-auth`是 **服务名**

- 

  不是 **context-path**

- 

  Spring Authorization Server **永远不会识别**

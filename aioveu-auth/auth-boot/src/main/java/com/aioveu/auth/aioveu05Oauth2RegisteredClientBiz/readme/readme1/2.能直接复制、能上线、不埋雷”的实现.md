## ✅ 一、`disableClient` 的标准实现（推荐）

### ✅ Mapper 层（MyBatis‑Plus）

```java
@Mapper
public interface Oauth2RegisteredClientBizMapper
        extends BaseMapper<Oauth2RegisteredClientBiz> {

    int disable(@Param("clientId") String clientId);
}
```

## ✅ 二、Service 层（✅ 你刚才要的最终版）

```java
@Slf4j
@Service
@RequiredArgsConstructor
public class Oauth2RegisteredClientBizServiceImpl
        implements Oauth2RegisteredClientBizService {

    private final Oauth2RegisteredClientBizMapper baseMapper;
    private final RedisTemplate<String, Object> redisTemplate;

    private static final String CLIENT_WHITELIST_KEY = "auth:client:whitelist";

    @Transactional(rollbackFor = Exception.class)
    @Override
    public void disableClient(String clientId) {

        // 1️⃣ 禁用 DB
        int rows = baseMapper.disable(clientId);
        if (rows == 0) {
            throw new BusinessException("客户端不存在或已禁用");
        }

        // 2️⃣ ✅ 同步删除 Redis 白名单
        Long removed = redisTemplate.opsForSet()
                .remove(CLIENT_WHITELIST_KEY, clientId);

        log.info("【Auth】禁用 clientId={}，DB影响行数={}，Redis移除={}",
                clientId, rows, removed);
    }
}
```

✅ 事务一致  
✅ DB / Redis 同成功或同失败  
✅ 日志可排查

## ✅ 三、启用（对称实现，顺便给你）

### Mapper

```xml
<update id="enable" parameterType="string">
    UPDATE oauth2_registered_client_biz
    SET
        enabled = 1,
        is_deleted = 0,
        update_time = NOW()
    WHERE
        client_id = #{clientId}
</update>
```

### Service

```java
@Transactional(rollbackFor = Exception.class)
@Override
public void enableClient(String clientId) {

    baseMapper.enable(clientId);

    redisTemplate.opsForSet()
            .add(CLIENT_WHITELIST_KEY, clientId);

    log.info("【Auth】启用 clientId={}，已加入 Redis 白名单", clientId);
}
```

# ✅ 一、`WechatAuthenticationToken / Converter`完整实现

## 1️⃣ WechatAuthenticationToken（认证令牌）

```
package com.aioveu.auth.oauth2.extension.wechat;

import org.springframework.security.authentication.AbstractAuthenticationToken;
import org.springframework.security.core.Authentication;
import org.springframework.security.core.SpringSecurityCoreVersion;
import org.springframework.util.Assert;

import java.io.Serial;
import java.util.Collections;
import java.util.Map;

/**
 * 微信小程序登录认证令牌
 */
public class WechatAuthenticationToken extends AbstractAuthenticationToken {

    @Serial
    private static final long serialVersionUID = SpringSecurityCoreVersion.SERIAL_VERSION_UID;

    /** 客户端认证信息 */
    private final Authentication clientPrincipal;

    /** 微信 code */
    private final String code;

    /** 附加参数 */
    private final Map<String, Object> additionalParameters;

    /** 微信小程序授权类型 */
    public static final AuthorizationGrantType WECHAT_MINI_APP =
            new AuthorizationGrantType("wechat");

    public WechatAuthenticationToken(
            Authentication clientPrincipal,
            String code
    ) {
        this(clientPrincipal, code, Collections.emptyMap());
    }

    public WechatAuthenticationToken(
            Authentication clientPrincipal,
            String code,
            Map<String, Object> additionalParameters
    ) {
        super(Collections.emptyList());
        Assert.notNull(clientPrincipal, "clientPrincipal cannot be null");
        Assert.hasText(code, "code cannot be empty");
        this.clientPrincipal = clientPrincipal;
        this.code = code;
        this.additionalParameters = additionalParameters;
        setAuthenticated(false);
    }

    @Override
    public Object getCredentials() {
        return this.code;
    }

    @Override
    public Object getPrincipal() {
        return this.clientPrincipal;
    }

    public String getCode() {
        return this.code;
    }

    public Map<String, Object> getAdditionalParameters() {
        return this.additionalParameters;
    }
}
```

## 2️⃣ WechatAuthenticationConverter（参数解析）

```
package com.aioveu.auth.oauth2.extension.wechat;

import com.aioveu.auth.util.OAuth2EndpointUtils;
import org.springframework.security.core.Authentication;
import org.springframework.security.oauth2.core.AuthorizationGrantType;
import org.springframework.security.oauth2.core.OAuth2ErrorCodes;
import org.springframework.security.oauth2.core.endpoint.OAuth2ParameterNames;
import org.springframework.security.web.authentication.AuthenticationConverter;
import org.springframework.util.MultiValueMap;

import jakarta.servlet.http.HttpServletRequest;

import static com.aioveu.auth.oauth2.extension.wechat.WechatAuthenticationToken.WECHAT_MINI_APP;

public class WechatAuthenticationConverter implements AuthenticationConverter {

    @Override
    public Authentication convert(HttpServletRequest request) {

        String grantType = request.getParameter(OAuth2ParameterNames.GRANT_TYPE);
        if (!WECHAT_MINI_APP.getValue().equals(grantType)) {
            return null;
        }

        MultiValueMap<String, String> params =
                OAuth2EndpointUtils.getParameters(request);

        String code = params.getFirst(OAuth2ParameterNames.CODE);
        if (code == null || code.isBlank()) {
            OAuth2EndpointUtils.throwError(
                    OAuth2ErrorCodes.INVALID_REQUEST,
                    OAuth2ParameterNames.CODE,
                    OAuth2EndpointUtils.ACCESS_TOKEN_REQUEST_ERROR_URI
            );
        }

        Authentication clientPrincipal =
                org.springframework.security.oauth2.server.authorization.web.OAuth2EndpointUtils
                        .getAuthenticatedClient(request);

        return new WechatAuthenticationToken(
                clientPrincipal,
                code,
                params.toSingleValueMap()
        );
    }
}
```

✅ **Converter 只做参数解析**

✅ **Provider 才处理租户 & 用户**

### ✅ 方式一（推荐）：**按租户动态获取**

```
@Component
public class WxMaServiceFactory {

    private final Map<String, WxMaService> serviceMap = new ConcurrentHashMap<>();

    public WxMaService getByAppid(String appid) {
        return serviceMap.computeIfAbsent(appid, k -> {
            WxMaDefaultConfigImpl config = new WxMaDefaultConfigImpl();
            // 从数据库 / 配置中心加载
            config.setAppid(k);
            config.setSecret(secretMap.get(k));
            WxMaServiceImpl service = new WxMaServiceImpl();
            service.setWxMaConfig(config);
            return service;
        });
    }
}
```

# ✅ 三、`TenantContextHolder`在 OAuth 链路会不会丢？

## ✅ 结论先给

> ✅ **不会丢，只要你用的是 `ThreadLocal`+ 同步 Filter**
>
> ❌ **会丢，如果你用了 WebFlux / 异步 / @Async / CompletableFuture**

------

## ✅ 正确用法（必须）

### 1️⃣ 清除 ThreadLocal（非常重要）

```
@Component
public class TenantContextFilter implements Filter {

    @Override
    public void doFilter(ServletRequest request,
                         ServletResponse response,
                         FilterChain chain)
            throws IOException, ServletException {

        try {
            chain.doFilter(request, response);
        } finally {
            TenantContextHolder.clear();
        }
    }
}
```

✅ **否则会内存泄漏 + 串租户**

------

### 2️⃣ OAuth 链路是同步的 ✅

Spring Authorization Server **默认是同步 Servlet**

👉 所以 **ThreadLocal 完全安全**

# ✅ 四、一个租户支持多个小程序，表怎么改？

## ✅ 正确模型（强烈推荐）

### 🔴 不要改 `sys_user`

### ✅ 新建「租户 – 小程序」中间表

```
CREATE TABLE sys_tenant_wx_app (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    tenant_id BIGINT NOT NULL,
    wx_appid VARCHAR(32) NOT NULL,
    is_default TINYINT DEFAULT 0,
    create_time DATETIME,
    UNIQUE KEY uk_wx_appid (wx_appid),
    KEY idx_tenant_id (tenant_id)
);
```

------

## ✅ 查询逻辑变成

```
Tenant tenant = tenantService.findByWxAppid(wxAppId);
```

```
SELECT t.*
FROM sys_tenant t
JOIN sys_tenant_wx_app a ON t.id = a.tenant_id
WHERE a.wx_appid = ?
```

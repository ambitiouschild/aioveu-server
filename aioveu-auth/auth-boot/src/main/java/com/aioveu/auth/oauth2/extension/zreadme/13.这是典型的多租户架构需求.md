我现在只需要前端用户登录的时候通过clientId，调用后端获取对应的wxAppid，然后通过wxAppid获取对应的tenantId，这样通过tenantId我就可以实现不同的前端小程序对应不同的租户

明白了！你需要**在前端登录时，通过 `clientId`获取对应的 `wxAppid`和 `tenantId`**，这样就能实现多租户隔离。这是典型的多租户架构需求。

## 一、整体流程设计

```
前端登录流程：
1. 用户打开小程序 → 获取微信 code
2. 前端携带 code + clientId → 调用后端登录接口
3. 后端通过 clientId → 查询 wxAppid + tenantId
4. 后端用 wxAppid 调用微信 API 验证 code → 获取 openid
5. 后端结合 tenantId + openid → 查询用户信息
6. 返回包含 tenantId 的 token
```

## 二、具体实现方案

### 方案A：在现有 Provider 中修改（推荐）

```
@Override
public Authentication authenticate(Authentication authentication) throws AuthenticationException {
    log.info("=====微信授权认证 Provider=====");
    
    // 1. 获取认证令牌
    WechatAuthenticationToken wechatAuthenticationToken = (WechatAuthenticationToken) authentication;
    
    // 2. 客户端认证
    OAuth2ClientAuthenticationToken clientPrincipal = OAuth2AuthenticationProviderUtils
            .getAuthenticatedClientElseThrowInvalidClient(wechatAuthenticationToken);
    RegisteredClient registeredClient = clientPrincipal.getRegisteredClient();
    
    // 3. 验证授权类型
    if (!registeredClient.getAuthorizationGrantTypes().contains(WechatAuthenticationToken.WECHAT_MINI_APP)) {
        throw new OAuth2AuthenticationException(OAuth2ErrorCodes.UNAUTHORIZED_CLIENT);
    }
    
    // 4. 获取请求参数
    Map<String, Object> additionalParameters = wechatAuthenticationToken.getAdditionalParameters();
    String code = (String) additionalParameters.get(OAuth2ParameterNames.CODE);
    
    // 5. ★ 关键修改：获取 clientId
    String clientId = registeredClient.getClientId();  // OAuth2 客户端ID
    
    log.info("获取客户端ID: {}", clientId);
    
    // 6. ★ 通过 clientId 查询 wxAppid 和 tenantId
    // 这里需要你实现数据库查询
    TenantWxAppInfo tenantWxAppInfo = getTenantWxAppInfoByClientId(clientId);
    if (tenantWxAppInfo == null) {
        throw new OAuth2AuthenticationException("无效的客户端ID");
    }
    
    String wxAppid = tenantWxAppInfo.getWxAppid();
    String tenantId = tenantWxAppInfo.getTenantId();
    
    log.info("查询到租户信息 - wxAppid: {}, tenantId: {}", wxAppid, tenantId);
    
    // 7. ★ 动态设置微信配置
    // 注意：这里需要根据 wxAppid 动态获取对应的 WxMaService
    WxMaService wxMaServiceForApp = getWxMaServiceByAppId(wxAppid);
    
    // 8. 调用微信API获取openid
    WxMaJscode2SessionResult sessionInfo;
    try {
        sessionInfo = wxMaServiceForApp.getUserService().getSessionInfo(code);
    } catch (WxErrorException e) {
        log.error("微信小程序登录失败, appid: {}", wxAppid, e);
        throw new OAuth2AuthenticationException(e.getMessage());
    }
    
    String openid = sessionInfo.getOpenid();
    log.info("获取到openid: {}", openid);
    
    // 9. ★ 使用 tenantId + openid 加载用户
    UserDetails userDetails = memberDetailsService.loadUserByOpenidAndTenantId(openid, tenantId);
    log.info("加载用户信息: {}", userDetails.getUsername());
    
    // 10. 将 tenantId 设置到用户信息中
    if (userDetails instanceof CustomUserDetails) {
        ((CustomUserDetails) userDetails).setTenantId(tenantId);
        ((CustomUserDetails) userDetails).setWxAppid(wxAppid);
    }
    
    // 11. 后续令牌生成逻辑保持不变...
    // ... [原有代码]
    
    return new OAuth2AccessTokenAuthenticationToken(
            registeredClient,
            clientPrincipal,
            accessToken,
            refreshToken,
            additionalParameters);
}
```

## 三、数据库查询实现

创建查询服务：

```
@Service
@Slf4j
public class TenantAppConfigService {
    
    @Autowired
    private JdbcTemplate jdbcTemplate;
    
    /**
     * 通过 clientId 查询租户和小程序配置
     */
    public TenantWxAppInfo getTenantWxAppInfo(String clientId) {
        String sql = """
            SELECT 
                m.wx_appid,
                m.tenant_id,
                t.tenant_name,
                m.client_id
            FROM mini_program_config m
            LEFT JOIN tenant t ON m.tenant_id = t.id
            WHERE m.client_id = ? 
            AND m.is_deleted = 0
            LIMIT 1
            """;
        
        try {
            return jdbcTemplate.queryForObject(sql, (rs, rowNum) -> 
                TenantWxAppInfo.builder()
                    .wxAppid(rs.getString("wx_appid"))
                    .tenantId(rs.getString("tenant_id"))
                    .tenantName(rs.getString("tenant_name"))
                    .clientId(rs.getString("client_id"))
                    .build(), 
                clientId
            );
        } catch (EmptyResultDataAccessException e) {
            log.warn("未找到对应的租户小程序配置, clientId: {}", clientId);
            return null;
        }
    }
    
    /**
     * 通过 wxAppid 查询租户ID
     */
    public String getTenantIdByWxAppid(String wxAppid) {
        String sql = "SELECT tenant_id FROM mini_program_config WHERE wx_appid = ? AND is_deleted = 0";
        try {
            return jdbcTemplate.queryForObject(sql, String.class, wxAppid);
        } catch (EmptyResultDataAccessException e) {
            return null;
        }
    }
}
```

## 四、修改 MemberDetailsService

```
@Service
public class MemberDetailsServiceImpl implements MemberDetailsService {
    
    @Autowired
    private UserMapper userMapper;
    
    @Override
    public UserDetails loadUserByOpenid(String openid) {
        // 原有实现，不修改
        return loadUserByOpenidAndTenantId(openid, null);
    }
    
    /**
     * 新增：通过 openid + tenantId 查询用户
     */
    public UserDetails loadUserByOpenidAndTenantId(String openid, String tenantId) {
        // 构建查询条件
        QueryWrapper<User> queryWrapper = new QueryWrapper<>();
        queryWrapper.eq("openid", openid);
        
        if (StringUtils.hasText(tenantId)) {
            queryWrapper.eq("tenant_id", tenantId);  // 多租户隔离查询
        }
        
        User user = userMapper.selectOne(queryWrapper);
        
        if (user == null) {
            // 用户不存在，可以自动创建
            user = createNewUser(openid, tenantId);
        }
        
        return new CustomUserDetails(
            user.getId(),
            user.getUsername(),
            user.getPassword(),
            user.getOpenid(),
            tenantId,  // 设置租户ID
            getAuthorities(user.getRole())
        );
    }
    
    /**
     * 自动创建新用户
     */
    private User createNewUser(String openid, String tenantId) {
        User newUser = new User();
        newUser.setOpenid(openid);
        newUser.setTenantId(tenantId);
        newUser.setUsername("user_" + openid.substring(0, 8));
        newUser.setPassword(UUID.randomUUID().toString());  // 随机密码
        
        userMapper.insert(newUser);
        return newUser;
    }
}
```

```
async function wechatLogin(code) {
  const response = await fetch('/oauth2/token', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/x-www-form-urlencoded',
    },
    body: new URLSearchParams({
      'grant_type': 'wechat_mini_app',  // 你的自定义授权类型

  });

  return data;
}
```

## 七、总结实现步骤

1. **修改 Provider**：
   - 在 `authenticate()`方法中获取 `clientId`
   - 通过 `clientId`查询 `wxAppid`和 `tenantId`
   - 动态获取对应小程序的 `WxMaService`
   - 使用 `tenantId + openid`查询用户
2. **创建查询服务**：
   - 实现 `TenantAppConfigService`查询数据库
   - 缓存查询结果，提高性能
3. **扩展 UserDetails**：
   - 添加 `tenantId`字段
   - 修改 `MemberDetailsService`支持租户查询
4. **配置多小程序**：
   - 实现根据 `wxAppid`动态获取微信配置
   - 可以使用 `WxMaConfig`和 `WxMaService`的多例模式

这样就能实现：**前端小程序 → clientId → wxAppid → tenantId → 租户隔离数据** 的完整链路。

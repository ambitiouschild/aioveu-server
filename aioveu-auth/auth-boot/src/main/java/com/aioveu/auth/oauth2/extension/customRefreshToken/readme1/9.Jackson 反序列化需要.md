现在问题清楚了！你的 `MemberDetails`类只有**一个带参数的构造函数**，**没有无参构造函数**，而且**没有 `@JsonCreator`注解**。

## 🔍 问题分析

Jackson 反序列化需要：

1. **无参构造函数** + setter 方法（通过反射设置字段）
2. **或者** 有 `@JsonCreator`注解的构造函数

你的 `MemberDetails`类：

- 只有 `public MemberDetails(MemberAuthDTO memAuthInfo)`一个构造函数
- 没有无参构造函数
- 没有 `@JsonCreator`注解
- 字段没有 setter 方法（虽然有 `@Data`生成）

### 方案3：添加 `@JsonCreator`注解

```
@Data
public class MemberDetails implements UserDetails {
    
    private Long id;
    private String username;
    private Long tenantId;
    private String wxAppid;
    private Boolean enabled;
    private String authenticationIdentity;
    
    // ✅ 添加 @JsonCreator 注解的构造函数
    @JsonCreator
    public MemberDetails(
        @JsonProperty("id") Long id,
        @JsonProperty("username") String username,
        @JsonProperty("tenantId") Long tenantId,
        @JsonProperty("wxAppid") String wxAppid,
        @JsonProperty("enabled") Boolean enabled,
        @JsonProperty("authenticationIdentity") String authenticationIdentity
    ) {
        this.id = id;
        this.username = username;
        this.tenantId = tenantId;
        this.wxAppid = wxAppid;
        this.enabled = enabled;
        this.authenticationIdentity = authenticationIdentity;
    }
    
    public MemberDetails(MemberAuthDTO memAuthInfo) {
        this.setId(memAuthInfo.getId());
        this.setUsername(memAuthInfo.getUsername());
        this.setTenantId(memAuthInfo.getTenantId());
        this.setEnabled(GlobalConstants.STATUS_YES.equals(memAuthInfo.getStatus()));
    }
    
    // ... 其他方法
}
```

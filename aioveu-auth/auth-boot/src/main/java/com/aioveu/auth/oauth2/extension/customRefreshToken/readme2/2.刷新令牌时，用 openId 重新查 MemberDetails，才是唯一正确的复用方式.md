  • 第二次请求

  • 你强转 → `ClassCastException`

## 七、为什么 Spring 官方示例从不存 Authentication？

因为：

• Token 可能跨 JVM

• 用户可能被禁用

• 权限可能变化

• 反序列化不可控


👉 Authorization 只存 identity，不存 Authentication

这是 OAuth2 / OIDC / Spring Authorization Server 的铁律

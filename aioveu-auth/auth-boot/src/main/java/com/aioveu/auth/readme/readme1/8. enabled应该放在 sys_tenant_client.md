## 三、✅ Auth 微服务里的“聚合校验逻辑”

```
public boolean isValid(String clientId) {

    // 1️⃣ Redis 白名单
    Boolean cached = redisTemplate.opsForSet()
            .isMember(CACHE_KEY, clientId);

    if (Boolean.TRUE.equals(cached)) {
        return true;
    }

    // 2️⃣ 查业务表
    SysTenantClient tenantClient =
            tenantClientMapper.selectOne(
                    new LambdaQueryWrapper<SysTenantClient>()
                            .eq(SysTenantClient::getClientId, clientId)
            );

    if (tenantClient == null) {
        return false;
    }

    boolean valid = tenantClient.getEnabled() == 1;

    if (valid) {
        redisTemplate.opsForSet()
                .add(CACHE_KEY, clientId);
    }

    return valid;
}
```

✅ **Auth 微服务不关心 tenant**

✅ **只关心 clientId + enabled**

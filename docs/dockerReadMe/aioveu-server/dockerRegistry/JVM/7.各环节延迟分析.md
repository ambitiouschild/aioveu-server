### 1. **验证码生成异步化**

```
// Auth 服务 - 立即返回，后台生成
@PostMapping("/captcha/async")
public CaptchaAsyncResponse sendCaptchaAsync(@RequestBody CaptchaRequest request) {
    String requestId = UUID.randomUUID().toString();
    
    // 提交到线程池异步处理
    executor.submit(() -> {
        generateAndSendCaptcha(request.getPhone());
    });
    
    return CaptchaAsyncResponse.builder()
        .requestId(requestId)
        .status("PROCESSING")
        .message("验证码发送中")
        .timestamp(System.currentTimeMillis())
        .build();
}
```

**前端立即显示“已发送”，后台轮询结果。**

### 2. **预生成验证码池（减少实时计算）**

```
@Component
public class CaptchaPool {
    private BlockingQueue<Captcha> pool = new LinkedBlockingQueue<>(1000);
    
    @PostConstruct
    public void init() {
        // 后台线程不断补充验证码
        Executors.newSingleThreadExecutor().submit(() -> {
            while (true) {
                pool.offer(generateCaptcha()); // 预生成
                Thread.sleep(100); // 控制生成速度
            }
        });
    }
    
    public Captcha getCaptcha() throws InterruptedException {
        return pool.poll(1, TimeUnit.SECONDS); // 从池中取
    }
}
```

**验证码生成从“实时计算”变成“内存池取”，速度提升 10 倍+**

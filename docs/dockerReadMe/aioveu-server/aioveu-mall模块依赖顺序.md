package和 compile的区别：

## **生命周期阶段对比**

| 阶段          | 作用                         | 是否解决依赖问题     |
| ------------- | ---------------------------- | -------------------- |
| **`compile`** | 只编译源代码                 | ❌ 不会解决依赖问题   |
| **`package`** | 编译 + 打包（生成jar/war）   | ❌ 不会解决依赖问题   |
| **`install`** | 编译 + 打包 + 安装到本地仓库 | ✅ **会解决依赖问题** |

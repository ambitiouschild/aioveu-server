      username: aioveu
      password: 775825
  # Seata配置中心
  config:
    type: nacos

      server-addr: nacos:8848
      namespace: 80147849-cdb8-4b7d-befc-b5aefa0e9ab6
      group: SEATA_GROUP
      data-id: seataServer.properties

## 🎯 为什么这样最有效？

### 配置加载时间线对比：

#### 之前（有问题）：

```
bootstrap.yml → 只配Nacos认证
    ↓
application.yml → 不配Seata
    ↓
Seata初始化 → 用默认值 → 失败 ❌
    ↓
Nacos配置加载 → 重新初始化 → 成功 ✅
```

#### 现在（在bootstrap.yml配Seata）：

```
bootstrap.yml → 配Nacos认证 + Seata完整配置
    ↓
Seata初始化 → 直接用正确配置 → 直接成功 ✅
    ↓
Nacos配置加载 → 如果需要可以覆盖
```

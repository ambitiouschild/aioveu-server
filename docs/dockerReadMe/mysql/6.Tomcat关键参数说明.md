## 🎯 关键参数说明

| 参数                 | 默认值 | 建议值 | 说明                              |
| -------------------- | ------ | ------ | --------------------------------- |
| `max-threads`        | 200    | 500    | 最大工作线程数，根据服务器CPU调整 |
| `min-spare-threads`  | 10     | 50     | 最小空闲线程，避免频繁创建线程    |
| `accept-count`       | 100    | 200    | 等待队列长度，不宜过长            |
| `max-connections`    | 10000  | 10000  | 最大并发连接数                    |
| `connection-timeout` | 20000  | 10000  | 连接超时时间（毫秒）              |

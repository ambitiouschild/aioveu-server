## 二、具体实现方案

### **方案1：Spring Boot 基础实现**

```
import org.springframework.http.CacheControl;
import org.springframework.http.ResponseEntity;
import org.springframework.web.bind.annotation.*;

@RestController
@RequestMapping("/api/images")
public class ImageController {
    
    @GetMapping("/{id}")
    public ResponseEntity<byte[]> getImage(@PathVariable String id) {
        byte[] imageData = imageService.getImage(id);
        
        return ResponseEntity.ok()
            // 核心缓存配置
            .cacheControl(CacheControl.maxAge(1, TimeUnit.HOURS))  // 缓存1小时
            .contentType(MediaType.IMAGE_JPEG)  // 根据实际类型
            .body(imageData);
    }
}
```

### **方案2：完整缓存配置（推荐）**

```
@GetMapping("/{id}")
public ResponseEntity<byte[]> getImage(
        @PathVariable String id,
        @RequestHeader(value = "If-None-Match", required = false) String ifNoneMatch,
        HttpServletResponse response) {
    
    ImageInfo image = imageService.getImageInfo(id);
    
    // 1. 生成ETag（基于内容哈希）
    String eTag = "\"" + DigestUtils.md5DigestAsHex(image.getData()) + "\"";
    
    // 2. 检查客户端缓存是否有效
    if (eTag.equals(ifNoneMatch)) {
        return ResponseEntity.status(HttpStatus.NOT_MODIFIED).build();  // 返回304
    }
    
    // 3. 返回图片和缓存头
    return ResponseEntity.ok()
        .cacheControl(CacheControl.maxAge(7, TimeUnit.DAYS)
            .cachePublic()
            .mustRevalidate())
        .eTag(eTag)  // ETag用于缓存验证
        .lastModified(image.getUpdateTime().toEpochMilli())  // 最后修改时间
        .contentType(MediaType.parseMediaType(image.getContentType()))
        .contentLength(image.getData().length)
        .body(image.getData());
}
```

### **方案3：不同图片类型的差异化缓存**

```
@GetMapping("/{type}/{id}")
public ResponseEntity<byte[]> getImageByType(
        @PathVariable String type,
        @PathVariable String id) {
    
    byte[] imageData = imageService.getImage(id);
    
    CacheControl cacheControl;
    
    // 根据图片类型设置不同缓存策略
    switch (type) {
        case "avatar":  // 用户头像：中等缓存
            cacheControl = CacheControl.maxAge(2, TimeUnit.HOURS);
            break;
        case "product": // 产品图片：长期缓存
            cacheControl = CacheControl.maxAge(30, TimeUnit.DAYS);
            break;
        case "temp":    // 临时图片：不缓存
            cacheControl = CacheControl.noCache();
            break;
        default:        // 默认：1天缓存
            cacheControl = CacheControl.maxAge(1, TimeUnit.DAYS);
    }
    
    return ResponseEntity.ok()
        .cacheControl(cacheControl)
        .contentType(getMediaType(id))
        .body(imageData);
}
```

## 三、缓存策略推荐

### **根据业务场景选择：**

| 图片类型     | 缓存时间 | 理由             |
| ------------ | -------- | ---------------- |
| **用户头像** | 1-2小时  | 用户可能更换头像 |
| **产品图片** | 7-30天   | 很少变化         |
| **文章配图** | 7天      | 基本不变         |
| **验证码**   | 不缓存   | 每次必须新       |
| **编辑预览** | 5分钟    | 短时有效         |

### **推荐配置：**

```
// 通用图片缓存配置
public static CacheControl getCacheControl(String imageType) {
    switch (imageType) {
        case "avatar":
            return CacheControl.maxAge(2, TimeUnit.HOURS)
                    .cachePublic()
                    .mustRevalidate();
        case "icon":
        case "logo":
            return CacheControl.maxAge(365, TimeUnit.DAYS)  // 一年
                    .cachePublic()
                    .immutable();  // 永不变
        default:
            return CacheControl.maxAge(7, TimeUnit.DAYS)
                    .cachePublic();
    }
}
```

### **1. 配合拦截器统一处理**

```
@Component
public class CacheControlInterceptor implements HandlerInterceptor {
    
    @Override
    public void postHandle(HttpServletRequest request, 
                          HttpServletResponse response, 
                          Object handler, ModelAndView modelAndView) {
        
        String uri = request.getRequestURI();
        
        // 图片请求统一加缓存头
        if (uri.matches(".*\\.(jpg|jpeg|png|gif|ico)$")) {
            response.setHeader("Cache-Control", "public, max-age=86400"); // 1天
        }
    }
}
```

### **2. 动态缓存时间**

```
@GetMapping("/dynamic/{id}")
public ResponseEntity<byte[]> getImageWithDynamicCache(@PathVariable String id) {
    ImageInfo image = imageService.getImageInfo(id);
    
    // 根据更新频率动态设置缓存
    long cacheHours = calculateCacheHours(image);
    
    return ResponseEntity.ok()
        .cacheControl(CacheControl.maxAge(cacheHours, TimeUnit.HOURS))
        .header("X-Cache-Hint", "dynamic-" + cacheHours + "h")
        .body(image.getData());
}

private long calculateCacheHours(ImageInfo image) {
    // 根据最后修改时间、访问频率等计算缓存时间
    if (image.getUpdateTime().isAfter(LocalDateTime.now().minusDays(1))) {
        return 1;  // 24小时内更新过：缓存1小时
    } else if (image.getUpdateTime().isAfter(LocalDateTime.now().minusDays(7))) {
        return 24; // 7天内更新过：缓存1天
    } else {
        return 24 * 7;  // 更早：缓存1周
    }
}
```

## 🔧 技术对比：

### 扩容系统盘的**缺点**：

1. **风险集中**：系统盘坏了，所有数据都没了
2. **IO竞争**：Docker/MySQL和系统抢IO
3. **备份麻烦**：系统盘快照包含所有数据，大而慢
4. **迁移困难**：想换服务器时，整个系统盘都要迁移

### 新购数据盘的**缺点**：

1. **管理复杂**：需要手动挂载、配置
2. **迁移麻烦**：应用要改配置路径
3. **额外费用**：多一块盘的钱
4. **可能闲置**：如果数据不多，浪费空间

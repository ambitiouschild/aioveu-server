## 🛠 四、优化建议（让缓存更稳定）

### 1. 确保CDN域名配置正确

- 域名：`cdn.aioveu.com`

- 回源类型：**OSS源站**

- 回源Host：填写你的OSS Bucket域名（如：`your-bucket.oss-cn-hangzhou.aliyuncs.com`）

- 缓存规则：

  ```
  文件类型：*.png;*.jpg;*.jpeg;*.gif;*.webp;*.ico;*.svg
  缓存时间：30天（2592000秒）
  忽略URL参数：是（避免?_t=xxx 导致缓存失效）
  ```

### 2. 设置OSS对象缓存头（可选，增强控制）

在OSS控制台，为Bucket设置默认缓存策略：

```
<CacheControl>public, max-age=2592000</CacheControl>
<Expires>Thu, 01 Jan 2030 00:00:00 GMT</Expires>
```

### 3. 强制使用CDN域名（前端代码）

确保前端代码中所有图片都使用 `cdn.aioveu.com`域名，而不是OSS内网或直连地址：

```
<!-- ✅ 正确 -->
<img src="https://cdn.aioveu.com/aioveu/1001/image/20260302/15b11bd2149840fdf9...png">

<!-- ❌ 错误（可能不走CDN） -->
<img src="https://your-bucket.oss-cn-hangzhou.aliyuncs.com/aioveu/1001/image/20260302/15b11bd2149840fdf9...png">
```

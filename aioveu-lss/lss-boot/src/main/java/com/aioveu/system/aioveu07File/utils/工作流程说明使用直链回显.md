### 1. 后端代码配置

**AliyunFileService.java 修改**：

```
@Service
@Slf4j
public class AliyunFileService implements FileService {
    
    @Value("${oss.aliyun.bucket-name}")
    private String bucketName;
    
    @Value("${oss.aliyun.endpoint}")
    private String endpoint;
    
    @Value("${oss.aliyun.cdn-domain}")
    private String cdnDomain;
    
    @Override
    public FileInfo uploadFile(MultipartFile file, String moduleName) {
        // 1. 上传文件到OSS
        String fileName = filePathGenerator.generatePath(tenantId, bucketName, moduleName, originalFilename);
        aliyunOssClient.putObject(bucketName, fileName, file.getInputStream(), metadata);
        
        // 2. 构建两种URL
        // OSS直链（用于立即显示）
        String ossDirectUrl = buildOssDirectUrl(fileName);
        
        // CDN地址（用于后续访问）
        String cdnUrl = buildCdnUrl(fileName);
        
        // 3. 异步预热CDN（不阻塞主流程）
        warmUpCdnAsync(cdnUrl);
        
        // 4. 返回包含双地址的FileInfo
        FileInfo fileInfo = new FileInfo();
        fileInfo.setName(file.getOriginalFilename());
        fileInfo.setUrl(cdnUrl);          // 主URL：CDN地址
        fileInfo.setDirectUrl(ossDirectUrl); // 备选URL：OSS直链
        fileInfo.setPath(fileName);
        fileInfo.setWarmUpStatus("pending"); // 预热状态
        fileInfo.setCreatedAt(new Date());
        
        log.info("文件上传成功，返回双地址 - OSS: {}, CDN: {}", ossDirectUrl, cdnUrl);
        
        return fileInfo;
    }
    
    /**
     * 构建OSS直链地址
     * 格式：https://bucket.endpoint/file/path
     */
    private String buildOssDirectUrl(String fileName) {
        return String.format("https://%s.%s/%s", 
            bucketName, endpoint, fileName);
    }
    
    /**
     * 构建CDN地址
     * 格式：https://cdn.domain.com/file/path
     */
    private String buildCdnUrl(String fileName) {
        if (StringUtils.isNotBlank(cdnDomain)) {
            return String.format("https://%s/%s", cdnDomain, fileName);
        } else {
            // 如果没有配置CDN域名，回退到OSS地址
            return buildOssDirectUrl(fileName);
        }
    }
    
    /**
     * 异步预热CDN
     */
    private void warmUpCdnAsync(String cdnUrl) {
        CompletableFuture.runAsync(() -> {
            try {
                log.info("开始异步预热CDN: {}", cdnUrl);
                
                // 方法1: 使用阿里云API预热
                String taskId = warmUpWithAliyunApi(cdnUrl);
                
                // 方法2: 触发HEAD请求（可选）
                triggerHeadRequest(cdnUrl);
                
                log.info("CDN预热任务已提交，任务ID: {}", taskId);
                
            } catch (Exception e) {
                log.warn("CDN预热失败: {}", e.getMessage());
            }
        });
    }
}
```

### 3. 前端配置示例

**前端调用上传接口**：

```
// 文件上传处理
async function uploadFile(file) {
    const formData = new FormData();
    formData.append('file', file);
    
    try {
        const response = await fetch('/api/v1/files/upload', {
            method: 'POST',
            body: formData
        });
        
        const result = await response.json();
        
        if (result.success) {
            const fileInfo = result.data;
            
            // 立即显示图片（使用OSS直链确保可用性）
            displayImageImmediately(fileInfo.directUrl);
            
            // 保存到数据库（保存CDN地址）
            await saveToDatabase({
                fileName: fileInfo.name,
                url: fileInfo.url,          // 保存CDN地址
                directUrl: fileInfo.directUrl, // 可选：保存直链备用
                path: fileInfo.path
            });
            
            // 监听CDN就绪状态（可选）
            monitorCdnStatus(fileInfo);
            
            return fileInfo;
        }
    } catch (error) {
        console.error('上传失败:', error);
    }
}

// 立即显示图片
function displayImageImmediately(directUrl) {
    const img = new Image();
    img.onload = () => {
        console.log('图片加载完成（OSS直链）');
        // 显示到页面
        document.getElementById('preview').appendChild(img);
    };
    
    img.onerror = () => {
        console.error('图片加载失败');
        // 可以显示占位图
    };
    
    img.src = directUrl;
    return img;
}

// 智能图片加载器
function loadImageSmart(fileInfo) {
    return new Promise((resolve) => {
        const img = new Image();
        let loaded = false;
        
        // 设置超时（2秒）
        const timeout = setTimeout(() => {
            if (!loaded) {
                console.log('主地址加载超时，尝试备选地址');
                img.src = fileInfo.directUrl;
            }
        }, 2000);
        
        // 尝试CDN地址
        img.onload = () => {
            loaded = true;
            clearTimeout(timeout);
            console.log('CDN加载成功');
            resolve(img);
        };
        
        img.onerror = () => {
            if (!loaded) {
                // 切换到OSS直链
                img.src = fileInfo.directUrl;
            }
        };
        
        // 先尝试CDN地址
        img.src = fileInfo.url;
    });
}
```

### 5. Controller配置

**FileController.java**：

```
@RestController
@RequestMapping("/api/v1/files")
@RequiredArgsConstructor
@Slf4j
public class FileController {
    
    private final FileService fileService;
    
    @PostMapping("/upload")
    public Result<FileInfo> uploadFile(@RequestParam("file") MultipartFile file,
                                        @RequestParam(value = "module", required = false, defaultValue = "common") String module) {
        try {
            log.info("接收文件上传: {}，模块: {}", file.getOriginalFilename(), module);
            
            // 调用Service上传
            FileInfo fileInfo = fileService.uploadFile(file, module);
            
            // 返回结果
            return Result.success("上传成功", fileInfo);
            
        } catch (Exception e) {
            log.error("文件上传失败", e);
            return Result.fail("上传失败: " + e.getMessage());
        }
    }
    
    /**
     * 检查CDN预热状态
     */
    @GetMapping("/check-cdn-status")
    public Result<CdnStatus> checkCdnStatus(@RequestParam String fileUrl) {
        try {
            CdnStatus status = fileService.getCdnStatus(fileUrl);
            return Result.success(status);
        } catch (Exception e) {
            return Result.fail("检查失败: " + e.getMessage());
        }
    }
    
    /**
     * 手动触发CDN预热
     */
    @PostMapping("/trigger-warm-up")
    public Result<String> triggerWarmUp(@RequestParam String fileUrl) {
        try {
            String taskId = fileService.triggerCdnWarmUp(fileUrl);
            return Result.success("预热已触发，任务ID: " + taskId);
        } catch (Exception e) {
            return Result.fail("预热失败: " + e.getMessage());
        }
    }
}
```

## 工作流程说明

1. **上传阶段**：
   - 用户上传文件 → 后端保存到OSS
   - 立即生成OSS直链地址和CDN地址
   - 异步提交CDN预热任务
   - 立即返回给前端（包含双地址）
2. **显示阶段**：
   - 前端立即使用OSS直链显示图片
   - 后台CDN预热任务执行中
   - 用户能看到图片，但可能加载稍慢
3. **缓存阶段**：
   - CDN预热完成，文件缓存到边缘节点
   - 后续访问使用CDN地址，实现快速加载
   - 数据库中保存CDN地址用于长期访问
4. **降级策略**：
   - 如果CDN地址访问失败，自动回退到OSS直链
   - 确保图片始终可访问

## 关键优势

1. **快速响应**：上传接口立即返回，不等待CDN预热
2. **高可用性**：使用OSS直链确保图片始终可访问
